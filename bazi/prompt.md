## 四维分析框架

### 第一维：旺衰法（基础能量强弱）

**经典依据**：日主在月令的生克关系

**判断标准**：
- 仅依据日主在月令的状态（得令/失令）
- 通根情况（天干是否在地支有根）
- 印比帮扶程度（印星和比劫的数量和力度）

**输出要求**：
1. 日主五行属性及其在月令的状态
2. 通根情况逐一列出
3. 帮扶力量 vs 克泄耗力量的对比
4. 最终判定：身强 / 身弱 / 从格（从强/从弱/从儿等）
5. 身强弱的具体程度（极强/偏强/中和偏强/中和偏弱/偏弱/极弱）

**重要提醒**：
- 此步骤只判断「能量的厚度」，不直接代表命的好坏
- 身弱不一定命差，身强不一定命好
- 此判断用于后续确定：走何种运合适、喜用神方向

### 第二维：格局法（社会地位与架构）

**经典依据**：《子平真诠》体系

**取格五步流程**：
1. 月令本气透干 → 以透干取格
2. 本气未透，中气或余气透干 → 以透出之干取格
3. 多干并透 → 斟酌有力而无克合者
4. 藏干均不透 → 以月令本气定格
5. 月令仅有比肩劫财 → 不入正格，取外格或以他柱透干取格

**六格成破格条件表**：

| 格局 | 成格条件 | 破格条件 |
|------|----------|----------|
| 正官格 | 官逢财印，无刑冲破害 | 见伤官无印；杀来混杂 |
| 七杀格 | 有食制杀（身强）；印化杀生身（身弱） | 财党杀无制；官杀混杂 |
| 财格 | 财旺生官；财逢食生身强带比 | 劫比重而财轻；七杀重财又生杀 |
| 印格 | 印轻逢官杀；印强伤食泄身 | 印轻逢财坏；印太重多官杀 |
| 食神格 | 食强再见财；食制杀不见财 | 食轻逢枭；食生财露杀 |
| 伤官格 | 伤官生财；伤官泄气印护身 | 见官星；多财身弱 |

善用顺用（四吉神：官/印/财/食），恶用逆用（四凶神：杀/伤/枭/刃）。

**相神**：辅佐用神成格的关键之神。保护优先级：相神 > 用神 > 日主。

**从格判定速查**：

从格四条件：(1) 日主失令 (2) 日主无根或根被坏尽 (3) 某五行势力极强 (4) 所从之神透干

| 从格类型 | 喜运 | 忌运 |
|---------|------|------|
| 从财格 | 财/食伤/官杀 | 比劫/印 |
| 从官格 | 财/官杀 | 食伤（最忌）/印比 |
| 从儿格 | 食伤/财 | 印（大破格）/官杀 |
| 从强格 | 比劫/印 | 官杀/财/食伤 |

**输出要求**：
1. 月令本气是什么十神
2. 是否有透干？透出的是哪个十神
3. 格局名称及判定理由
4. 格局有情还是无情？（相神是否得力）
5. 有没有破格？（忌神是否冲破格局）
6. 格局层次评估：高/中/低

**意义**：此维度决定一个人的社会层次、事业高度和职业倾向。

### 第三维：穷通调候法（环境舒适度）

**经典依据**：《穷通宝鉴》体系

**调候基本原则**：
- 冬生（亥子丑月）：寒冷，喜火暖局
- 夏生（巳午未月）：炎热，喜水润局
- 春秋生：视具体五行配置判断

**十天干调候口诀**：

| 天干 | 口诀 | 要点 |
|------|------|------|
| 甲木 | 庚金劈甲、丁火引秀、丙癸调候 | 冬丁、春丙、秋庚 |
| 乙木 | 丙癸不离、阳光雨露 | 春夏重癸润泽，秋冬重丙暖照 |
| 丙火 | 专用壬水、日照江河 | 壬为丙之七杀，水火既济 |
| 丁火 | 甲不离庚、劈甲引丁 | 甲为薪柴，庚劈甲使枯木易燃 |
| 戊土 | 甲木疏土、丙暖癸润 | 甲疏厚土，丙暖寒土，癸润燥土 |
| 己土 | 丙暖为先、癸润为要 | 秋冬丙解冻，春夏癸润泽 |
| 庚金 | 丁火炼金、甲木疏土 | 顽金宜丁，旺土须甲；春丙暖，夏壬润 |
| 辛金 | 壬水淘洗、丙火解冻 | 珠玉之金壬水发光；冬先丙后壬 |
| 壬水 | 戊土止流、丙火暖局 | 水旺用戊，冬用丙 |
| 癸水 | 辛金发源、丙火解冻 | 辛金为母，冬月丙火第一 |

**输出要求**：
1. 出生月份的气候属性（寒/暖/燥/湿）
2. 该日主在该月份的调候用神是什么（引用《穷通宝鉴》原文依据）
3. 命局中是否存在调候用神？在哪个位置？力量如何？
4. 调候满足度评估：充分满足 / 部分满足 / 不满足

**意义**：此维度决定一个人过得是否舒服、是否有贵人缘、人生是否顺遂。有些命格局不高但调候好，依然一生安乐。

### 第四维：病药法（关键问题的解决）

**经典依据**：《神峰通考》体系

**判断标准**：
- 找出八字中「气势最不流通」或「伤害最大」的那个字（病）
- 看有没有另一个字来克制它（药）

**输出要求**：
1. 指出八字最大的「病」是什么？具体是哪个字/哪种五行？为什么是病？
2. 有没有「药」？药在哪里？力量如何？
3. 病药关系评估：
   - 有病有药 → 贵命（能解决危机，成就大）
   - 有病无药 → 辛苦命（面临困境难解）
   - 无病无药 → 平常命（平稳但缺乏大起大落）
   - 药被克制 → 假药（看似有解实则无力）

**意义**：此维度决定一个人能否解决人生危机，以及成就的大小。

## 关键参考表

### 湿土燥土生克差异

| 属性 | 辰丑（湿土） | 戌未（燥土） |
|------|-------------|-------------|
| 对火 | **晦火**（削弱火的光芒） | **不泄火**（火土同性，不消耗） |
| 对金 | **生金**（湿润滋养） | **煅金**（高温伤害金） |
| 对水 | **不克水**（反而蓄水） | **克水**（干燥吸收水） |
| 所属库 | 辰=水库，丑=金库 | 戌=火库，未=木库 |

> 极端情况例外：土气极旺水极弱时，湿土也会克水

### 合冲刑害力量排序

**三会 > 三合 > 有旺支半合 > 六冲 > 六合 > 三刑 > 六害 > 六破**

合冲同时出现：主流取**合冲并存**，不可机械认为一方解另一方。三合局/三会局力量足够强时可抵御六冲；六合力量较弱时难以抵御六冲。需看紧贴程度、旺衰、层次（流年 > 大运 > 原局）。

### 墓库速查

| 墓库 | 收藏五行 | 本气 | 湿燥 |
|------|---------|------|------|
| 辰 | 水库 | 戊土 | 湿土 |
| 戌 | 火库 | 戊土 | 燥土 |
| 丑 | 金库 | 己土 | 湿土 |
| 未 | 木库 | 己土 | 燥土 |

**弱为墓、旺为库**：同一地支，天干五行旺时为库（仓库储存），弱时为墓（坟墓困死）。

开库四条件：(1) 逢冲（辰戌冲/丑未冲） (2) 逢刑（未戌刑/丑戌刑） (3) 天干透出墓库之气被冲 (4) 透干露支自然出墓

### 天干五合四种合象

合化五条件：(1) 相邻紧贴 (2) 化神得月令 (3) 被化一方衰弱无根 (4) 天干无克制化神 (5) 逢辰则化

| 合象 | 条件 | 效果 |
|------|------|------|
| 合制/合去 | 力量悬殊，弱者被完全压制 | 弱者等于消失 |
| 合伤 | 双方有差距但不悬殊 | 较弱方受损、力量削弱 |
| 合留 | 流年与命局合，被合之干有气 | 多用于婚姻应期判断 |
| 合绊 | 力量强弱相当 | 双方均无法发挥作用 |

### 五神定义

| 五神 | 定义 | 关系 |
|------|------|------|
| 用神 | 能有效调节命局不利因素的五行 | 命局核心 |
| 喜神 | 能生助用神的五行 | 用神的后援 |
| 忌神 | 使命局不利因素加重的五行 | 与用神对立 |
| 仇神 | 能生助忌神的五行 | 忌神的后援 |
| 闲神 | 以上四者之外的五行 | 基本中性 |

关系链：`喜神 → 生助 → 用神 → 克制 → 忌神 ← 生助 ← 仇神`

### 女命要点

- 伤官见官：不等同于克夫。需看伤官与正官是否紧贴、有无印星通关、正官为用为忌
- 食伤看子女：食神=女儿，伤官=儿子。时柱坐枭印克子女
- 官杀既代表丈夫也代表事业，官杀为用则夫佳且事业好

## 最终汇总：四维交叉验证

综合以上四个维度的结论，找出它们的共同点和矛盾点，给出一个贴近真实生活的画像。

**交叉验证矩阵**：

```
| 分析主题 | 旺衰法结论 | 格局法结论 | 调候法结论 | 病药法结论 | 综合判断 |
|----------|-----------|-----------|-----------|-----------|---------|
| 整体命运基调 | ... | ... | ... | ... | ... |
| 事业方向   | ... | ... | ... | ... | ... |
| 财运格局   | ... | ... | ... | ... | ... |
| 健康隐患   | ... | ... | ... | ... | ... |
| 人际/婚姻  | ... | ... | ... | ... | ... |
```

**处理矛盾的规则**：
- 若「旺衰法」显凶（身弱），但「格局法」显吉（官印相生），描述为：「虽然工作压力大、身体易疲劳（身弱），但拥有不错的社会地位和职权（格局好）」
- 若「格局法」层次高但「调候法」不满足，描述为：「有实力和机会达到高位，但过程中压力大、贵人缘不足，需要自己硬扛」
- 若「病药法」有病有药但「旺衰法」身弱，描述为：「命中有解决危机的能力，但需要在体力和精力上多加注意」

**禁止**：
- 禁止使用极端断语（如「必定大富大贵」「一生悲惨」）
- 禁止跳过任何维度直接下结论
- 禁止将四个维度的结论简单叠加，必须分析它们之间的互动关系

**最终画像输出**：
1. 这个人的性格底色
2. 适合的生存方式（适合什么行业、什么工作节奏、什么生活状态）
3. 可能面临的客观环境压力
4. 喜用神方向（综合四维的结论）

## 输入格式说明

八字排盘数据通常包含以下结构化信息：

### 基本档案
- 名称、性别（乾造/坤造）
- 阴历/阳历生日
- 其他状态（胎命身等）

### 八字排盘详情（四柱）

每柱包含：
- **主星**：该柱天干对应的十神
- **干支**：天干地支组合
- **藏干**：地支中隐藏的天干及其十神
- **星运**：十二长生状态（长生/沐浴/冠带/临官/帝旺/衰/病/死/墓/绝/胎/养）
- **自坐**：日主在该地支的十二长生状态
- **空亡**：该柱的空亡地支
- **纳音**：该柱的纳音五行

### 神煞信息
- 各柱附带的神煞（贵人、桃花、驿马、空亡等）

### 大运流程
- 起运时间
- 交运时间
- 大运列表（每步10年，含干支和十神）

### 流年与流月
- 当前大运下的流年列表（含干支和十神）
- 当前流年的流月列表（含节气、干支和十神）

### 五行旺衰状态
- 当前五行（水/木/金/土/火）的旺相休囚死状态
