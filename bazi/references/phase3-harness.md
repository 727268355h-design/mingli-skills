### f-pre. 用神过旺反转检查（红线规则，2026-04-03 新增）

**在确定最终用神之前，必须检查：要取的用神五行，在命局中是否已经过旺？**

触发条件（满足**任一**即触发反转）：
1. 用神五行**得月令** + 天干有≥2个该五行 → 过旺
2. 用神五行在命局（天干+地支本气+中气）出现≥4次 → 过旺
3. 穷通宝鉴有"X旺忌X"的明确警告（如"水旺木漂"、"火炎土燥"）→ 过旺

过旺时的处理：
- **禁止**再取该五行为用神（印旺不再用印，比旺不再用比）
- 回退到其他取用法重新选择（优先调候法或泄耗法）
- 标注 ⚠️ "用神反转：原扶抑法取[X]，但[X]已过旺，改取[Y]"

经典反转案例：
- 水旺木漂（冬月水多）→ 不用水印，改用火调候 + 土止水
- 印多为忌（印枭过旺夺食）→ 印成忌神，取财制印
- 比劫争财（身太旺）→ 不用比劫，取官杀制比
- 火炎土燥（夏月火旺）→ 不用火，改用水调候

> 教训案例（2026-04-03）：甲木亥月，壬水当令+透干+三处根，扶抑法机械取"印星水"为用，
> 但水已极旺，再加水=木漂。穷通宝鉴明确"水旺木漂"。触发反转后改取丁火（调候+泄秀）。

### 用神冲突决策树

当五种取用法给出不同用神时，按此优先级裁决：

```
START
 |
 +-- 格局清晰可辨？
 |   +-- 是 --> 以格局用神为主
 |   |        +-- 调候急需？（查三级判定表）
 |   |        |   +-- 极端急需(子丑午未) --> 调候优先于格局
 |   |        |   +-- 急需(亥巳) --> 格局用神 + 调候用神并重，冲突时启动多派系模式
 |   |        |   +-- 参考级 --> 纯以格局用神
 |   |        +-- 有明显病？
 |   |             +-- 是 --> 格局用神 + 病药用神交叉验证
 |   |             +-- 否 --> 纯以格局用神
 |   +-- 否 --> 用旺衰法取扶抑用神
 |            +-- 两行相战？
 |            |   +-- 是 --> 通关用神优先
 |            |   +-- 否 --> 扶抑用神
 |            +-- 调候急需？
 |                 +-- 是 --> 扶抑 + 调候双取
 |                 +-- 否 --> 纯扶抑用神
 +-- 从格？
      +-- 是 --> 顺势取用（从财/从官/从儿/从强各有喜忌）
```

**核心原则**（非极端月默认）：格局 > 扶抑 > 病药 > 通关 > 调候。冬夏极端月例外：调候优先于一切（→ 多派系模式）。

**调候紧急度三级判定**（2026-04-03 修订，替代原二级判定）：

| 等级 | 月份 | 调候优先级 | 说明 |
|------|------|----------|------|
| **极端急需** | 子月、丑月、午月、未月 | 调候 > 一切 | 严冬严夏，无调候则凶 |
| **急需** | **亥月、巳月** | 调候与格局并重 | 初冬初夏，调候不可忽视 |
| 参考级 | 寅卯辰申酉戌月 | 格局 > 调候 | 气候中和，调候降为辅助 |

注意：
- 亥月壬水帝旺，水气已重，甲乙木尤忌水漂 → **不可降级为"参考级"**
- 巳月丙火临官，火势已旺，金水日主已感炎热 → 同理
- "急需"≠"极端急需"，但调候用神必须在决策中占重要权重

**冲突裁决原则**（多源共识）：
- "调候定生死，格局分贵贱" — 调候关乎健康存亡，格局关乎富贵层次
- 冬月（子丑）无丙火、夏月（午未）无壬癸水 → 调候优先于一切
- 春秋月气候中和 → 格局优先
- 调候为急但格局用神恰好也满足调候 → 最佳（如丙火冬生用壬水，壬既是格局用神也是调候用神）

## 3.3 找病药

### a. 找"病"

**病 = 命局中气势最不流通、对命主伤害最大的字或组合。**

四种常见病型：

| 病型 | 定义 | 识别方法 |
|------|------|---------|
| 雕病（太旺伤身）| 某五行极旺，克泄耗日主 | 官杀太旺克身；食伤太旺泄身 |
| 枯病（太弱无用）| 用神太弱或缺失 | 需要的用神不见或被克尽 |
| 旺病（偏枯不流通）| 某行一枝独秀，气势堵塞 | 一行独旺，其余衰弱，流通断裂 |
| 弱病（关键环节断链）| 流通链中某环节太弱 | 五行相生链在某处断裂 |

> **找病三步法**（实战提炼）：
> 1. **找最突出矛盾**（不是列所有问题）：哪个字/五行最"碍事"？去掉它全局就通了？核心问题："什么让日主最难受？"
> 2. **七种常见病源**：
>    - 官杀太旺攻身（身弱无制杀）-- 身弱最常见的病
>    - 印星太重泄官夺食（印枭过旺，生多为克）
>    - 财星坏印（财克印，学业/贵人受损）
>    - 比劫争财（比劫旺抢财星）-- 身强最常见的病
>    - 食伤太旺泄身（聪明反被聪明误）
>    - 伤官见官（伤克官，名声受损）
>    - 五行偏枯（某行过旺或完全缺失，流通链断裂）
> 3. **验证法**：假设去掉这个字，命局是否立刻变好？是 = 确认为病。
>    - 病的严重度看位置：月令为病最重（占 40-50%）> 日支 > 时柱 > 年柱
>    - 药的有效性三看：力量够不够 / 位置对不对（远水救不了近火）/ 有没有被破坏（药被冲克 = 无药）

**执行步骤**：

1. 检查命局五行流通链：`木→火→土→金→水→木`，哪个环节断裂或堵塞？
2. 检查哪个字/组合对日主/用神伤害最大？
3. 检查哪个字/组合导致格局破败？
