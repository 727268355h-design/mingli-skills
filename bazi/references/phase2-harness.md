## 2.3 定旺衰（六级判定）

**严格按以下六步执行，每步写出结论：**

### a. 月令得令否（权重 40-50%）

```
日主[X]在[月]月 = [旺/相/休/囚/死] → [得令 ✓ / 失令 ✗]
```

### b. 日支得地否（权重 15-20%）

日支藏干中是否有与日主同五行的天干（比肩/劫财/印星的根）？
```
日支[X]藏干：[本气/中气/余气]
其中与日主同五行或生日主的：[有/无] → [得地 ✓ / 失地 ✗]
```

### c. 通根检查

日主天干在四柱所有地支的藏干中，有无同类根气：
```
年支[X]藏干：[列出] → 有无[日主同类]？[有根 ✓ / 无根 ✗]
月支[X]藏干：[列出] → 有无[日主同类]？[有根 ✓ / 无根 ✗]
日支[X]藏干：[列出] → 有无[日主同类]？[有根 ✓ / 无根 ✗]
时支[X]藏干：[列出] → 有无[日主同类]？[有根 ✓ / 无根 ✗]

通根总结：共[N]处通根，根气[强/中/弱/无]
```

### d. 帮扶力量统计

统计命局中能帮助日主的力量（印星生身 + 比劫帮身）：
```
印星（生我者）：[逐一列出，标注位置和力量]
比劫（同我者）：[逐一列出，标注位置和力量]
帮扶总力量：[强/中/弱]
```

### e. 克泄耗力量统计

统计命局中削弱日主的力量：
```
官杀（克我者）：[逐一列出]
食伤（我生者=泄）：[逐一列出]
财星（我克者=耗）：[逐一列出]
克泄耗总力量：[强/中/弱]
```

> **得势加权判定**（替代简单计数，2026-04-03 修订）：
>
> ⚠️ **禁止使用简单计数法**（"帮扶个数-克泄耗个数"）做得势判定。等权计数不考虑月令权重和季节折扣，容易误判。
>
> **加权规则**：
> 1. **月令加权**：月令帮扶的五行，力量 ×3（一票否决级）。月令帮且其他不太差 = 基本身强。
> 2. **季节折扣**：当令五行 ×1.5；囚死五行 ×0.5（如冬月火为死、土为囚，力量大减）
> 3. **间接帮扶**：杀印相生链中，七杀通过印星间接帮身 → 从"克"移入"帮"（庚→壬→甲，庚为间接帮扶）
> 4. **力量折扣**：被克制/截脚/合绊的字，力量减半
> 5. **位置权重**：天干透出 ×1.2，地支本气 ×1.0，中气 ×0.6，余气 ×0.3
> 6. 三合/三会局有决定性作用：3 个地支同时帮扶，力量巨大
>
> 加权后比较帮扶总分 vs 克泄耗总分 → 得势/失势。
>
> **注意：此结果仅作为三得法"得势"判定的输入值，不得单独用于旺衰最终判定。**
> 真正中和极少，大多数八字非偏强即偏弱。

### f. 综合判定

**方法 A：三得法定性（主要方法）**

| 组合 | 得令 | 得地 | 得势 | 判定 |
|------|------|------|------|------|
| 1 | ✓ | ✓ | ✓ | 偏强~极强 |
| 2 | ✓ | ✓ | ✗ | 偏强 |
| 3 | ✓ | ✗ | ✓ | 中和偏强 |
| 4 | ✓ | ✗ | ✗ | 中和（偏强侧）|
| 5 | ✗ | ✓ | ✓ | 中和偏强 ⚠️ 失令但得地得势仍可身旺 |
| 6 | ✗ | ✓ | ✗ | 中和偏弱 |
| 7 | ✗ | ✗ | ✓ | 偏弱 ⚠️ 若天干同类3+透出，升级为中和偏强（身杀两停特例） |
| 8 | ✗ | ✗ | ✗ | 极弱（检查从格）|

得势判定 = 帮扶力量（印+比劫） > 克泄耗力量（官杀+食伤+财）

**方法 B：量化打分法（辅助验证）**

总分 560 分制（基于天干地支力量量化）：

| 分值区间 | 判定 | 说明 |
|---------|------|------|
| >450 | 极强（旺极） | 专旺格条件 |
| 281-450 | 偏强（太旺） | 身旺，喜克泄耗 |
| 120-280 | 偏强 | 身稍旺 |
| 105-119 | 中和 | 平衡 |
| 45-104 | 偏弱 | 身稍弱，喜生扶 |
| 16-44 | 太弱 | 身很弱，检查假从 |
| <15 | 极弱（弱极） | 真从格条件 |

> 打分法为辅助，实战以三得法定性为主，打分法验证。两者矛盾时以三得法为准。

**方法冲突裁决规则（不可 Override，2026-04-03 新增）**：

| 优先级 | 方法 | 使用场景 |
|--------|------|---------|
| 1 | **三得法定性** | 主裁决，所有情况下优先 |
| 2 | 量化打分法 | 辅助验证，与三得法矛盾时以三得法为准 |
| 3 | 加权得势判定 | 仅决定三得法的"得势/失势"输入值 |

**红线**：
- 三得法组合编号(1-8)的参考值 = 旺衰判定的**基准线**
- 任何试图用加权得势或打分法推翻三得法最终结论的行为 = **违规**
- 如果分析者认为三得法结论明显不合理，必须标注 ⚠️ 走"多派系模式"交叉验证，**不得自行 Override**

> 教训案例（2026-04-03）：甲木亥月命局，三得法组合4="中和偏强侧"，
> 但分析者用简单计数法（帮2-克5=-3）override 为"中和偏弱"，
> 导致用神取壬水（印星已极旺），喜忌全反。正确答案是喜火土。

**从格硬判定标准**：
- 日主有 1 个**本气**通根 → 不论从格
- 日主仅有**余气**根（力量<10%）→ 可论假从
- 日主完全无根无帮扶 + 某行极旺 + 该行透干 → 真从

```
综合旺衰判定：[极强/偏强/中和偏强/中和/中和偏弱/偏弱/太弱/极弱/从格]
三得法组合：得令[✓/✗] + 得地[✓/✗] + 得势[✓/✗] → 组合[1-8]
量化验证（如适用）：[X]分 → [对应判定]
```

### 从格检查（仅当极弱/太弱时执行）

四条必须同时满足：
- [ ] 日主失令
- [ ] 日主无本气通根（仅余气根或完全无根）
- [ ] 命局中某种五行势力极强
- [ ] 所从之神透干

全满足 → 从格（标注从财/从官/从儿/从强）。否则 → 不从，仍为正格身弱。
有余气根无本气根 → 假从（大运见帮扶即不从）。

## 2.4 取格局

**严格按以下五步执行：**

### a. 月令本气透干否？

```
月支[X]本气 = [Y]
[Y]是否出现在年干/月干/时干中？→ [是：以[Y]取格] / [否：下一步]
```

### b. 中气/余气透干否？

```
月支[X]中气 = [Y1]，余气 = [Y2]
[Y1]透干？→ [是：以[Y1]取格]
[Y2]透干？→ [是：以[Y2]取格]
都不透 → 下一步
```

### c. 多干并透

若月令藏干有多个透出天干 → 取有力而无克合者。

### d. 藏干均不透

以月令本气直接定格。

### e. 月令为比劫

月令本气为日主同类（比肩/劫财）→ 不入正格，取建禄格/月刃格，另以他柱透干取用。

### 格局命名

以透出/取定的十神命名格局：
```
格局 = [正官格/七杀格/正财格/偏财格/正印格/偏印格/食神格/伤官格/建禄格/月刃格/从X格]
```

### 成格/破格速查

查下表判断格局是否成立：

| 格局 | 成格条件 | 破格条件 |
|------|---------|---------|
| 正官格 | 官逢财印，无刑冲破害 | 见伤官无印；杀来混杂 |
| 七杀格 | 食制杀（身强）；印化杀（身弱） | 财党杀无制；官杀混杂 |
| 财格 | 财旺生官；财逢食生+身强带比 | 劫比重财轻；杀重财又生杀 |
| 印格 | 印轻逢官杀；印强食伤泄身 | 印轻逢财坏；印太重多官杀 |
| 食神格 | 食强再见财；食制杀不见财 | 食轻逢枭；食生财露杀 |
| 伤官格 | 伤官生财；伤泄气印护身 | 见官星；多财身弱 |

```
格局成败判定 = [成格 ✓ / 破格 ✗ / 成中有败（标注破格因素）]
相神 = [辅佐用神成格的关键十神]
```

**格局层次四级判定**（子平真诠"有情有力"体系）：

| 层次 | 条件 | 说明 |
|------|------|------|
| 一等（上格） | 有情 + 有力 | 用神得力，相神护卫，格局纯清 |
| 二等（中上） | 有情 + 力弱 或 有力 + 情杂 | 格局成但有小瑕疵 |
| 三等（中下） | 无情 + 有力 或 有情 + 无力 | 半成品，需运弥补 |
| 四等（下格） | 无情 + 无力 | 废格，格局不成 |

**"有情"三层次判定**（子平真诠精义 + 实战案例提炼）：
1. **救应有情**：格局被破时有另一字来救，且救神有力有根（如壬合丁救官格，比壬克丁更有情）
2. **制化有情**：凶神被制（如食制杀），制神与被制者同根月令 = 最高级有情（能量同源共振）
3. **协同增效**：多十神各归其位不打架（如财印兼格互不侵犯）；凶神转向（伤官不伤官而生财 = 有情）

速判：合 > 生 > 制 > 克（合是最高级的有情，硬克是最低级）。同根月令 = 有力。
核心检验：问"这个十神是在帮忙还是在添乱？"帮忙 = 有情，添乱 = 无情。

有力 = 用神得令、通根、透干，不被克合
