## 8.2 层次判定

**命局层次标准表**：

| 层次 | 格局条件 | 旺衰条件 | 调候条件 | 病药条件 |
|------|---------|---------|---------|---------|
| **上等**（大富大贵） | 格局成立+相神得力+清纯 | 身旺能担财官 | 调候充分满足 | 有病有药且药力强 |

## 8.3 矛盾调和

当四维结论出现矛盾时，不可忽略，需调和解读：

### 三种常见矛盾的调和模板

**矛盾 1：旺衰凶 + 格局吉**

```
表现：日主偏弱（旺衰角度不利），但格局成立且高（格局角度有利）
调和：压力大但地位不低。此人一生承受较大压力，但因格局成立，
      社会地位和成就不会差。类似"苦干型成功者"。
      大运走帮身运时更好，走克身运时压力倍增。
```

**矛盾 2：格局高 + 调候差**

```
表现：格局成立相神得力，但调候用神缺失/不满足
调和：有实力但缺贵人，需硬扛。此人能力出众但发展环境不佳，
      起步慢或早年辛苦。大运补调候时会有跃升。
      类似"英雄无用武之地，待时而动"。
```

**矛盾 3：病药佳 + 身弱**

```
表现：有病有药且药力足（命局有化解机制），但日主偏弱
调和：有解决危机的能力但体力/精力有限。此人善于化解困难，
      但过程消耗大，需注意健康。大运帮身时如鱼得水。
      类似"智慧型但体弱的谋士"。
```

**四维矛盾优先级**：调候 > 格局 > 旺衰 > 大运（调候是生存，格局是方向，旺衰是能力，大运是时机）

**五种常见矛盾处理模板**（实战提炼）：

| 矛盾 | 表述 | 实际含义 |
|------|------|---------|
| 格局好+身弱 | "有格无力" | 天花板高但难落地，需大运扶身时爆发。有印则"印为桥梁可成大器"，无印则"心高命薄有志难伸"或考虑从格 |
| 调候差+格局好 | "有才缺运" | 能力强但生存环境恶劣，犹如寒冬锦衣。必须找同时解决调候和格局的五行才能两全 |
| 身强+格局破 | "有力无台" | 精力充沛但方向不明，格局定层次而非旺衰 |
| 病药佳+身弱 | "有解药但体虚" | 善于化解困难但过程消耗大，大运帮身时如鱼得水 |
| 三维一致但调候差 | "底色优但气候寒/燥" | 基本面好但生活舒适度差，调候是生死线不可忽略 |

关键：找到一个字能同时解决多维矛盾 = 真正的喜用神（如甲木同时疏土、生火、调候暖局）。
核心原则：不要用一个维度否定另一个维度，用"XX好但XX不足"的并列句式，再判断哪个维度更紧迫。

**执行**：检查本命局是否存在以上矛盾，逐一调和：

```
矛盾调和：
[矛盾描述] → [调和解读]
```

## 8.4 回溯验证

**如果命主提供了已知的过往事件，用以下方法验证分析结论：**

### 验证步骤

1. **列出已知事件**：命主告知的重大人生事件及发生年份
2. **反查大运流年**：该年的大运和流年干支是什么？
3. **对照分析结论**：大运流年与原局的互动是否支持该事件？
4. **吻合度判断**：

| 吻合度 | 处理 |
|--------|------|
| 完全吻合 | 验证通过，增强信心 |
| 大致吻合 | 可接受，细节可能受未知因素影响 |
| 不吻合 | ⚠️ 回查以下环节：|

**不吻合时的回查清单**：

| 回查项 | 可能的错误 |
|--------|----------|
| 旺衰判定 | 月令权重是否给足？通根是否遗漏？ |
| 格局判定 | 透干取格是否正确？成败判定是否有误？ |
| 用神选取 | 三维用神是否一致？优先级是否正确？ |
| 地支关系 | 是否遗漏了关键的合/冲/刑？ |
| 大运分析 | 天干地支分管前后5年是否考虑？ |

```
回溯验证：
事件：[X]（[年份]）
当年大运/流年：[干支]
分析结论是否支持：[吻合/大致吻合/不吻合]
若不吻合 → 回查[X]环节，修正结论：[修正内容]
```

## 禁止事项

| 禁止 | 理由 |
|------|------|
| 禁止极端断语（"必定大富大贵"/"一生悲惨"） | 八字只定概率和趋势，不定绝对结果 |
| 禁止跳过任何 Phase 直接下结论 | 每一步都是后一步的基础，跳步必出错 |
| 禁止四维结论简单叠加 | 必须分析四维之间的互动和矛盾 |
| 禁止忽略矛盾信号 | 矛盾信号 = 最有价值的分析线索 |
| 禁止编造未验证的事件 | 只描述趋势和概率，不编造具体事件 |
| 禁止过度依赖神煞 | 神煞只是辅助，格局旺衰才是核心 |
| 禁止断生死 | 命理伦理底线 |
