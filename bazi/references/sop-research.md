### 来源 1：韦千里《千里命稿·评断篇》(1935)

- URL：https://www.quanxue.cn/qt_mingxiang/qianlimg/qianlimg19.html
- 流派：综合派（现代命学评断标准制定者）
- 步骤数：8 步
- 步骤列表：
  1. **看强弱** — 以日干为主，以多寡、盛衰、失时、得令为标准
  2. **定格局** — 以月支为标准（外格不在此例）
  3. **取用神** — 以助强扶弱为标准
  4. **论喜忌** — 以用神为标准
  5. **查岁运** — 以喜忌为标准
  6. **推六亲** — 以四柱六神为标准
  7. **评性情** — 以五行用神等为标准
  8. **断事业** — 以用神及喜忌为标准
- 特色/亮点：民国时期最权威的标准化评断格式，"此书评断篇基本上定出了现代命书的格式"。每一步都有明确的判断标准锚点，层层递进，后步依赖前步结论。至今仍是大多数命理教材的基础框架。

### 来源 2：《三命通会·看命口诀》古法七步

- URL：https://c.m.163.com/news/a/IHDKC00T0525AEBH.html
- 流派：古法传统派
- 步骤数：7 步
- 步骤列表：
  1. **查看月支定格局** — "有官先看官，无官再看格"。月令为命也，从月令层层向外扩展
  2. **以日干为主** — 日干为命主本身，日干与月令的互动是核心切入点
  3. **查看天干** — 向上观察透出的天干，代表命主表现在外的东西（尤其财官印），判断天干是否有根
  4. **观察地支** — 地支为宫位，看六亲关系及地支间刑冲破害
  5. **分析十神组合** — 各种"局"的组合关系，如官印相生
  6. **判断旺衰** — 五行生旺死绝分析（贯穿整个分析过程）
  7. **其他深层分析** — 气象、化气等
- 特色/亮点：古法强调"先看月支有无财官，方看其他"。与现代旺衰派"先定旺衰"不同，古法是"先看月令格局，旺衰贯穿始终"。体现"由内向外、层层展开"的分析逻辑。

### 来源 3：网易"八字分析流程说明"五步法

- URL：https://www.163.com/dy/article/JNL7QDTS0525AEBH.html
- 流派：综合实战派
- 步骤数：5 步
- 步骤列表：
  1. **由远及近平扫四柱** — 大概观看财官印布局，有无伤官见官、财星坏印，地支有无严重冲克，有无旬空，地支合冲刑害关系
  2. **从宏观到微观聚焦** — 聚焦于日干和月令，主要分析日干与月令之间的关系
  3. **从月令向两边延伸** — 详细分析年柱（根/世运）、月干、日支（妻妾宫）、时柱（子息/归宿），逐柱提取信息
  4. **定格局、用神、喜忌** — 对四柱信息有足够了解后才进行格局和用神的判定
  5. **微观返回宏观** — 将眼光从四柱内部扩展到大运、流年、流月，分析流运对四柱的影响
- 特色/亮点：强调"由远及近 → 聚焦核心 → 展开细节 → 定格局 → 放眼大运"的分析节奏。格局和用神不是第一步，而是在充分了解四柱信息之后才进行的第四步。整个过程"几分钟内完成"，强调熟练度。

### 来源 4：君无忧风水命理网"分析命局四大步骤"

- URL：https://www.jun5you.com/welcome/artview1413
- 流派：旺衰实战派
- 步骤数：4 步
- 步骤列表：
  1. **定格局** — 确定八字属于"扶抑格"还是"从格"。先不看干支之间的作用关系
  2. **找出主要矛盾** — 以日干为太极点，找出使日主弱或旺的关键五行，确定用神和忌神。"八字预测中的灵魂和难题"
  3. **提取原命局主要信息** — 用神忌神确定后，依次分析：具体含义、六亲信息、婚姻财运
  4. **结合大运看应期** — 将原命局信息"落到实处"，定出应期
- 特色/亮点：极简四步，"为初学者提供一条捷径"。核心理念：先判断格局类型（扶抑/从格），再找关键矛盾。强调第一步"先不看干支作用关系"，避免循环分析。

### 来源 5：知乎"八字入门六个核心步骤"

- URL：https://zhuanlan.zhihu.com/p/1920095246248804704
- 流派：综合入门派
- 步骤数：6 步
- 步骤列表：
  1. **辨明气候（调候）** — 辨四季冷暖：寒气冻结用火解冻，热气灼烧用水润局
  2. **确定四柱与日主** — 排出年月日时干支，明确日主，标注十神，明确五行分布，排大运
  3. **判断格局** — 以月令为核心看"骨架"，定正官格、七杀格等

  6. **结合大运流年分析** — 大运干支与原局互动，用神运发福，忌神运多困
- 特色/亮点：独特之处在于把"辨明气候（调候）"放在第一步，认为气候是整个八字的"底色"。这与大多数流程不同（多数先看旺衰或格局）。

### 来源 7：盲派论命六大要诀

- URL：https://www.shenjige.cn/details/dkJSNhapY.html + https://www.suanzhun.net/article/2488.html + https://exzh.net/877.html
- 流派：盲派
- 步骤数：核心 6 步 + 10 项分析
- 核心六大要诀：
  1. **干支配置分析命局** — 打开八字看组合，看反映了命主哪些信息
  2. **宾主理论** — 日干为主（主位），他干支为宾（宾位）。"财官在主位就是我的财官"
  3. **十神活论** — 十神不固定含义，根据结构灵活解读
  4. **虚实喜忌** — 天干为虚，地支为实
  5. **应期理论** — 大运流年的应期判断
  6. **看八字之象** — 取象是盲派最核心概念，八字如画像
- 十项分析步骤：
  1. 组合分析（看整体结构）
  2. 财官真假鉴别（"真神须从月中出"）
  3. 清浊判断（清贵 vs 浊命劳碌）
  4. 象的转换（转象、换象、暗象）
  5. 财官取用（通过什么十神获取财官）
  6. 职业定位（食伤=技术，印=学历，比劫=劳动）
  7. 累度分析（化、合、制三种作用）
  8. 十神关系分析
  9. 做功方式（比去财、伤去官、食制杀、印制食、财制印）
  10. 综合取象断事
- 特色/亮点：**完全抛弃旺衰和用神概念**。"盲派体系里根本就没有日主旺衰和用神这些概念，格局也基本不用"。核心看三件事：结构、做功、取象。五大工具：宾主、体用、贼神捕神、正用反用、正局反局。

### 来源 8：子平真诠格局法论命流程

- URL：https://www.suanzhun.net/article/2204.html + 格局法搜索结果综合
- 流派：格局派（子平真诠体系）
- 步骤数：6 步
- 步骤列表：
  1. **月令取格** — "八字用神，专求月令"。透干取格优先：主气透干 → 余气透干 → 本气定格。杂气月令（辰戌丑未）特殊处理
  2. **定相神** — 找辅佐用神成格之物。四吉神顺用（保护性使用），四凶神逆用（制化性使用）
  3. **看成败（过三关）** — 护卫（保护/抑制机制是否完善）→ 真假（有根为真，无根为假）→ 清纯（用神相神单一为清）
  4. **定格局高低** — 真神得用为最佳，驾驭方法得当则富贵
  5. **看大运对格局的影响** — 大运是否有利于格局成格或破格
  6. **流年应期** — 具体事件的时间节点判断
- 论命主线：五行 → 干支 → 十神 → 用神 → 相神 → 格局 → 行运
- 特色/亮点：**以月令为绝对中心**。"月令无法取用时再求他柱"。不关心日主旺衰，只看月令格局的成败。保护优先级：相神 > 用神 > 日主。"真神得用，得之必富贵"。

## 二、各版本共通步骤（所有版本都有的）

以下步骤在所有流派和版本中均出现，可视为**八字分析的必做步骤**：

| 序号 | 共通步骤 | 所有版本均认同的做法 |
|------|----------|---------------------|
| 1 | **排盘** | 排出年月日时四柱天干地支 |
| 2 | **确定日主** | 以日柱天干为命主本人 |
| 3 | **看月令** | 月支/月令是分析的核心锚点，所有流派都认为月令占最大权重 |
| 4 | **定格局或定旺衰** | 虽然先后不同，但都必须做（格局派先格局，旺衰派先旺衰） |
| 5 | **确定用神/喜忌** | 所有流派都需要找到命局的关键五行/十神 |
| 6 | **看大运** | 分析人生各阶段的运势起伏 |
| 7 | **看流年** | 具体年份的吉凶判断 |
| 8 | **断具体事项** | 事业、财运、婚姻、健康等主题分析 |

## 四、"最大公约数"版本：综合分析 SOP

综合 12 个来源的共识，以下是一个覆盖各流派核心的**通用八字分析 SOP**：

### Phase 0：排盘准备

| 步骤 | 操作 | 要点 |
|------|------|------|
| 0.1 | 排出四柱八字 | 年柱、月柱、日柱、时柱 |
| 0.2 | 标注天干地支五行 | 金木水火土 |
| 0.3 | 标注十神 | 以日干为中心推算其他七字的十神 |
| 0.4 | 标注地支藏干 | 本气、中气、余气 |
| 0.5 | 排大运 | 确定起运年龄和每步大运干支 |
| 0.6 | 检查空亡 | 查看日柱空亡的地支 |

### Phase 1：宏观初览（1-2 分钟）

| 步骤 | 操作 | 要点 |
|------|------|------|
| 1.1 | 四柱平扫 | 远观财官印的分布，注意有无伤官见官、财星坏印等明显冲突 |
| 1.2 | 地支关系速查 | 有无六冲（子午、卯酉等）、三合、三会、三刑、六害 |
| 1.3 | 五行缺否速查 | 快速统计五行数量，注意缺失五行 |
| 1.4 | 气候速判 | 出生月份的寒暖燥湿：冬生偏寒、夏生偏热 |
| 1.5 | 形成第一印象 | 这个八字大致是什么调性？偏强偏弱？有无明显特征？ |

### Phase 2：核心定位（核心步骤）

| 步骤 | 操作 | 具体做法 |
|------|------|----------|
| 2.1 | **聚焦日干与月令** | 确定日主五行；看月支本气是什么十神；日干在月令处于旺相休囚死的哪个状态 |
| 2.2 | **定旺衰** | (a) 得令否：日主在月令的状态 (b) 得地否：日支、年支、时支是否有根 (c) 得势否：天干有无印比帮扶 (d) 综合判定：身强/身弱/从格 |
| 2.3 | **定格局** | (a) 月令本气透干否？→ 透干取格 (b) 本气未透，余气透干否？→ 余气取格 (c) 均未透 → 本气定格 (d) 月令比劫 → 另取格局 (e) 判断是否从格/特殊格局 |
| 2.4 | **看调候** | 冬生是否有火？夏生是否有水？查《穷通宝鉴》对应十干调候表 |
| 2.5 | **取用神** | 综合旺衰、格局、调候三方面确定用神：(a) 旺衰用神：扶抑平衡 (b) 格局用神：月令取用+相神 (c) 调候用神：寒暖燥湿调节。三者一致则强，矛盾则需权衡 |
| 2.6 | **定喜忌** | 用神确定后：生助用神者为喜神，克制用神者为忌神，生助忌神者为仇神 |

### Phase 3：格局成败与层次判定

| 步骤 | 操作 | 具体做法 |
|------|------|----------|
| 3.1 | **看成败** | 格局是否成立？过"护卫-真假-清纯"三关 |
| 3.2 | **找病药** | 八字最大的"病"是什么字？有无"药"来克制？有病有药为贵 |
| 3.3 | **定层次** | 格局高低 + 病药关系 + 调候满足度 → 综合判断命局层次 |

### Phase 4：四柱逐柱细读

| 步骤 | 操作 | 具体做法 |
|------|------|----------|
| 4.1 | **年柱** | 祖上/家境/幼年运（1-16 岁），看纳音、神煞 |
| 4.2 | **月柱** | 父母/兄弟/青年运（17-32 岁），格局核心，事业宫 |
| 4.3 | **日柱** | 自身/配偶/中年运（33-48 岁），日支=配偶宫 |
| 4.4 | **时柱** | 子女/晚年运（49 岁后），归宿和结局 |
| 4.5 | **干支关系** | 天干合克关系 + 地支合冲刑害关系逐一分析 |

### Phase 5：主题专项断事

| 主题 | 看什么 | 具体方法 |
|------|--------|----------|
| **事业** | 官杀星 + 月柱（事业宫）+ 格局层次 | 官杀为用则仕途好；食伤生财则适合经商；印星旺则适合文职学术 |
| **财运** | 财星 + 食伤（生财源）+ 比劫（劫财风险） | 财星有根为真财；食伤生财财源不断；比劫重则破财 |
| **婚姻** | 配偶星 + 配偶宫（日支）| 男看财星，女看官星；日支为配偶宫喜静不喜动；冲穿不利婚姻 |
| **健康** | 五行偏枯 + 被克最重的五行 | 缺某五行则相应脏腑易病；用神受克健康堪忧 |
| **子女** | 食伤星 + 时柱（子女宫）| 食神=女，伤官=男（男命反之）；时柱坐枭印不利子女 |
| **学业** | 印星 + 文昌贵人 + 华盖 | 印星有力学业好；官印相生利仕途考试 |
| **性格** | 日主五行 + 十神配置 + 调候 | 日主强弱决定外向/内敛；十神组合决定性格特质 |

### Phase 6：大运流年动态分析

| 步骤 | 操作 | 具体做法 |
|------|------|----------|
| 6.1 | **大运逐步分析** | 每步大运（10 年）的天干地支与原局的互动：是否帮用神/克用神/成格/破格 |
| 6.2 | **关键流年标记** | 在每步大运中找出与原局有刑冲合害关系的流年 |
| 6.3 | **断应期** | (a) 命局断应事：原局决定会发生什么 (b) 大运断阶段：何时进入好运/坏运 (c) 流年断应期：具体哪一年应验。宫位也有时间含义：年柱幼少年，月柱青年，日柱中年，时柱晚年 |
| 6.4 | **岁运并临** | 大运与流年同柱时注意重大事件 |

### Phase 7：综合评判

| 步骤 | 操作 |
|------|------|
| 7.1 | 交叉验证各维度结论，找共同点和矛盾点 |
| 7.2 | 给出整体命运基调（富贵贫贱、吉凶寿夭） |
| 7.3 | 针对命主具体问题给出分析和建议 |
| 7.4 | 标注确定性等级（强信号/弱信号/矛盾信号） |

## 五、关键子操作详解

### "定旺衰"的子步骤

1. **看月令（得令否）** — 日主在月令是旺/相/休/囚/死？月令独占 40-50% 权重
2. **看日支（得地否）** — 日支是否为日主的根？占 15-20% 权重
3. **看通根** — 日主天干在四柱地支中有无根气（同类藏干）
4. **看帮扶** — 印星和比劫的数量和力度
5. **看克泄耗** — 官杀（克）、食伤（泄）、财星（耗）的力量
6. **综合判定** — 得令+得地+得势 → 身强；失令+失地+失势 → 身弱；极端则从格

### "取格局"的子步骤

1. **看月令本气** — 子月=癸水，寅月=甲木，午月=丁火...
2. **看透干** — 本气是否透出天干？优先透干取格
3. **看余气透干** — 本气未透时查中气、余气
4. **定格局名** — 以透出的十神命名：正官格、七杀格、正财格...
5. **看成败** — 护卫(是否有保护)、真假(是否有根)、清纯(是否单一)
6. **找相神** — 辅佐用神成格的关键十神

### "断应期"的子步骤

1. **以大运为基准确定应年** — 大运天干地支与原局互动
2. **以年为基准确定应月** — 流月与流年/原局的关系
3. **以月为基准确定应日** — 进一步缩小时间范围
4. **关键判断指标** — 刑冲合害出现的时间点；用神/忌神被引动的时间；宫位被冲动的流年

## 六、各流派分析顺序一览对照

```
格局派（子平真诠）：
  月令 → 透干取格 → 定相神 → 看成败(护卫/真假/清纯) → 定层次 → 看大运 → 断流年

旺衰派（新派/现代主流）：
  日主 → 定旺衰(得令/得地/得势) → 取用神(扶抑) → 定喜忌 → 看大运 → 断流年 → 推六亲/事业/婚姻

盲派：
  看结构 → 定宾主 → 看做功(谁制谁) → 取象(干支之象) → 断事(直断) → 看应期

古法（三命通会）：
  月令 → 日干 → 天干(表) → 地支(里) → 十神组合 → 旺衰(贯穿) → 气象化气

韦千里八步：
  看强弱 → 定格局 → 取用神 → 论喜忌 → 查岁运 → 推六亲 → 评性情 → 断事业

陆致极三阶段：
  概念建立 → 静态分析(多视角) → 动态分析(大运流年) → 实务应用(投射人生)
```

## 七、核心发现与结论

### 1. 排盘后第一步看什么？

各流派答案不同：
- **格局派**：先看月令。"大凡看命，先看月支有无财官，方看其他"
- **旺衰派**：先看日主旺衰。"正确理解日干旺衰是批准八字的第一步"
- **盲派**：先看整体组合结构。"打开八字看组合"
- **综合派**：先四柱平扫全局，再聚焦日干与月令
- **调候派**：先辨气候（寒暖燥湿）

### 2. 所有流派的共识

- 月令是最重要的位置（40-50% 权重）
- 日主是分析的参照中心
- 必须看大运流年的动态变化
- 最终要落到具体主题（事业/财运/婚姻/健康）的实际断语

### 3. 最大的分歧

- **旺衰派 vs 格局派**：是先定旺衰还是先定格局？旺衰派认为旺衰是基础；格局派认为格局才是核心，旺衰只是辅助
- **盲派 vs 其他所有派**：盲派完全抛弃了旺衰和用神概念，用做功和取象替代

### 4. 实战建议

根据多源交叉验证，实战中最常用的综合分析顺序为：
1. 排盘 + 标注十神
2. 宏观平扫（快速形成第一印象）
3. 聚焦日干与月令（定旺衰 + 定格局并行）
4. 看调候
5. 取用神（综合三维）
6. 逐柱细读 + 干支关系
7. 主题断事
8. 大运流年动态分析
9. 综合评判
