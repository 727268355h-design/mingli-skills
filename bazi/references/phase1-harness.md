# Phase 1：宏观初览 — 执行指令

> 前置：Phase 0 排盘数据卡已完成
> 输入：Phase 0 输出的结构化排盘数据卡
> 输出：初览标记卡（下方模板）
> 原则：**只标记不判断，只提问不回答**。这一步是"提出问题"，不是"给出答案"。
> 耗时：2 分钟快速扫描

---

## 1.1 四柱平扫 — 十神分布标记

从 Phase 0 排盘卡的天干十神中，做以下检查：

**检查项 A：天干十神组合冲突**

逐一扫描，命中则标记 🔴：

| 冲突组合 | 条件 | 标记 |
|---------|------|------|
| 伤官见官 | 天干同时出现伤官和正官 | 🔴 伤官见官 |
| 官杀混杂 | 天干同时出现正官和七杀 | 🔴 官杀混杂 |
| 财星坏印 | 天干财星与印星紧贴（相邻柱） | 🔴 财星坏印 |
| 枭神夺食 | 天干偏印与食神紧贴 | 🔴 枭神夺食 |
| 比劫争财 | 天干比劫与财星紧贴 | 🔴 比劫争财 |

**检查项 B：十神分布偏态**

统计天干+藏干本气中各十神出现次数：

```
官杀系（正官+七杀）：X 个
印系（正印+偏印）：X 个
财系（正财+偏财）：X 个
食伤系（食神+伤官）：X 个
比劫系（比肩+劫财）：X 个
```

标记偏态：某系 ≥ 3 个标 🟡偏重，某系 = 0 标 🟡缺失。

---

## 1.2 地支关系速查 — 逐对扫描

将四柱地支列出（年支/月支/日支/时支），逐对检查：

**共 6 对需检查**：年-月、年-日、年-时、月-日、月-时、日-时

对每一对，依次查：

| 关系 | 查法 | 发现则标记 |
|------|------|-----------|
| 六冲 | 子午/丑未/寅申/卯酉/辰戌/巳亥 | 🔴 [X][Y]冲 |
| 六合 | 子丑/寅亥/卯戌/辰酉/巳申/午未 | 🟢 [X][Y]合 |
| 六害 | 子未/丑午/寅巳/卯辰/申亥/酉戌 | 🟡 [X][Y]害 |
| 三刑 | 寅巳申/丑未戌/子卯（需三柱凑齐或流年补齐） | 🔴 三刑[类型] |
| 自刑 | 辰辰/午午/酉酉/亥亥 | 🟡 [X]自刑 |

**再查三合/三会**（需三柱地支配合）：

| 关系 | 组合 | 化五行 |
|------|------|--------|
| 三合 | 申子辰→水、亥卯未→木、寅午戌→火、巳酉丑→金 | 标记合局 |
| 三会 | 亥子丑→水、寅卯辰→木、巳午未→火、申酉戌→金 | 标记会局 |
| 半合 | 两柱凑齐三合局的任意两支 | 标记半合 |

**暗合速查**（紧贴两柱之间）：
- 寅丑暗合（甲己+丙辛+戊癸全合）
- 卯申暗合（乙庚合）
- 午亥暗合（丁壬+甲己合）

---

## 1.3 五行缺否速查

统计四柱八字中五行数量（天干 + 地支本气）：

```
木：X 个（天干甲乙 + 地支寅卯）
火：X 个（天干丙丁 + 地支巳午）
土：X 个（天干戊己 + 地支辰戌丑未）
金：X 个（天干庚辛 + 地支申酉）
水：X 个（天干壬癸 + 地支亥子）
```

标记：
- 某行 = 0 → 🟡 缺[五行]（需查藏干中气余气是否有补）
- 某行 ≥ 4 → 🟡 [五行]偏旺

---

## 1.4 气候速判

根据月支（出生月份）判断气候：

| 月支 | 季节 | 气候属性 | 调候紧迫度 |
|------|------|---------|-----------|
| 亥、子 | 冬 | 寒 | 急需火暖 🔴 |
| 丑 | 冬末 | 寒湿 | 需火暖 🔴 |
| 寅、卯 | 春 | 温 | 一般不急 ⚪ |
| 辰 | 春末 | 温湿 | 视情况 ⚪ |
| 巳、午 | 夏 | 热 | 急需水润 🔴 |
| 未 | 夏末 | 热燥 | 需水润 🔴 |
| 申、酉 | 秋 | 凉 | 一般不急 ⚪ |
| 戌 | 秋末 | 燥 | 视情况 🟡 |

标记气候：`[寒/温/热/凉] + [调候紧迫度]`

---

## 1.5 形成第一印象 — 标记异常点

基于 1.1-1.4 的标记，列出所有 🔴 和 🟡 项目作为「待验证问题清单」。

**不下结论。** 只提出问题，例如：
- "🔴 伤官见官 — Phase 2 需确认是否有印星通关"
- "🔴 月日支卯酉冲 — Phase 5 需详细分析此冲的影响"
- "🟡 缺火 + 冬生 — Phase 2 需重点查调候"
- "🟡 官杀系偏重(3个) — Phase 2 需看是否官杀混杂"
