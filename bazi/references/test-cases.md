## 案例 1：身强正官格

**四柱：** 乙卯 丁亥 丁未 庚戌
**性别：** 男（未明确标注，按通用论）
**来源已知结论：**
  - 旺衰：身强（丁火得乙木生、未戌中火土助力）
  - 格局：正官格（亥中壬水本气为丁火正官，月令藏官不透，仅一位官星）
  - 用神：财星庚金（身旺喜财生官）
  - 调候：亥月水旺，丁火有乙木相生不惧寒，调候基本满足
  - 整体评价：标准正官格，官星纯一，时柱庚金正财配官，格局清正
**来源URL：** https://www.shenjige.cn/details/AHHzuUA68.html
**边缘特征：** 官星藏于月支不透天干，需要判断"藏官是否成格"的边界问题；丁亥月柱天干丁火与日主同为丁火，可能干扰旺衰判定。

## 案例 3：从财格（子平真诠经典例）

**四柱：** 庚申 乙酉 丙申 乙丑
**性别：** 男
**来源已知结论：**
  - 旺衰：从格（丙火日主生于酉月，地支三金一土生金，天干乙庚合化金，丙火完全无根无助）
  - 格局：真从财格
  - 用神：财星金（顺从财势）
  - 调候：不适用（从格不论调候）
  - 整体评价：大富之命（"王十万"），从财格纯粹，财星得势
**来源URL：** https://blog.sina.com.cn/s/blog_3f28a6260100von3.html / 《子平真诠评注》
**边缘特征：** 天干乙庚合是否化金是关键判定点；丙火坐申绝地+酉月死地，"真从"还是"假从"的临界判断；乙木两透是否为日主残余生源（若论乙生丙则非从格）。SOP 必须处理"合化是否成立"的逻辑。

## 案例 7：冬生命 -- 调候优先（乙木丑月）

**四柱：** 辛卯 辛丑 乙卯 己卯
**性别：** 女
**来源已知结论：**
  - 旺衰：身强（乙木三卯通根极旺，虽丑月土旺但木有强根）
  - 格局：偏印格（辛金偏印两透）
  - 用神：丙火（调候第一优先，寒谷之木专用丙火）
  - 调候：严重不满足（四柱无丙火，全局寒湿，调候用神缺失）
  - 整体评价：身强格局尚可但调候严重不足。大运寅、巳、丙、午运为佳；癸、辰运为衰。命局虽然不差但因缺丙火暖局，富贵层次受限
**来源URL：** https://www.suanzhun.net/article/2248.html
**边缘特征：** 典型的"身强但调候不满足"案例，SOP 必须优先处理调候逻辑；辛金两透是否论偏印成格还是群印夺食需判断；身强+调候缺失的矛盾如何在最终评价中权衡。

## 使用说明

### 测试流程
1. 将四柱输入 SOP 系统，不提供性别以外的任何提示
2. 对比 SOP 输出与"来源已知结论"
3. 重点检查"边缘特征"描述的争议点是否被正确识别和处理

### 评分维度
| 维度 | 权重 | 说明 |
|------|------|------|
| 旺衰判定 | 25% | 身强/身弱/从格 是否与来源一致 |
| 格局判定 | 25% | 格局名称是否匹配 |
| 用神选取 | 20% | 用神五行是否一致 |
| 调候判定 | 15% | 满足/不满足 是否正确 |
| 边缘识别 | 15% | 是否识别出边缘特征并给出警告/多解读 |

### 预期通过标准
- 案例 1-2（基础正格）：旺衰+格局+用神必须全部正确
- 案例 3-5（特殊格局）：格局类型必须正确识别，从格/化格/专旺不能误判为正格
- 案例 6（女命）：必须触发女命专论路径
- 案例 7-8（调候）：调候满足度必须正确，且调候用神优先级必须体现
- 案例 9（官杀混杂）：必须识别并标注官杀混杂
- 案例 10（伤官见官）：必须触发多派系模式，给出至少两种解读

## 案例 A：失令+得地+得势（三得法组合5验证）

### 案例信息

```
四柱：壬寅 庚辰 甲子 甲子
性别：男
来源结论：身旺，正官格，用神庚金
来源URL：https://blog.csdn.net/qingyang789/article/details/134518135 + 综合多源构建
边缘特征：甲木生辰月（四季土月），日主失令（木在辰月为囚），但坐子水（印星得地），
         天干双甲+壬水印星，地支寅木禄根+子水生身，比劫印星众多 → 失令但得地+得势，
         三得法组合5边缘场景
```

### Phase 2 核心定位

```
━━━━━━━━━━ 核心定位卡 ━━━━━━━━━━

【日主】
甲（阳木）—— 参天大树，刚直正义、宁折不弯

【月令】
月支 = 辰，本气 = 戊土（偏财）
日主甲木在辰月：囚 → 失令 ✗
当令藏干：辰中藏 乙木(余气9天) 癸水(中气3天) 戊土(本气18天)

【旺衰六级判定】
a. 月令：失令 ✗（木在四季土月为囚）
b. 日支：得地 ✓（日支子水藏癸水，癸水为正印，生甲木）
c. 通根：
   年支 寅 藏干：甲木(本气)、丙火(中气)、戊土(余气) → 有甲木本气根 ✓
   月支 辰 藏干：戊土(本气)、乙木(余气)、癸水(中气) → 有乙木余气根 ✓（弱）
   日支 子 藏干：癸水(本气) → 无木根 ✗（但癸水印星生身）
   时支 子 藏干：癸水(本气) → 无木根 ✗（但癸水印星生身）
   通根总结：共2处通根（寅木本气强根+辰中乙木余气弱根），根气中等

d. 帮扶力量统计：
   印星（生我者）：壬水（年干，透出）= 大帮；子水x2（日支+时支，本气癸水）= 中帮x2
   比劫（同我者）：甲木（时干，透出）= 大帮；寅木（年支，甲木本气）= 中帮
   帮扶总力量：强（印2透+2藏 + 比劫1透+1藏 = 6处帮扶）

e. 克泄耗力量统计：
   官杀（克我者）：庚金（月干，七杀）= 1个
   食伤（我生者=泄）：无
   财星（我克者=耗）：戊土（辰中本气偏财）= 1个（藏干，力弱）
   克泄耗总力量：弱（仅庚金1透干 + 戊土1藏干 = 2处）

   得势速判：帮扶6 - 克泄耗2 = +4 ≥ 2 → 得势 ✓

f. ▸ 综合判定：
   三得法组合：得令 ✗ + 得地 ✓ + 得势 ✓ → 组合5
   组合5判定：中和偏强 ⚠️ 失令但得地得势仍可身旺

   实际上此造帮扶力量远超克泄耗（+4差值），壬水透干生身+双子水印星+
   寅木强根+时干甲木比肩 → 判定偏强（接近偏强上限）

   ▸ 综合旺衰判定 = 【偏强】

【格局】
取格过程：月支辰，本气戊土（偏财）→ 戊土未透干（年月时干为壬庚甲，无戊己）→
         中气癸水 → 癸水未透（壬水非癸水）→ 余气乙木 → 乙木未透 →
         均不透，以月令本气定格 → 偏财格

         ⚠️ 但月干庚金（七杀）透出且有力，实际取格应考虑：
         庚金七杀透月干，虽非月令藏干，但为月令辰中所生（土生金），
         且紧贴月令，可参取七杀格。

         综合取格：以偏财格为主，兼看七杀（庚金透月干）

▸ 格局 = 【偏财格】（月干透七杀，偏财格兼杀）
成败判定：身强财弱，劫比重财轻 → 成中有败（破格因素：比劫重夺财）
         但庚金七杀制比劫，有救应 → 格局可用
相神 = 庚金（七杀制比劫护财）

【调候】
日主甲木 + 辰月（三月春末）→ 调候用神 = 庚金（甲木生春末，庚金劈甲引丁）
命局中：有庚金，位置在月干透出，力量中等（辰土生金有根）
▸ 调候满足度 = 【充分满足】

【天干合化】
甲己合？→ 无己土，不合
乙庚合？→ 无乙在天干，不合
壬与甲之间无合关系
天干无合

【对比来源结论】
来源结论：身旺 → Phase 2 判定：偏强（身旺）✅ 一致
来源结论：用神庚金 → Phase 2 判定：庚金为调候+制比劫护财 ✅ 一致

━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**边缘验证结论**：✅ 三得法组合5成立。甲木辰月失令，但寅木本气强根（得地）+ 壬甲双透+双子水印（得势），帮扶力远超克泄耗，证实"失令+得地+得势 → 仍可身旺"。

**边缘验证结论**：✅ 空亡对命局的实质影响得到验证。三支空亡（年月时支）导致六亲缘薄、财运虚空。特别是寅卯（木=财星）全空 → 财星虽多却虚 → 命主反贫。空亡不改变旺衰判定本身，但深刻影响实际命运吉凶表现。身极弱+财多+财空 = 贫困残疾的典型组合。

## 综合测试总结

| 案例 | 边缘类型 | 三得法组合 | 旺衰判定 | 格局 | 与来源一致性 |
|------|---------|-----------|---------|------|-------------|
| A | 失令+得地+得势 | 组合5 | 偏强 | 偏财格 | ✅ 一致 |
| B | 假从格（天干杂质） | 组合8→从格 | 假从杀 | 从杀格(假) | ✅ 一致 |
| C | 真太阳时跨界 | 组合4（两版本同） | 中和偏强 | 建禄格（十神变） | ✅ 时柱差异验证 |
| D | 空亡多柱 | 组合8 | 极弱（不从） | 偏财格（破格） | ✅ 一致 |

### 关键发现

1. **组合5验证成功**：失令不等于身弱。当得地+得势的帮扶力量足够大（差值>=4），即使月令不帮，日主仍可判偏强。月令虽占40-50%，但不是绝对一票否决。

2. **假从格判定边界清晰**：日主完全无根（地支无任何水的藏干） + 天干有同类透出（年干癸水）→ 假从。关键区分点是"天干有无同类透出"，而非"有无余气根"（本案例连余气根都没有，更极端）。

3. **真太阳时改变的不是旺衰方向而是格局细节**：组合判定可能不变，但十神属性（正官vs七杀）、天干合化（有vs无）、性格/事业推断会根本不同。边界时辰（北京时间在整点附近且出生地偏西30度以上）必须做真太阳时校正。

4. **空亡是旺衰之外的独立维度**：空亡不改变三得法判定（不影响得令得地得势），但直接影响命运吉凶。财星全空 = 有财如无财。多柱空亡 = 人生多处虚空。
