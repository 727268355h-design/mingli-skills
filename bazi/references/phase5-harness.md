**执行步骤**：

1. 列出四柱地支：年[X] 月[X] 日[X] 时[X]
2. 逐对检查 6 对（年月/年日/年时/月日/月时/日时）是否命中六合
3. 命中则判断：
   - 是否合化？（化神得月令 + 紧贴）
   - 合化成功 → 以化后五行论
   - 合而不化 → 合绊（双方力量各减约30%）
   - 合住用神 → 凶（用神被绊）
   - 合住忌神 → 吉（忌神被控）

```
六合检查：

```

4. 检查半合/三合是否被冲破（旺支被冲则合局不成）


## 5.7 空亡

从 Phase 0 已确定空亡地支，此处复查并分析影响：

**空亡作用规则**：

| 条件 | 效果 |
|------|------|
| 年支/月支空亡 | 假空（仍有力量，暂时休眠，逢大运流年填实时爆发） |
| 时支空亡 | 真空（力量大减） |
| 日支不论空亡 | 日主自坐不空 |
| 空亡逢冲 | 冲实（空亡被激活，力量恢复） |
| 空亡逢合 | 合实（解空，力量恢复） |
| 空亡地支为用神 | 用神暂时不发力 → ⚠️ 重点关注填实之年 |
| 空亡地支为忌神 | 忌神暂不为害 → 暂时吉利，但填实之年反而凶 |


## 5.8 墓库

**墓库对照表**：

| 墓库 | 所藏五行 | 弱为墓/旺为库 |
|------|---------|-------------|
| 辰 | 水墓/库 | 水弱入辰为墓（困死）；水旺入辰为库（储存） |
| 戌 | 火墓/库 | 火弱入戌为墓；火旺入戌为库 |
| 丑 | 金墓/库 | 金弱入丑为墓；金旺入丑为库 |
| 未 | 木墓/库 | 木弱入未为墓；木旺入未为库 |

**湿土燥土区分表**（极重要，不可混淆）：

| 土 | 属性 | 对火 | 对金 | 对水 |
|----|------|------|------|------|
| 辰 | 湿土 | 晦火（减弱火力）| 生金 | 蓄水 |
| 丑 | 湿土 | 晦火 | 生金 | 蓄水 |
| 戌 | 燥土 | 不泄火（助火势）| 煅金（脆金）| 克水 |
| 未 | 燥土 | 不泄火 | 煅金 | 克水 |

> ⚠️ **辰丑湿土和戌未燥土对五行的作用完全不同，这是常见错误点。**

**墓库开合条件**：

| 条件 | 效果 |
|------|------|
| 墓库逢冲 | 开库（辰戌冲/丑未冲）→ 库中之物释出 |
| 墓库逢刑 | 开库（丑戌未三刑）→ 库中之物激活 |
| 透干引出 | 库中藏干透出天干 → 自然出库 |
| 墓库逢合 | 合住 → 库门关闭，库中之物不出 |

**执行步骤**：

1. 检查四柱中有无辰戌丑未
2. 判断对应五行是弱（墓）还是旺（库）
3. 检查是否有开库条件
4. 特别关注：财库（与财运相关）、官库（与事业相关）

```
墓库检查：
[X]为[五行][墓/库]（[该五行在命局中旺/弱]）
开库条件：[有/无]（[逢冲/逢刑/透干]）
```

## 5.10 伏吟反吟

**伏吟 = 大运/流年出现与原局相同的干支（同柱重现）**
**反吟 = 大运/流年出现与原局天克地冲的干支**

| 类型 | 定义 | 象征 | 含义 |
|------|------|------|------|
| 伏吟 | 同干支重现 | 悲泣之象 | 重复、纠结、痛苦、丧事 |
| 反吟 | 天克地冲 | 反复之象 | 变动、翻覆、突发、冲击 |

**原局内伏吟**：四柱中是否有两柱干支相同（如年柱甲子、时柱甲子）。
**原局内反吟**：四柱中是否有两柱天克地冲（如甲子与庚午）。

**重点标记**（⚠️ 高危信号）：

| 伏吟/反吟位置 | 含义 |
|-------------|------|
| 日柱伏吟 | 本人重大事件/丧事 |
| 日柱反吟 | 本人突变/婚变 |
| 时柱伏吟 | 子女相关忧伤 |
| 年柱伏吟 | 长辈相关忧伤 |

```
伏吟检查（原局内）：[有/无] → [哪两柱]
反吟检查（原局内）：[有/无] → [哪两柱]
需关注的大运/流年伏吟反吟：[列出关键年份]
```

## 地支关系力量排序总表

**力量从大到小**：

```
三会 > 三合 > 含旺支半合 > 六冲 > 六合 > 三刑 > 六害 > 不含旺支半合 > 暗合 > 六破
```

> ⚠️ 合冲同时出现时：主流取合冲并存，不可机械认为一方解另一方。判断依据：
> 1. 紧贴度：紧贴的力量 > 隔柱
> 2. 旺衰：旺方主导
> 3. 层次：流年 > 大运 > 原局
