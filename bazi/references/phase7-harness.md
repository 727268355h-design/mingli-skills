# Phase 7：大运流年动态分析 — 执行指令

> 前置：Phase 6 专项断事卡已完成

## 7.1 大运逐步分析

从 Phase 0 大运列表中，逐步大运分析。**每步大运回答以下五个核心问题**：

| 核心问题 | 判断标准 |
|---------|---------|
| Q1：是否帮用神？ | 大运干支五行是否为用神/喜神 |
| Q2：是否克用神？ | 大运干支五行是否为忌神/仇神 |
| Q3：是否修复"病"？ | 大运干支是否为 Phase 3 确定的"药" |
| Q4：是否增强相神？ | 大运干支是否生助相神 |
| Q5：是否补调候？ | 大运干支是否为调候用神 |

**天干管前 5 年，地支管后 5 年**（⚠️ 这是近似规则，非绝对分界）：

| 大运阶段 | 主看 |
|---------|------|
| 大运前半（前5年） | 大运天干为主，地支为辅 |
| 大运后半（后5年） | 大运地支为主，天干为辅 |

## 7.2 关键流年标记

**在每步大运中，找出以下类型的关键流年并标记**：

### 关键流年筛选条件

| 条件 | 标记级别 | 说明 |
|------|---------|------|
| 流年与原局有六冲 | 红标 | 变动之年（冲什么宫位就动什么事） |
| 流年与原局有三刑 | 红标 | 刑伤之年（三刑齐聚尤甚） |
| 流年与原局合（六合/三合） | 黄标 | 合动之年（合什么宫位就动什么事） |
| 流年与原局有六害 | 黄标 | 暗伤之年 |
| 岁运并临（大运=流年同柱） | 红标 | 重大事件之年 ⚠️ 不一定凶，但事大 |
| 伏吟（流年=原局某柱） | 红标 | 悲泣/重复之年 |
| 反吟（流年天克地冲原局某柱） | 红标 | 突变/翻覆之年 |

| 流年为忌神且旺 | 红标 | 坏年份 |
| 流年透出配偶星+冲合日支 | 黄标 | 婚动之年 |
| 流年透出官星 | 黄标 | 事业变动之年 |
| 流年逢财库冲开 | 黄标 | 财运大变之年 |

**标记格式**：


## 7.3 断应期 7 步

**应期 = 事件发生的具体时间。** 按以下 7 步逐级缩小时间范围：

### 步骤 1：宫位划分定年龄段

| 宫位 | 年龄段 | 人生阶段 |
|------|--------|---------|
| 年柱 | 1-16 岁 | 幼年 |
| 月柱 | 17-32 岁 | 青年 |
| 日柱 | 33-48 岁 | 中年 |
| 时柱 | 49 岁及以后 | 晚年 |

```
相关事件涉及[X]宫位 → 基准年龄段 = [X-X岁]
```

### 步骤 2：先定大运基准

```
当前/相关大运 = [干支]（[年龄段]）
大运性质 = [喜用运/忌仇运/平运]
好运推迟凶事，坏运加速凶事
```

### 步骤 3：以年定月，以月定日

```
确定流年后 → 看流月（月支与原局关系）→ 进一步缩小到月份
```

### 步骤 4：信息大小定应验速度

| 信息强度 | 应验速度 |
|---------|---------|
| 用忌神力量悬殊（信息大） | 应事早（开年即应） |
| 用忌神力量接近（信息小） | 应事迟（下半年应） |

### 步骤 5：前移与后推条件

| 条件 | 效果 |
|------|------|
| 限运用神有力 + 行喜用运 | 吉事前移 |
| 限运为忌但行用神大运 | 凶事后推 |
| 限运为忌 + 行忌神大运 | 凶事加速前移 |

### 步骤 6：虚实作用判断

| 作用 | 效果 |
|------|------|
| 实忌制实用 | 大凶（真正的打击） |
| 实用制实忌 | 大吉（真正的收获） |
| 虚忌制实用 | 小凶（虚惊/小损失） |
| 虚用制实忌 | 小吉（暗中有助） |

> **实神** = 有根有力的干支；**虚神** = 无根无力的干支

### 步骤 7：引动机制确认

| 引动方式 | 触发条件 |
|---------|---------|
| 冲 | 流年地支冲原局某支 → 该支代表的事项被引动 |
| 合 | 流年地支合原局某支 → 该支代表的事项被合动 |
| 会 | 流年补齐三会局 → 该五行力量暴增 |
| 刑 | 流年补齐三刑 → 刑伤事件触发 |
| 穿（害） | 流年穿原局某支 → 暗中受伤 |

```
引动"病药"对应五行时 → 事件发生
引动宫位时 → 对应人生领域出事
```

| 学业提升 | 印星得力之年（逢考试年份） |
| 贵人相助 | 天乙贵人被引动之年 |

**大运流年作用原则**：

| 原则 | 说明 |
|------|------|
| 流年为君 > 大运为臣 > 命局为民 | 流年权力最大，可改变大运趋势 |
| 大运影响力 > 流年影响力 | 十年趋势比一年波动重要 |
| 好大运+坏流年 | 小挫折不成大灾 |
| 坏大运+好流年 | 小改善不成大运 |
| 断事四法递进 | 象法定性(什么事) → 理法定量(多大事) → 应期定时(何时) → 全局定果(什么结果) |
