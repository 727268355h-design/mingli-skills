## 10.1 三命通会日时断语查询 🟢查表

> 来源：调研-古籍断语查表系统.md 一.1.1
> 数据规模：720条（10天干 x 6日柱 x 12时辰）

### 查询方法

```
操作：
1. 从 Step 0 取日柱天干 → 确定"六X日"组（如甲日 → 六甲日）
2. 从 Step 0 取时柱干支 → 确定时辰（如甲子时）
3. 定位路径：六[X]日[时干支]时断 → 在该条下找日柱对应的逐日断语
4. 查表键 = 日柱 + 时柱 → 唯一定位1条断语

示例：
  日柱 = 甲子，时柱 = 甲子时
  → 查"六甲日甲子时断" → 找"甲子日"条目
  → 断语："子遥巳格，年月无庚辛申酉，丑绊午冲，离祖自立，贵..."
```

### 断语解读框架

每条断语通常包含以下维度，逐一提取：

| 维度 | 提取内容 | 与现代分析对应 |
|------|---------|-------------|
| 格局提示 | 断语中的格局名（如"子遥巳格"） | 对照 Step 0 格局判定 |
| 月份分论 | "X月，Y" 形式的月令差异论述 | 对照命主实际出生月 |
| 运程分论 | "行X运，Y" 形式的大运论述 | 对照 Step 6-7 大运分析 |
| 吉凶条件 | "年月有/无XX则吉/凶" | 检查命局是否满足 |
| 六亲论断 | "克妻""子嗣难为"等 | 对照 Step 5 六亲信号 |
| 忌神标注 | "忌X""怕X" | 对照 Step 0 用神忌神 |

```
操作：
1. 查出断语原文（照录，不改写）
2. 按上表逐维度提取断语中的信息
3. 每个维度与 Step 6-9 的对应分析做交叉比对
4. 标注：[印证] / [冲突] / [无关]
5. 冲突项需说明冲突点和可能原因
```

## 10.3 纳音大运法 🟡查表+判定

> 来源：调研-古籍断语查表系统.md 三.3.3
> 定位：辅助验证系统，非主流分析法，推理强度标 🟡

### 核心口诀

> 日柱纳音定真神，四柱纳音看财官。
> 财官无伤有生助，肯定富贵不用断。
> 大运纳音来克伐，恐怕富贵要减半。

### 纳音强弱等级表（同五行内比较）

| 五行 | 由强到弱 |
|------|---------|
| 金 | 剑锋金 > 钗钏金 > 砂石金 > 海中金 > 白蜡金 > 金箔金 |
| 木 | 大林木 > 松柏木 > 石榴木 > 杨柳木 > 桑柘木 > 平地木 |
| 水 | 大海水 > 天河水 > 长流水 > 大溪水 > 涧下水 > 井泉水 |
| 火 | 天上火 > 霹雳火 > 炉中火 > 山头火 > 山下火 > 覆灯火 |
| 土 | 大驿土 > 城头土 > 屋上土 > 壁上土 > 路旁土 > 沙中土 |

### 纳音大运旺衰判断（十二长生位）

| 纳音五行 | 长生 | 旺 |
|---------|------|-----|
| 金 | 巳 | 酉 |
| 水 | 申 | 子 |
| 木 | 亥 | 卯 |
| 火 | 寅 | 午 |
| 土 | 申（寄火） | 午 |

### 纳音生克特殊规则

| 规则 | 说明 |
|------|------|
| 大海水不能生任何纳音之木 | 水太盛则木漂 |
| 长流水、天河水可以生任何纳音之木 | 适量水灌溉 |
| 覆灯火不能克任何纳音之金 | 火太弱 |
| 天上火、霹雳火可以克任何纳音之金 | 大火方能炼金 |
| 大驿土不能克任何纳音之水 | 土虽厚但水大无碍 |
| 纳音上克下为小凶 | 月时攻日有凶残 |

### 换甲运断法

```
操作：
1. 取年柱纳音五行（年命）
2. 取每步大运干支的纳音五行
3. 判断大运纳音对年命纳音的关系：
   - 大运纳音 生 年命纳音 + 大运纳音处生旺状态 → "生运"（吉）
   - 大运纳音 克 年命纳音 + 大运纳音处死墓状态 → "死运"（凶）
   - 大运纳音 与 年命纳音 同五行 → 平（需看旺衰）
4. 与 Step 6-7 的大运评级做交叉比对
5. 标注：[印证] / [冲突] / [无关]
```


## 10.5 交叉验证判定规则

对每个古籍断语与现代分析的比对结果，按以下标准标注：

| 标注 | 定义 | 处理 |
|------|------|------|
| **[印证]** | 古籍断语与 Step 6-9 分析方向一致 | 信心加强，确定性标注升级 |
| **[冲突]** | 古籍断语与 Step 6-9 分析方向相反 | 保留两说，标注"古籍断语与现代分析存在分歧" |
| **[无关]** | 古籍断语涉及的维度在 Step 6-9 未分析 | 作为补充信息记录 |
| **[待查]** | 古籍断语原文无法在当前上下文中获取 | 标注"断语待查，来源：三命通会卷[X]" |

**冲突处理原则**：
- 冲突不代表任何一方错误，八字命理本有流派分歧
- 现代量化分析（Step 1-3）与古籍定性断语服务于不同层面
- 冲突项在 Step 11 综合输出中标注为"⚠️存在分歧"
