## 7.1 流年扫描 9 项硬规则

对每步大运内的 10 个流年，逐一执行以下 9 项检查。每命中一条记录规则编号。

### 检查 a：流年干支五神判定 🟢查表

```
操作：
1. 流年天干 → 判定五行 → 查五神归属（用/喜/忌/仇/闲）
2. 流年地支 → 取本气藏干 → 判定五行 → 查五神归属
3. 输出：流年天干五神=[X]，流年地支五神=[X]

此项本身不触发"关键流年"标记，但为后续检查提供基础数据。
```

### 检查 b：流年与大运四格交叉矩阵 🟢查表

```
操作：
1. 取本步大运的五神归属（用/忌，取主导方）
2. 取本流年的五神归属（用/忌，取主导方）
3. 查四格交叉矩阵（来源：调研-八字用神大运判定硬规则 五.5.3）

| | 流年为用 | 流年为忌 |
|---|---|---|
| 大运为用 | 大吉：双用神叠加 | 多吉少凶：好大运无大坏事 |
| 大运为忌 | 能行难成：坏大运无大好事 | 大凶：双忌神叠加 |

触发关键流年条件：
→ 大吉（用×用）：标记为关键流年，标签=[大吉]，规则编号=[B-用用]
→ 大凶（忌×忌）：标记为关键流年，标签=[大凶]，规则编号=[B-忌忌]

大运基调决定论（来源：调研-八字用神大运判定硬规则 五.5.4）：
- 好大运无大坏事 → 坏流年以"吉中有瑕"论
- 坏大运无大好事 → 好流年以"有机遇但落实难"论

闲神处理规则：
- 大运或流年为闲神时，四格矩阵中按"平"处理（不归入用也不归入忌）
- 闲神不触发关键流年标记（不标记 B-用用 或 B-忌忌）
- 闲神与用/忌组合时：闲×用=小吉（不标关键），闲×忌=小凶（不标关键）
```

### 检查 c：流年与命局天克地冲 🟢查表

```
操作：
1. 流年天干与命局各柱天干逐一检查：是否存在五行相克
2. 流年地支与命局各柱地支逐一检查：是否存在六冲
3. 若同柱（年/月/日/时）同时满足天干相克 + 地支六冲 = 天克地冲

天克地冲效果分级表（来源：调研-八字断事信号硬规则 三.3.2）：

| 流年天克地冲 | 影响对象 | 严重程度 |
|-------------|----------|----------|
| 冲年柱 | 父母、长辈、祖业 | 3星 |
| 冲月柱 | 兄弟姐妹、同辈 | 3星 |
| 冲日柱 | 配偶、婚姻 | 4星 |
| 冲时柱 | 子女、事业晚运 | 3星 |

天克地冲吉凶硬规则（来源：调研-八字断事信号硬规则 三.3.3）：

| 条件 | 判定 |
|------|------|
| 用神冲克忌神 | 以吉断 |
| 忌神冲克用神 | 以凶断 |
| 冲掉忌神 | 反而有利 |
| 冲掉喜用神 | 灾难 |
| 日主旺相 | 虽凶无碍 |
| 日主休囚 | 必凶 |

触发关键流年条件：
→ 天克地冲日柱：标记为关键流年，规则编号=[C-冲日]，对应 H04
→ 天克地冲年柱：标记为关键流年，规则编号=[C-冲年]，对应 H05
→ 天克地冲月柱/时柱：标记为关键流年，规则编号=[C-冲月/C-冲时]
```

### 检查 d：岁运并临 🟢查表

```
操作：
1. 比较流年干支与本步大运干支是否完全相同（天干地支皆同）
2. 若相同 = 岁运并临

岁运并临判定表（来源：调研-八字断事信号硬规则 一.1.3）：

| 条件 | 判定 | 规则编号 |
|------|------|----------|
| 岁运并临 + 并临之字为喜用神 | 大吉 | H02 |
| 岁运并临 + 并临之字为忌神 | 大凶 | H01 |
| 岁运并临 + 并临之字为闲神 | 平 | -- |
| 岁运并临 + 羊刃/七杀同时出现 | 极凶 | H03 |
| 岁运并临 + 枭神/劫财 | 凶 | H01变体 |
| 岁运并临 + 财官印绶为喜用 | 反吉 | H02变体 |
| 岁运并临 + 与四柱某柱形成天克地冲 | 大凶 | H01+H04/H05 |

触发关键流年条件：
→ 岁运并临（无论吉凶）：必标记为关键流年，规则编号=[D-并临]
```

### 检查 e：伏吟反吟 🟢查表

```
操作：
1. 伏吟检查：流年干支是否与命局某柱干支完全相同
   - 流年=年柱 → 年柱伏吟
   - 流年=月柱 → 月柱伏吟
   - 流年=日柱 → 日柱伏吟（H11）
   - 流年=时柱 → 时柱伏吟

2. 反吟检查：流年是否与命局某柱形成天克地冲
   （此项与检查 c 有重叠，但此处聚焦伏吟反吟的专门规则）
   - 流年天克地冲日柱 = 日柱反吟（H12）

伏吟效果表（来源：调研-八字断事信号硬规则 二.2.4）：

| 场景 | 效果 |
|------|------|
| 流年与年柱伏吟 | 长辈/父母有灾或变动 |
| 流年与月柱伏吟 | 兄弟姐妹或事业有变 |
| 流年与日柱伏吟 | 本人或配偶有灾，或破财 |
| 流年与时柱伏吟 | 子息或晚年有灾 |
| 流年与日柱反吟 | 婚姻宫被冲，影响婚姻稳定 |
| 流年与年柱反吟 | 留意长辈健康 |

伏吟反吟吉凶核心判断（来源：调研-八字断事信号硬规则 二.2.5）：

| 条件 | 结果 |
|------|------|
| 用神伏吟 | 吉上加吉（力量叠加） |
| 忌神伏吟 | 凶上加凶（凶星翻倍） |
| 忌神反吟 | 反而有利（忌神被冲去） |
| 用神反吟 | 不吉（用神根基被冲动） |

特殊关注（来源：调研-八字断事信号硬规则 九 H25）：
→ 60岁逢年柱伏吟：需特别警惕

触发关键流年条件：
→ 日柱伏吟：规则编号=[E-伏日]，对应 H11
→ 日柱反吟：规则编号=[E-反日]，对应 H12
→ 年柱伏吟（60岁附近）：规则编号=[E-伏年60]，对应 H25
→ 年柱/月柱伏吟或反吟：规则编号=[E-伏年/E-伏月/E-反年]
```

### 检查 f：犯太岁 🟢查表

```
操作：
1. 取命主生肖地支（年支）
2. 与流年地支对比，查犯太岁五种类型

犯太岁五种类型（来源：调研-大运流年作用细则与应期 九.9.1~9.5）：

| 类型 | 定义 | 影响程度 |
|------|------|----------|
| 值太岁（本命年） | 流年地支 = 年支 | 中高 |
| 冲太岁 | 流年地支与年支六冲 | 最高 |
| 刑太岁 | 流年地支与年支三刑/自刑 | 中高 |
| 害太岁 | 流年地支与年支六害 | 中 |
| 破太岁 | 流年地支与年支六破 | 低 |

影响程度排名：冲太岁 > 值太岁 >= 刑太岁 > 害太岁 > 破太岁

六冲对：子午、丑未、寅申、卯酉、辰戌、巳亥
六害对：子未、丑午、寅巳、卯辰、申亥、酉戌
六破对：子酉、卯午、辰丑、未戌、寅亥、巳申

<!-- fix: P1-R3-1 刑太岁内联三组三刑对照，无需跳转检查g -->
三刑对照表（刑太岁判定用）：
| 三刑组 | 构成地支 | 刑名 |
|--------|---------|------|
| 寅巳申 | 寅、巳、申 | 无恩之刑 |
| 丑未戌 | 丑、未、戌 | 恃势之刑 |
| 子卯 | 子、卯 | 无礼之刑 |
自刑：午午、酉酉、亥亥、辰辰

同时检查日支犯太岁（来源：调研-大运流年作用细则与应期 九.9.4）：
- 年支犯太岁 = 最重要（大环境）
- 日支犯太岁 = 次重要（本人/配偶）

综合判断（来源：调研-大运流年作用细则与应期 九.9.5）：

### 检查 g：三刑齐聚 🟡查表+判定

```
操作：
1. 取命局四柱地支 + 大运地支 + 流年地支，检查是否凑齐三刑

寅巳申三刑（无恩之刑）（来源：调研-八字断事信号硬规则 五.5.1）：
- 命局有其中两个，大运或流年出现第三个 → 三刑成立
- 命局有其中一个，大运有第二个，流年出现第三个 → 三刑成立
- 不论位置是否隔位

丑未戌三刑（恃势之刑）（来源：调研-八字断事信号硬规则 五.5.2）：
- 成立必要条件：命局或岁运必须有戊、己土透干
- 无戊己土透干 = 只有刑之意而无刑之事（硬判定条件）
- 对应规则：有透干 → H09（刑成立）；无透干 → H10（有意无事）

子卯相刑（无礼之刑）：
- 子卯同现即成立

辰午酉亥自刑：
- 同一地支重复出现

触发关键流年条件：
→ 寅巳申三刑齐聚：规则编号=[G-三刑寅巳申]
→ 丑未戌三刑齐聚 + 戊己土透干：规则编号=[G-三刑丑未戌]，对应 H09
→ 丑未戌三刑齐聚 + 无戊己土透干：标注=[有意无事]，对应 H10，不计入关键流年
→ 子卯相刑：规则编号=[G-刑子卯]
```

填实后吉凶判定（来源：调研-八字断事信号硬规则 六.6.4）：

## 7.2 岁运交战特别检查

```

岁运交战（来源：调研-八字断事信号硬规则 三.3.2 最后一行）：
- 大运与流年天克地冲 = 岁运交战

- 严重程度 = 5星（最高）

检查方法：
1. 流年天干与大运天干是否五行相克
2. 流年地支与大运地支是否六冲

- 日干克流年天干 = 日犯岁君
- 流年天干克日干 = 岁君犯日

| 条件 | 判定 | 规则编号 |

| 日犯岁君 + 五行有救 | 反得财 | H08 |

## 7.4 关键流年判定标准

```
统计每个流年命中的规则数量（来自 a-i + 岁运交战 + 特殊组合）：

| 命中规则数 | 判定 | 标签 |
|-----------|------|------|
| >= 3 | 极关键流年 | ★★★ |

## Step 7 Gate Check

```
ASSERT 关键流年.count >= 5
ASSERT 关键流年分散在 >= 3 个不同大运中
ASSERT 每个关键流年.命中规则数 >= 2
ASSERT 每个关键流年.规则编号列表 != null AND .length >= 2
ASSERT 每个关键流年.预判吉凶 IN [大吉, 吉, 小吉, 平, 小凶, 凶, 大凶]
ASSERT 所有流年.扫描完成 == true（无遗漏的大运或流年）
ASSERT 每个流年.九项检查 == done（a-i全部执行）
ASSERT 关键流年汇总表.行数 >= 5
ASSERT 非关键流年.标注 == "平" 或 "次关键"

→ 全部通过 → 进入 Step 8
→ 任一不通过 → 补齐后重检

特别注意：
- 若关键流年 < 5，检查是否有遗漏的规则检查项
- 若关键流年全部集中在同一步大运中，检查其他大运的扫描是否足够细致
