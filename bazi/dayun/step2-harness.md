## 2.1 天干五合扫描 🟡判定（决策树）

```
操作：
  输入 → 四柱天干（年干/月干/日干/时干）
  判定 → 逐对检查是否构成天干五合
  查表 → 五合配对表 + 合化条件决策树
  输出 → 每对合的判定结果（真化/半化/合绊/不合）+ 力量变化
```

### 天干五合配对表 🟢查表

| 合组 | 化神 | 性质 |
|------|------|------|

> 原理：阳干只与阴干合，同性不合。

### 天干五合判定决策树 🟡判定

```
发现两天干构成五合配对
│
├─ 两干是否紧贴相邻？
│  ├─ 否（隔柱/遥隔）→ 【不论合】，跳过
│  └─ 是（年月/月日/日时）→ 继续判断
│
├─ 有无妒合/争合？
│  ├─ 两个阳干合一个阴干（如庚-乙-庚）→ 争合，一般不成真化
│  ├─ 两个阴干合一个阳干（如乙-庚-乙）→ 妒合，不能合化
│  └─ 一对一正常合 → 继续判断
│
├─ 化神是否得月令？
│  │  查表：合化月令条件表（下方）
│  ├─ 否 → 【合绊】（双方各减力30%）
│  └─ 是 → 继续判断
│
├─ 化神有无强根？（地支中有化神五行的本气根）
│  ├─ 否 → 【合绊】
│  └─ 是 → 继续判断
│
├─ 被化方是否衰弱无根？
│  ├─ 否（被化方有根有助）→ 【合绊】
│  └─ 是（被化方极弱无根）→ 继续判断
│
├─ 天干有无克化神的五行？
│  ├─ 有 → 【合绊】
│  └─ 无 → 继续判断
│
├─ 日干参与合化？
│  ├─ 是 → 日干仍以原五行论（日干不化），但有合的倾向
│  └─ 否 → 继续判断
│
└─ 判定结果：
   ├─ 七条件全满足（被化方无根+化神得令+化神有根+无克害+一对一紧贴）→ 【真化】
   ├─ 仅一支持化神 → 【半化】
   └─ 其余 → 【合绊】
```

### 六冲判定决策树 🟡判定


## 2.10 冲合刑害并见处理 🟡判定

当同一地支同时涉及多种关系时，按以下规则处理：

### 并见处理规则 🟢查表

| 规则 | 处理 |
|------|------|
| 紧贴优先 | 刑冲合害并见时，紧贴者有力，优先判断 |
| 三会最稳 | 三会局见冲不作冲，仍以合论 |
| 三合与六冲 | 旺支被紧贴冲 → 以冲论；非旺支被冲或隔柱冲 → 以合论 |
| 六合与六冲 | 冲神在合神中间 → 破合论冲；冲神在合外相隔 → 贪合忘冲 |
| 合能解冲 | 有合化则冲力减弱或消失 |
| 冲能解合 | 有冲击则合力被破 |

> 来源：调研-八字地支作用硬规则.md 六.6.2。可靠度：✅确定。

## 常见错误防护

| 错误 | 正确做法 |
|------|---------|
| 隔柱天干论合化 | 只有紧贴（年月/月日/日时）才论合化，隔柱不论 |
| 忘记天干五合的"妒合"检查 | 两阴合一阳 = 妒合，不能合化 |
| 合化后仍按原五行计力 | 真化后原五行力量归零，转入化神五行 |
| 日干合化后不以原五行论 | 日干不论合化与否，仍以原五行看 |
| 地支关系重复扣减 | 合化成功后，不再另算冲/刑（合能解冲） |
| 冲后忘记处理藏干力量变化 | 余气最易被冲散，本气最不易 |
| 不区分克害和生害 | 克害破坏力远大于生害 |
| 把刑的力量等同于冲 | 冲>刑>害>破，力量依次递减 |
| 忽略并见冲突处理 | 同一地支涉及多种关系时必须按2.10规则处理 |
