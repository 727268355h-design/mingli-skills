> 前置：Step 3 旺衰校验卡已完成 + Phase 2 核心定位卡（格局/调候初判）

| 维度 | 判定 | 量化参考 |
|------|------|---------|
| 有力？ | 用神得令/通根/透干，不被克合 = [有力/无力] | 有力 = 用神五行得分 >= 总分10%且透干或有本气根；无力 = 得分 < 总分5%或仅余气根 |
| 有根？ | 天干在地支有同五行本气或中气藏干 = [有根/无根] | 有根 = 至少1个地支有同五行本气/中气根；仅余气根视为"弱根" |
| 有情？ | 忌神被制化合住 = [有情/无情] | 有情 = 忌神被制后力量减损 >= 50% |
| 层次 | 有情+有力=[上格] / 缺一=[中格] / 都无=[下格/败格] | -- |

> 量化边界为辅助判断参考线，传统命理以定性为主。当量化值处于边界附近时，以格局成败的定性判断优先。

## 4.3 调候满足度三级判定

### 透干 vs 藏干效力五级表

| 用神状态 | 效力层级 | 穷通宝鉴原文信号 |
|---------|---------|-----------------|
| 两用神皆透干 | 最高（科甲） | "庚丁两透，科甲定然" |
| 一透一藏 | 次高（小贵） | "庚透丁藏，小贵" |
| 两用神皆藏 | 中等（衣衿） | "甲癸俱藏，只可云富" |
| 有一缺一 | 低（偏科） | "有甲无丁，平常之辈" |
| 用神全无 | 最低（贫贱） | "丙甲全无，下流之造" |

### 三级判定标准

**充分满足（上格）**-- 需全部满足：

| 条件 | 检查 |
|------|------|
| 第一用神**透干**且**通根**（有力） | [✓/✗] |
| 第二用神透干或有力藏支 | [✓/✗] |
| 无冲克破坏用神的干支 | [✓/✗] |
| 不存在贪合/化合破格 | [✓/✗] |

**部分满足（中格）**-- 至少一条成立：

| 条件 | 检查 |
|------|------|
| 第一用神透干但无根（虚浮） | [✓/✗] |
| 第一用神藏干有根 | [✓/✗] |
| 有第一用神无第二用神 | [✓/✗] |
| 用神虽全但被合化/冲克减力 | [✓/✗] |

**不满足（下格）**-- 任一即成立：

| 条件 | 检查 |
|------|------|
