## 1.5 通根力量计算 🟢查表 + 🟡判定

```
操作：
  输入 → 四天干 + 四地支藏干
  判定 → 每个天干在四地支中是否有同五行的藏干（通根判定）
  查表 → 通根类型基础力量 + 通根距离衰减
  输出 → 每个天干的通根加力
```

**通根类型与基础力量值：**

| 通根类型 | 力量转化 | 说明 |
|---------|---------|------|
| 本气根 | 3.0 个日干元素（相当于 +36分 x 月令系数） | 最强根 |
| 中气根 | 1.5 个日干元素（相当于 +18分 x 月令系数） | 中等根 |
| 余气根 | 0.5 个日干元素（相当于 +6分 x 月令系数） | 弱根 |

> 来源：调研-八字天干作用硬规则.md 四.4.2。"得天干三比劫不如得地支一本气根"。可靠度：✅确定。

**通根距离衰减表：**

| 天干位置→地支位置 | 衰减 | 保留比例 |
|-----------------|------|---------|
| 同柱（坐支） | 不减力 | 100% |
| 相邻（邻柱） | 减力 | 90%（减10%） |
| 相隔（隔一柱） | 减力 | 75%（减25%） |
| 远隔（隔两柱，年干-时支或时干-年支） | 减力 | 60%（减40%） |

> 来源：调研-五行力量量化详表.md 四.4.2。变速递减（0.5度→1度→2度）。可靠度：⚠️中（衰减具体数值存在不同说法，取多源交集的保守值）。

**柱位距离速查（以日干为原点）：**

| 日干到 | 距离 | 保留比例 |
|--------|------|---------|
| 日支 | 同柱 | 100% |
| 月支 | 相邻 | 90% |
| 时支 | 相邻 | 90% |
| 年支 | 隔一柱 | 75% |

**其他天干到各地支的距离，同理推算：**

| 天干 | 年支 | 月支 | 日支 | 时支 |
|------|------|------|------|------|
| 年干 | 同柱100% | 邻柱90% | 隔柱75% | 远隔60% |
| 月干 | 邻柱90% | 同柱100% | 邻柱90% | 隔柱75% |
| 日干 | 隔柱75% | 邻柱90% | 同柱100% | 邻柱90% |
| 时干 | 远隔60% | 隔柱75% | 邻柱90% | 同柱100% |

<!-- fix: P2-R1-3 通根"同五行"判定加示例 -->
> **通根判定示例**：甲木天干，地支辰中有乙木（中气），乙木属木 = 同五行 → 通根（中气根）。判定关键：只要藏干与天干属同一五行即为通根，不要求天干地支完全同字。

```
逐个天干检查通根：

  日干 [X]（[五行]）：
    → 日支藏干中有同五行[A]吗？ 有=[本气/中气/余气]根，距离=同柱(100%)
    → 月支藏干中有同五行[B]吗？ 有=[本气/中气/余气]根，距离=邻柱(90%)
    → 时支藏干中有同五行[C]吗？ 有=[本气/中气/余气]根，距离=邻柱(90%)
    → 年支藏干中有同五行[D]吗？ 有=[本气/中气/余气]根，距离=隔柱(75%)
    → 通根总加力 = Σ(根基础分 x 月令系数 x 距离保留比例)

  （年干、月干、时干同法逐个检查）
```

## 1.6 按五行汇总静态力量 🟢查表

```
操作：
  输入 → 1.3 天干力量 + 1.4 地支藏干力量 + 1.5 通根加力
  汇总 → 按金/木/水/火/土五行分类累加
  输出 → 五行力量静态得分表
```

**汇总规则：**
1. 每个天干的力量归入其五行
2. 每个地支藏干的力量归入其五行
3. 每个天干的通根加力归入该天干的五行
4. 同一五行的所有力量直接累加

```
五行力量静态得分 =
  木 = Σ(木天干力量 + 木藏干力量 + 木天干通根加力)
  火 = Σ(火天干力量 + 火藏干力量 + 火天干通根加力)
  土 = Σ(土天干力量 + 土藏干力量 + 土天干通根加力)
  金 = Σ(金天干力量 + 金藏干力量 + 金天干通根加力)
  水 = Σ(水天干力量 + 水藏干力量 + 水天干通根加力)

五行总分 = 木 + 火 + 土 + 金 + 水
日主力量 = 日干五行得分（含同类五行：比劫+印星）
```

## Step 1 Gate Check

```
ASSERT 五行得分表.count == 5（木火土金水各有分值）
ASSERT 五行得分.每项 > 0（不可能有某五行为0分，至少有藏干贡献）
ASSERT 五行得分.sum ∈ [500, 600]（544分制合理范围校验：天干144+地支400=544，通根加力后应在此范围）
ASSERT 日主得分 != null AND 日主得分 > 0
ASSERT 天干得分.每项 ∈ [7.2, 36]（36x0.2=7.2 为最低，36x1.0=36 为最高）
ASSERT 地支藏干得分.每个地支的藏干分之和 ∈ [20, 100]（最低：100x0.2=20，最高100）
ASSERT 通根加力.每项 >= 0（通根加力不可能为负）

→ 全部通过 → 进入 Step 2
→ 任一不通过 → 检查计算过程，修正后重新汇总
```

## 常见错误防护

| 错误 | 正确做法 |
|------|---------|
| 忘记乘月令系数，直接用基础分 | 每个力量值 = 基础分 x 月令系数，无例外 |
| 通根加力忘记乘距离衰减 | 通根力量 = 根基础分 x 月令系数 x 距离保留比例 |
| 藏干分配比例错误 | 纯气100；二气70:30；三气60:30:10 |
| 同柱干支的五行力量重复计算 | 天干和地支藏干是独立的力量来源，分别计算后汇总 |
| 忘记区分本气/中气/余气的分值差异 | 本气60(或70/100)，中气30，余气10，严格按表 |
