## 9.1 应期六大判据 🟡查表+判定

> 来源：调研-大运流年作用细则与应期.md 十.10.2

按以下六大判据逐一检查每个关键事件，确定应期早晚。

| 序号 | 判据 | 操作 | 效果 |
|------|------|------|------|
| J1 | 命局信息量 | 统计该事件在原局中的信号密度（Step 5 信号清单） | 信号多 → 应事早；信号少 → 应事迟 |
| J2 | 行运调整 | 查 Step 6/7 大运基调 | 好运推迟应凶 / 坏运加速应凶 |
| J3 | 大运干支旺衰 | 查大运地支在当前月令的旺衰状态 | 旺则应事在前5年 / 衰则应事在后5年 |
| J4 | 用忌神上运 | 查 Step 8 该流年的事件预判卡中的十神 | 忌神主事 → 应事快且重 / 用神主事 → 应事缓且轻 |
| J5 | 同异性力度 | 判断流年干支与命局干支的阴阳关系 | 同性 → 应事明显 / 异性 → 应事不明显 |
| J6 | 年运作用结果最明显时 | 流年干支与大运干支作用后，对命局影响最大的月份 | 该月 = 应期 |

```
操作：
1. 从 Step 8 取出所有标注了事件类型的关键流年
2. 每个关键流年，逐一过 J1-J6 六大判据
3. J1-J5 判定应期方向（早/迟/快/慢/明显/隐蔽）
4. J6 判定具体月份（需配合 9.3 流月推算）
5. 综合六判据，输出应期窗口
```

## 9.2 引动方式与应期对应表 🟢查表

> 来源：调研-大运流年作用细则与应期.md 十.10.3
> 来源：调研-八字断事信号硬规则.md 八.8.7

命局中已存在的关系类型，决定了事件被引动的方式。查下表确定应期。

| 编号 | 命局已有关系 | 引动方式（应期） | 原理 | 推理强度 |
|------|-------------|-----------------|------|---------|
| T1 | 原局有合（天合/地合） | 以**冲**为应期 | 冲破合局引发事件 | 🟢高 |
| T2 | 原局有冲 | 以**合**为应期 | 合住冲力引发变化 | 🟢高 |
| T3 | 原局有刑 | 以合或冲为应期 | 解刑引发变化 | 🟡中 |
| T4 | 原局有害 | 以合或冲为应期 | 解害引发变化 | 🟡中 |
| T5 | 空亡 | 以**冲空**或**填空**为应期 | 空亡填实优先级最高 | 🟢高 |
| T6 | 用神入墓 | 以**冲开墓库**为应期 | 用神释放 | 🟢高 |
| T7 | 忌神入墓 | 以**冲开墓库**为应期 | 忌神释放 | 🟢高 |
| T8 | 合者见冲 | 事件发生 | 合局被破 | 🟢高 |
| T9 | 冲者见合 | 事件发生 | 冲动被合住 | 🟢高 |

**应期口诀**（辅助记忆，不作硬规则）：
- "远慢断年月，近快断日时"
- "反吟主快应近时，伏吟主慢应远时"
- "命局忌神 + 岁运忌神 = 最凶之时"

```
操作：
1. 逐个检查关键流年涉及的命局关系类型
2. 查 T1-T9 表确定引动方式
3. 找出满足引动条件的流月地支
4. T5（空亡填实）优先级最高，先查空亡
5. 记录每个应期的引动编号（T1-T9）
```

## 9.3 五步应期判断流程 🟡决策树

> 来源：调研-大运流年作用细则与应期.md 十.10.4

对每个关键事件，按以下五步**顺序**执行，不可跳步。

### Step A：检查空亡（最优先）

```
操作：
1. 从 Step 0 取空亡二字
2. 检查该关键流年的流月地支是否填实/冲实空亡
3. 若有 → 该流月为应期（T5 引动）
4. 空亡填实方式：
   - 流月地支 = 空亡之字 → 填实（最正规）
   - 流月地支 冲 空亡之字 → 冲实
   - 流月地支 合 空亡之字 → 合实
5. 填实后吉凶查表：
   | 空亡之字原属性 | 出空效果 |
   |--------------|---------|
   | 吉神（财官禄印）+ 用神 | 应吉 |
   | 凶神（恶煞）+ 忌神 | 应凶 |
   | 忌神逢空（在空中） | 暂吉（凶神暂不发作） |
   | 用神逢空（在空中） | 暂不吉（吉神暂不发力） |
```

### Step B：分析驿马（主动变化信号）

```
操作：
1. 从 Step 0 取命局驿马（神煞列表）
2. 检查关键流年的流月地支是否冲/合驿马
3. 马星被冲合时 → 主动变化应事（迁移/调动/出行）
4. 驿马查法：
   申子辰马在寅，寅午戌马在申，
   巳酉丑马在亥，亥卯未马在巳
```

### Step C：审视墓库（能量封存/释放）

```
操作：
1. 检查关键流年是否涉及墓库开关
2. 四墓库对应表：
   | 五行 | 墓库 |
   |------|------|
   | 木 | 未 |
   | 火 | 戌 |
   | 土 | 戌 |
   | 金 | 丑 |
   | 水 | 辰 |
3. 流月地支冲开墓库 → 能量释放月
   - 辰戌冲 / 丑未冲 = 最常见开库方式
   - 未戌刑 / 丑戌刑 / 戌未刑 = 刑开方式
4. 用神入墓逢冲月 → 该月应吉
5. 忌神入墓逢冲月 → 该月应凶
```

### Step D：考察刑冲合害（常规引动分析）

```
操作：
1. 流月地支与命局四柱逐一检查六冲/六合/三刑/六害
2. 合冲到哪个五行 = 应哪类事
3. 流月冲四柱的象意：
   | 流月冲 | 影响 |
   |--------|------|
   | 天克地冲年柱 | 长辈病灾 / 房产破财 |
   | 天克地冲月柱 | 兄弟姐妹有灾 / 破财 |
   | 天克地冲日柱 | 本人或配偶病灾 |
   | 天克地冲时柱 | 本人及子女灾病 |
4. 流月吉凶与流年叠加：
   | 流月 | 流年 | 结果 |
   |------|------|------|
   | 吉 | 吉 | 大吉 |
   | 吉 | 凶 | 吉中有凶 |
   | 凶 | 凶 | 凶上加凶 |
   | 凶 | 吉 | 凶中有吉 |
```

### Step E：应用合德原则（精密判断）

```
操作：
1. 检查流月是否同时满足：
   - 天干五合成立（天干层面合）
   - 地支六合成立（地支层面合）
2. 天地合德同时满足 → 应大事月
3. 天合而地不合 → 应小事
4. 地合而天不合 → 应小事
5. 天地均冲 → 应大动荡
```

## 9.5 流月断事层级规则 🟡判定

> 来源：调研-大运流年作用细则与应期.md 三.3.3

| 规则编号 | 规则 | 说明 |
|---------|------|------|
| M1 | 流月依附流年 | 流年好则流月无大凶，流年差则流月好也"昙花一现" |
| M2 | 流月不直接作用命局 | 流月先在流年上起作用，旺时才可直接作用命局 |
| M3 | 事件层级 | 大运应事 = 大事，流年应事 = 中事，流月应事 = 小事 |
| M4 | 断流月以流年为本 | 干对干作用，支对支作用 |
| M5 | 流月吉凶应期 | 多在交下一个节气之前应验 |

## 常见陷阱

| 陷阱 | 正确做法 |
|------|---------|
| 凭直觉定月份 | 必须走五步流程，每步有查表依据 |
| 空亡不检查就跳过 | 空亡填实是优先级最高的应期判据（T5） |
| 流月推算用公历月 | 流月以节气分界，非公历月初 |
| 应期说"X月必定发生Y" | 只说"X月信号较强"，不做绝对断语 |
| 忽略流月依附流年规则（M1） | 流年差则流月好也短暂，不可孤立看流月 |
| 伏吟反吟不区分应期快慢 | "反吟主快应近时，伏吟主慢应远时" |
