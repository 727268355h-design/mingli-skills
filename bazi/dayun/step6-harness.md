# Step 6：逐步大运分析 -- 执行指令

> 前置：Step 0-5 全部完成（前序数据卡、力量表、旺衰校验、格局调候校验、原局信号扫描）
> 输入：Step 0 前序数据卡 + Step 1 五行力量表 + Step 3 旺衰校验卡 + Step 4 格局调候校验卡
> 输出：大运逐步卡（下方模板）
> 原则：**每步大运必须执行 a-h 全部 8 项判定，不得跳步或简化。查表为主，公式评分，不凭感觉。**

## !! Context Anxiety 防护（必读）

**本 Step 是整个 SOP 分析量最大的环节。** 必须遵守以下规则：

1. **分段执行**：先完成前 4 步大运（Step 6a），输出中间卡片并自检质量，再继续后续大运（Step 6b）
2. **密度一致性**：每步大运的分析字段不得少于 8 个，后面不得比前面粗糙
3. **质量优先于完成度**：宁可只分析 6 步大运但每步深入，也不要 8 步但后 4 步敷衍
4. **自检信号**：出现"类似前述""不再赘述""同上"等省略语 = 立即停下重新展开
5. **每步大运独立展开**：不可引用"参见第 N 步大运"来代替分析

### 判定 a：大运干支五神判定 🟢查表

```
操作：
1. 取本步大运天干 → 判定其五行 → 查五神归属（用/喜/忌/仇/闲）
2. 取本步大运地支 → **加权法**判定五神归属：
   - 查地支藏干（本气/中气/余气）各自的五行
   - 按力量权重加权：本气60% + 中气30% + 余气10%（来源：调研-八字旺衰量化打分规则 二.2.1，曲炜/易子著体系三气60:30:10）

```
操作：
1. 取本步大运天干，与命局 4 个天干逐一比对

| 大运合住/合化命局用神 | 用神力量减弱或消失 | 大凶 |
| 大运被命局忌神合住 | 忌神贪合忘克 | 逢凶化吉 |

| 情境 | 效果 |
|------|------|
| 大运天干克命局用神 | 用神受损，该运不利 |
| 大运天干克命局忌神 | 忌神被制，该运有利 |
| 大运天干克日干 | 官灾讼事之运，身弱者尤需谨慎 |
| 日干克大运天干 | 财星之运；身强则进财，身弱需防破财 |

作用力量排序（来源：调研-八字天干作用硬规则 五.5.4）：
合 > 克、生、泄 > 助 > 耗

同性作用应事明显，异性作用应事不明显。

**合绊格局核心天干特殊扣分**（来源：调研-八字天干作用硬规则 五.5.2"合住用神=大凶"）：

  若大运天干合住命局天干（五合成立），额外检查被合住天干是否为**格局核心**：
  - 格局核心天干 = 格局配置中的关键十神天干（如杀印相生格中的七杀/印星天干）
  - 合绊格局核心用神/喜神天干 → **综合分额外扣 20 分**（格局链断裂=极凶）
    → A8（2026-03-29）：原A5值-15，实测偏差+24。回调至-20（折中，避免对单案例过拟合）。待更多案例验证
  - 合绊格局核心忌神天干 → **综合分额外加 12 分**（忌神被锁=大吉）
    → A8：忌神锁定同步从+10调至+12
  - 合绊闲神天干 → 不额外加减

  **杀极旺豁免**（来源：cross-validation毛泽东命造 2026-03-29）：

    IF 判定 f 已标记"杀极旺" AND 被合天干为格局核心**七杀**天干 AND 大运天干为**食伤**：
      → 食伤合住杀 = 制杀有功 = **改为额外加 10 分**（而非扣 15）
      → 原理：杀极旺时，杀被合绊=减压=好事，标准"合绊格局核心=凶"不适用

→ 输出：
  大运天干[X] × 年干[X] = [合/克/生/比/泄]，效果=[X]
  大运天干[X] × 月干[X] = [合/克/生/比/泄]，效果=[X]
  大运天干[X] × 日干[X] = [合/克/生/比/泄]，效果=[X]
  大运天干[X] × 时干[X] = [合/克/生/比/泄]，效果=[X]
  合绊格局核心=[有/无]，额外加减=[+10/-15/0]
  天干作用总结=[X]
```

> **两派争议说明（命理学重大分歧之一）**：
> - **阳顺阴逆派（传统主流，本表所用）**：阴干逆行排列，乙木长生在午、辛金长生在子等。来源于洛书阴阳顺逆理论。《滴天髓》《子平真诠》均承认此排法。
> - **阴阳同顺派（反对派）**：主张阴干应与同五行阳干查同一行（即乙同甲、丁同丙），理由是辛金长生在子="水生金"、癸水长生在卯="木生水"违反五行生克。民国徐乐吾主张"生旺墓绝仅分五行，不必分阴阳"。
>
> **本SOP默认方案**：采用"阳顺阴逆"排列（传统主流），但阴干长生的力量权重降低：
> - 阴干长生视为"有根但不强"，相当于余气级别的力量（约为阳干长生的 50-60%）
> - 《子平真诠》原文："阳长生有力，阴长生不甚有力，然亦不弱...亦为有根，比得一余气"
> - 判定e中，若用神为阴干，十二长生分值按 70% 折算
>
> **备选方案**：若命理师倾向"阴阳同顺"派，可直接查上方五阳干表（乙同甲行、丁/己同丙/戊行、辛同庚行、癸同壬行），此时不需折算权重。

    → 大运反吟日柱
    → 调整分 = **-18 分**
    → 含义：整步大运天克地冲日柱，婚姻宫被冲动，人生大动荡
    → 置信度：高（来源：H12"反吟日柱=婚姻动荡"）
    → 用忌区分：
       IF 冲去日支忌神 → 反吟反吉（调整改为 -5）
       IF 冲去日支用神 → 反吟大凶（调整保持 -18）
    → A7（2026-03-29）：原A1值-15，与伏吟保持约7:10比例。待更多案例验证后微调

非伏吟非反吟：
  → 调整分 = 0（无额外加减）

→ 输出：伏吟反吟=[伏吟(-20)/反吟(-15)/无(0)]，调整分=[X]
```

- 大运虚神与命局字存在 合/冲/刑 关系 → 虚神作实神论

## 6.4 分段执行检查点

```
【Step 6a 检查点】完成前4步大运后执行：
1. 逐步抽查：随机选1步大运，验算 a项加权计算 和 h项综合分加法
2. 检查每步大运 a-i 全部 9 项是否都有 INPUT→RULE→SCORE 三段
3. 检查自检清单 7 项是否全部打勾
4. 检查后面的步骤推导密度 >= 前面（不得出现省略）
→ 通过 → 继续 Step 6b（后续大运）
→ 不通过 → 定位缺失项，补齐推导链后再继续

【Step 6b 检查点】完成全部大运后执行：
1. 重复 Step 6a 的全部检查
2. 额外检查：总步数 >= 6
3. 额外检查：覆盖命主当前所在大运
4. 横向一致性检查：
   - 同一五神组合在不同大运中得分是否一致（如两步都是喜×闲，应都是3星18分）
   - 同一格局配置在不同大运中查表是否一致
5. 趋势合理性检查：
   - 用神大运(壬/癸)得分是否 > 忌神大运(戊/己)得分
   - 有伏吟/反吟的大运得分是否显著低于相邻大运
→ 通过 → 进入 Step 6 Gate Check
→ 不通过 → 补齐
```
