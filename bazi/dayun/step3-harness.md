# Step 3：旺衰定级校验 -- 执行指令

> 前置：Step 1 五行力量静态得分表 + Step 2 调整后五行力量表已完成
> 输入：Step 1-2 的量化得分 + Phase 2 核心定位卡（定性旺衰）
> 输出：旺衰校验卡（下方模板）
> 原则：**用量化得分对照阈值表，与 Phase 2 的定性判断做交叉校验。** 两者一致则确认，不一致则标注分歧并给出优先判断。

---

## 3.1 日主同类/异类得分汇总

从 Step 2 调整后五行力量表中提取：

```
日主五行 = [X]

同类五行（帮扶日主）：
  - 日主本五行得分 = [A]分
  - 印星五行（生日主）得分 = [B]分
  - 同类合计 = [A + B]分

异类五行（克泄耗日主）：
  - 官杀五行（克日主）得分 = [C]分
  - 食伤五行（日主生）得分 = [D]分
  - 财星五行（日主克）得分 = [E]分
  - 异类合计 = [C + D + E]分

五行总分 = [同类 + 异类]分
日主同类占比 = 同类 / 五行总分 x 100% = [X]%
```

## 3.2 旺衰分级阈值表（三套体系对比）

### 体系A：曲炜体系（544分制）
> 来源：调研-八字旺衰量化打分规则.md 三.3.1 旺衰六级表

| 等级 | 同类分数范围 | 同类占比 | 典型特征 |
|------|------------|---------|---------|
| 旺极 | >435 | >80% | 6-8个同类，可论专旺格 |
| 太旺 | 272-435 | 50%-80% | 4-6个同类，身旺喜克泄耗 |
| 偏旺 | 109-272 | 20%-50% | 2-4个同类 |
| 中和 | ~109 | ~20% | 均衡值=544/5=108.8 |
| 偏弱 | 45-109 | 8%-20% | 身弱喜生扶 |
| 太弱 | <45 | <8% | 极少同类，检查从格 |

### 体系B：易子著体系（560分制）-- 主用体系
> 来源：调研-五行力量量化详表.md 三.3.3 易子著旺衰分级

| 等级 | 同类分数范围 | 说明 |
|------|------------|------|
| 旺极 | >450 | 专旺格条件 |
| 太旺 | 281-450 | 身旺，喜克泄耗 |
| 偏旺 | 120-280 | 身稍旺 |
| 中和 | 105-119 | 均衡值=560/5=112，真正中和极少 |
| 偏弱 | 45-104 | 身稍弱，喜生扶 |
| 太弱 | 15-44 | 身很弱，检查假从 |
| 弱极 | <15 | 真从格条件 |

### 体系C：潘爱民体系（相对比值法）

| 判定 | 条件 |
|------|------|
| 偏强 | 同类合计 > 异类合计 |
| 偏弱 | 同类合计 < 异类合计 |
| 中和 | 同类与异类差值 < 总分的5% |

> **选定主用体系**：以**体系B（560分制）**为主判定体系。Step 1-2 使用潘爱民查表计算的得分需换算至560分制后对照阈值（换算方法：潘爱民得分 x 换算系数，使五行总分归一至560）。体系A和C作辅助验证。

## 3.3 执行旺衰定级

### a. 查阈值定级

```
Step 2 调整后日主同类得分 = [X]分
换算至560分制 = [Y]分（若已是560分制则直接用）
对照体系B阈值表 → 量化旺衰等级 = [旺极/太旺/偏旺/中和/偏弱/太弱/弱极]
```

### b. 辅助体系交叉验证

```
体系A（544分制）换算 → [等级]
体系C（潘爱民比值法）→ 同类[X] vs 异类[Y] → [偏强/偏弱/中和]
三套体系一致性 = [全部一致 / 2:1分歧 / 三方分歧]
```

### c. 与 Phase 2 定性判断交叉校验

```
Phase 2 定性旺衰 = [极强/偏强/中和偏强/中和/中和偏弱/偏弱/极弱/从格]
Phase 2 判定方法 = 三得法（得令[✓/✗] + 得地[✓/✗] + 得势[✓/✗]）

量化结果 = [旺极/太旺/偏旺/中和/偏弱/太弱/弱极]
定性结果 = [Phase 2 结论]

一致性判定：
  ├─ 完全一致 → 直接确认，标注 ✅
  ├─ 方向一致但程度不同（如定性"偏强"量化"太旺"）→ 取中间值，标注 ⚠️
  └─ 方向相反（如定性"偏强"量化"偏弱"）→ 必须重新检查，标注 ❌

▸ 最终旺衰等级 = [确定结论]
▸ 一致性状态 = [一致 ✅ / 偏差 ⚠️ / 冲突 ❌]
```

**方向相反时的处理流程**：
1. 检查 Step 1-2 的量化计算是否有遗漏（最常见：遗漏通根、合化未扣减）
2. 检查 Phase 2 是否有月令判断错误
3. 若确认计算和定性均无误 → 以三得法定性为主（✅确定），量化为辅，标注分歧原因

## 3.4 从格硬判定

仅当日主得分处于极端区间时执行此检查。

### 从弱格检查（日主同类得分 < 45分 / 560分制）
> 来源：调研-八字格局判定硬规则.md 五.5.2.1 从弱格条件表

四条必须**同时满足**才论从格：

| 序号 | 条件 | 检查结果 |
|------|------|---------|
| 1 | 日主失令（月令不帮日主） | [✓/✗] |
| 2 | 日主无**本气**通根（仅余气根或完全无根） | [✓/✗] |
| 3 | 命局中某种五行势力极强（占比>60%） | [✓/✗]，极强五行=[X] |
| 4 | 所从之神透干 | [✓/✗] |

```
四条全满足 → 从格（标注从财/从官/从杀/从儿/从强/从旺）
有余气根无本气根 → 假从格（大运见帮扶即不从）
任一不满足 → 不从，仍为正格身弱
```

### 从强格检查（日主同类得分 > 450分 / 560分制）

| 序号 | 条件 | 检查结果 |
|------|------|---------|
| 1 | 日主得令 | [✓/✗] |
| 2 | 满盘印比，无财官食伤（或极弱虚浮） | [✓/✗] |
| 3 | 旺神透干 | [✓/✗] |
| 4 | 地支无克泄耗力量 | [✓/✗] |

```
全满足 → 专旺格/从强格/从旺格
  ├─ 印强比弱 → 从强格
  ├─ 比强印弱 → 从旺格
  └─ 日主五行单独极旺 → 专旺格（曲直/炎上/稼穑/从革/润下）
不满足 → 正格身旺
```

### 真从 vs 假从判定

| 判定项 | 真从格 | 假从格 |
|--------|--------|--------|
| 日主根气 | 完全无根 | 有余气根或暗藏微弱印比 |
| 成分纯度 | 纯粹，无对立十神 | 含少量对立十神 |
| 稳定性 | 稳定，大运不改变 | 不稳定，大运可切换正格/从格 |
| 对大运影响 | 用神固定 | 需逐步大运重新判定用神 |
