# Step 8：断事信号匹配 -- 执行指令

> 前置：Step 7 关键流年筛选已完成（关键流年汇总表已输出）
> 输入：Step 0 前序数据卡 + Step 5 原局信号清单 + Step 6 大运逐步卡 + Step 7 关键流年汇总表
> 输出：事件预判卡（下方模板）
> 原则：**对每个关键流年，逐条检查 H01-H25 全部 25 条硬断规则 + 六类事件信号表。不可跳过任何一条规则。**

## !! Context Anxiety 防护

1. **25 条规则必须全部过一遍**：不可因为已命中某条就跳过剩余规则
2. **六类事件信号表必须全部检查**：结婚/升职/破财/丧亲/官非/疾病，逐类扫描
3. **每个关键流年独立分析**：不可引用"同前述"来简化
4. **信号强度必须有依据**：强/中/弱的判定基于命中规则数量和置信度
5. **分段执行检查点**：每完成3个关键流年的分析后，输出中间结果并自检（检查H表是否25条全查、六类事件是否全扫描、后面的分析密度 >= 前面）

## 8.1 H01-H25 综合硬断速查表逐条检查

对每个关键流年，逐条检查以下 25 条规则。来源：调研-八字断事信号硬规则 九。

```
对关键流年 [干支]年（[公元]，[虚岁]岁），逐条执行：

H01. 岁运并临 + 忌神
  检查：流年干支 = 大运干支？ AND 并临之字为忌神？
  判定：大凶
  置信度：高
  → 命中/未命中

H02. 岁运并临 + 喜用神
  检查：流年干支 = 大运干支？ AND 并临之字为喜用神？
  判定：大吉
  置信度：高
  → 命中/未命中

H03. 岁运并临 + 羊刃七杀
  检查：流年干支 = 大运干支？ AND 同时出现羊刃和七杀？
  判定：极凶
  置信度：高（古籍硬标注）
  → 命中/未命中

H04. 流年天克地冲日柱 + 用神被冲
  检查：流年天干克日干？ AND 流年地支冲日支？ AND 被冲动的五行为用神？
  判定：大凶（婚灾/自身灾）
  置信度：高
  → 命中/未命中

H05. 流年天克地冲年柱
  检查：流年天干克年干？ AND 流年地支冲年支？
  判定：长辈有灾
  置信度：中高
  → 命中/未命中

H06. 岁运交战（大运流年天克地冲）
  检查：流年天干克大运天干（或反之）？ AND 流年地支冲大运地支？
  判定：该年大动荡
  置信度：高
  → 命中/未命中

H07. 日犯岁君 + 日主休囚
  检查：日干克流年天干？ AND 日主旺衰 IN [偏弱,太弱,弱极]？
  判定：必凶
  置信度：高（古籍明断）
  → 命中/未命中

H08. 日犯岁君 + 五行有救
  检查：日干克流年天干？ AND 命局或大运有制约力量？
  判定：反得财
  置信度：中高
  → 命中/未命中

H09. 丑未戌三刑 + 戊己土透干
  检查：命局+大运+流年地支凑齐丑未戌三字？ AND 天干有戊或己透出？
  判定：三刑成立，断事（健康/是非/破财）
  置信度：高（无透干则不成事）
  → 命中/未命中

H10. 丑未戌三刑 + 无戊己土透干
  检查：命局+大运+流年地支凑齐丑未戌三字？ AND 天干无戊己透出？
  判定：有意无事
  置信度：高
  → 标注但不计入凶信号

H11. 伏吟日柱（流年 = 日柱）
  检查：流年天干 = 日干？ AND 流年地支 = 日支？
  判定：本人或配偶有灾/破财
  置信度：中高
  → 命中/未命中

H12. 反吟日柱（流年天克地冲日柱）
  检查：流年天干克日干？ AND 流年地支冲日支？
  判定：婚姻动荡
  置信度：高
  → 命中/未命中
  注意：H12 与 H04 可同时命中（H04 额外要求用神被冲）

H13. 用神入墓 + 逢冲年
  检查：用神所对应的五行有入墓库？ AND 流年地支冲该墓库？
  判定：吉运到来（用神释放）
  置信度：中高
  → 命中/未命中

H14. 忌神入墓 + 逢冲年
  检查：忌神所对应的五行有入墓库？ AND 流年地支冲该墓库？
  判定：凶气爆发（忌神释放）
  置信度：中高
  → 命中/未命中

H15. 空亡忌神 + 流年填实
  检查：空亡地支所藏干为忌神？ AND 流年地支 = 空亡地支？
  判定：应凶（忌神重新作祟）
  置信度：中高
  → 命中/未命中

H16. 空亡用神 + 流年填实
  检查：空亡地支所藏干为用神？ AND 流年地支 = 空亡地支？
  判定：应吉（用神回归发力）
  置信度：中高
  → 命中/未命中

H17. 杂气财官透干 + 适度冲刑 + 身强
  检查：命局有辰戌丑未（杂气）？ AND 其中财官藏干已透出天干？ AND 流年冲该墓库？ AND 日主旺衰 IN [偏旺,太旺,旺极]？
  判定：越冲越发（三条件缺一不可）
  置信度：高
  → 命中/未命中

H18. 伤官见官 + 流年官星透出
  检查：原局有伤官透干？ AND 流年天干为正官？ AND 正官为命局所用？
  判定：官非牢狱风险
  置信度：中高
  → 命中/未命中

H19. 三刑刑入正印
  检查：三刑成立？ AND 被刑的地支藏干含正印星？
  判定：丧母之危
  置信度：中
  → 命中/未命中

H20. 三刑刑入偏财
  检查：三刑成立？ AND 被刑的地支藏干含偏财星？
  判定：丧父之危
  置信度：中
  → 命中/未命中

H21. 强者喜冲（身旺遇冲）
  检查：日主旺衰 IN [偏旺,太旺,旺极]？ AND 流年有冲（冲日支或冲月支）？
  判定：冲则发
  置信度：高
  → 命中/未命中

H22. 弱者忌冲（身弱遇冲）
  检查：日主旺衰 IN [偏弱,太弱,弱极]？ AND 流年有冲（冲日支或冲月支）？
  判定：冲则拔
  置信度：高
  → 命中/未命中

H23. 合者见冲为应
  检查：原局有合局（天干合或地支合）？ AND 流年冲开该合局？
  判定：事件发生应期（合被冲开 = 事件兑现）
  置信度：高
  → 命中/未命中

H24. 冲者见合为应
  检查：原局有冲局（地支六冲）？ AND 流年合住冲局中某字？
  判定：事件发生应期（冲被合住 = 变化定型）
  置信度：高
  → 命中/未命中

H25. 60岁逢年柱伏吟
  检查：命主虚岁 >= 58 AND <= 62？ AND 流年干支 = 年柱干支？
  判定：需特别警惕
  置信度：中高
  → 命中/未命中
```

## 8.3 通用应期规则交叉（辅助验证）

```
对每个关键流年命中的事件类型，用通用应期规则交叉验证（来源：调研-八字断事信号硬规则 八.8.7）：

| 规则 | 说明 |
|------|------|
| 合者见冲为应 | 原局合住的东西，流年冲开时事件发生（H23） |
| 冲者见合为应 | 原局冲动的东西，流年合住时事件发生（H24） |
| 入墓逢冲/刑 = 开库 | 能量释放，吉凶兑现 |
| 入墓无冲刑 = 封库 | 能量停滞，吉凶不显 |
| 流年先与大运作用，再作用八字 | 层级关系（君臣民） |
| 大运好+流年好 | 吉上加吉 |
| 大运好+流年差 | 吉中带凶 |
| 大运差+流年好 | 凶中带吉 |
| 大运差+流年差 | 凶上加凶 |

交叉验证结果：
→ 通用应期规则与事件类型一致 = 信号增强
→ 通用应期规则与事件类型矛盾 = 信号减弱，标注 ⚠️
```

## 8.4 信号强度评定

```
对每个关键流年，根据以下标准评定信号强度：

| 强度 | 条件 | 说明 |
|------|------|------|
| 强 | H表命中 >= 3 条 + 事件信号表命中 >= 1 条 + 通用应期规则验证通过 | 多重信号交叉印证，该年大概率有事 |
| 中 | H表命中 2 条 + 事件信号表命中 >= 1 条 | 信号明确但交叉验证不完全 |
| 弱 | H表命中 2 条 + 事件信号表未命中 | 有触发条件但事件类型不明确 |

信号强度 = 强时，该关键流年的事件预判可信度高
信号强度 = 中时，该关键流年需要结合 Step 9 应期进一步定位
信号强度 = 弱时，该关键流年的事件可能为"有信号无实事"或影响轻微

特别注意（来源：调研-八字断事信号硬规则 十 使用注意事项）：
1. 喜忌是前提 — 喜忌判断错误则所有规则吉凶反转
2. 力量是关键 — 同样信号在身旺和身弱命局中效果可能相反
3. 宫位定六亲 — 受冲柱位决定应事对象
4. 单一信号不足以定论 — 需多信号交叉验证
5. 不做绝对断语 — 说"此年有较强信号"，不说"X年必定发生Y"
```

## Step 8 Gate Check

```
ASSERT 每个关键流年.H表检查数 == 25（H01-H25全部逐条检查）
ASSERT 每个关键流年.六类事件信号检查 == 6（M/P/F/D/L/S全部检查）
ASSERT 每个关键流年.事件类型 != null
ASSERT 每个关键流年.事件类型 IN [婚姻, 升职, 破财, 丧亲, 官非, 疾病, 综合, 不明确]
ASSERT 每个关键流年.信号强度 IN [强, 中, 弱]
ASSERT 每个关键流年.信号强度来源 已标注（H命中数+事件信号命中数）
ASSERT 每个关键流年.确定性标注 IN [✅, ⚠️, ❌]
ASSERT 事件时间线.行数 == 关键流年.count
ASSERT 无绝对断语（不出现"必定""一定""肯定"等词语）

→ 全部通过 → 进入 Step 9
→ 任一不通过 → 补齐后重检

特别注意：
- 若某关键流年 H表全部未命中 → 检查 Step 7 的筛选是否有误
- 若某关键流年事件类型 = "不明确" → 正常，标注 ⚠️ 即可
- 若多个关键流年事件类型相同 → 检查是否为同一事件在不同年份的前兆/高潮/余波
```
