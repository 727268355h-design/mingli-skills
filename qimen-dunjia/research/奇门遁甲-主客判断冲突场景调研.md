# 奇门遁甲 - 主客判断冲突场景调研

## 调研日期：2026-03-28

## 一、三种主客判断方法回顾

### 1. 时辰判主客
- 五阳时（甲乙丙丁戊）利客，利于主动进攻
- 五阴时（己庚辛壬癸）利主，利于等待保守

### 2. 伏吟反吟判主客
- 伏吟局利主不利客（利静不宜动）
- 反吟局利客不利主（利动不利静）

### 3. 天地盘生克判主客
- 天盘星门克地盘宫 → 利客
- 地盘宫克天盘星门 → 利主
- 地盘宫生天盘星门 → 利客
- 天盘星门生地盘宫 → 利主
- 天盘星门与地盘宫比和 → 主客皆可

## 二、核心发现：传统如何处理矛盾信号

### 发现 1：存在明确的优先级层次（✅确定）

来自知乎"炳成老师"长文的关键原文：

> "在确定主客的过程中，**首重大局，次重八门，再看奇仪，最后看八神**。"
> "大局：伏吟利主，反吟利客。"

这意味着：
- **第一优先**：伏吟/反吟（大局层面）
- **第二优先**：八门的天地盘生克关系
- **第三优先**：奇仪（天盘地盘干的五行生克）
- **第四优先**：八神（九天利客，九地利主）

时辰（五阳时/五阴时）在这个体系中属于**辅助参考**，不是最高优先级。

### 发现 2：伏吟反吟不是绝对的，存在"反转"机制（✅确定）

来自知乎原文的关键论述（这是整个调研最有价值的发现）：

> "常理上讲：伏吟局利主不利客，利静不宜动；反吟局利客不利主，利动不利静。这是一个大的局势，**但这不是绝对的**。"

反转机制如下：

**伏吟局的反转**：
> "尤其是遇到伏吟局，你在条件成熟时，一下子做到反主为客，是有可能取胜的。伏吟局，本身就停了很久了。你突然举事，这反而是有利的。"

**反吟局的反转**：
> "还有反吟局，如果你正在动态之中，遇到了反吟局，而且这件事正动态的往前走着，这样你突然停下来，突然静止反而是有利的。"

**触发反转的条件——区分"事前"和"事中"**：
> "伏吟、反吟在什么情况下不利呢？**在事物尚未开始之前**，你还没有干呢，就遇到反吟局了，那时利动不利静，我要干，利于主动进攻。在尚未干之前，就遇到了伏吟局，你就尽量不要动。"
> "**如果是你已经在这个过程中了**，遇到伏吟局，停滞不如反戈一击；遇到反吟局，前进到不如立马站住不动，看看静下来会出现什么效果。实践看，常常会出现比较有利的结局。"

总结：
| 状态 | 伏吟局 | 反吟局 |
|------|--------|--------|
| 事未开始 | 利主/利静（常规） | 利客/利动（常规） |
| 事已进行中 | 反而利突然举事（反转） | 反而利突然停下（反转） |

### 发现 3：时辰利主利客与伏吟冲突时的实战处理（✅确定）

来自原文：

> "还有一种情况，比如我要去求人办事，遇到了利主不利客的时间，如果我不动人家不给办啊。这怎么做？这就如刚才讲的事体进行中的伏吟局一样，要反戈一击才行。这种情况下你可以**先去找此人办事或先去求他，办完后你就等待，不要接二连三的去求**。先去求，再等待。"

> "当利客时，我应该主动进攻，但有时候很明显我先做就会吃亏。这种情况，你可以**间接性的采取措施，做完后就等待**；等一段时间后再做，这样也是有利的。"

这说明实战中，断卦师不是机械地选择"利主"或"利客"，而是**拆解动作节奏**——在矛盾信号下采用"动一下+静一段"的混合策略。

### 发现 4：事物性质（好/坏/中间）是最终裁判（✅确定）

> "好事往前推，利于动；坏事就刹车，利于静。"
> "大原则就是坏事动则无利；好事动则得利。"

这意味着：当多种主客信号矛盾时，最终看的是格局整体指向好还是坏：
- 整体格局好 → 利于动/利客 → 即使时辰利主，也应动
- 整体格局坏 → 利于静/利主 → 即使时辰利客，也应静
- 不好不坏 → 等待

### 发现 5：特殊格局可以覆盖主客判断（✅确定）

多种特殊格局的存在会优先于一般的主客分析：

| 格局 | 效果 | 对主客的覆盖 |
|------|------|-------------|
| 空亡 | 等待，动也没结果 | 无论利主利客，都要等 |
| 刑冲墓迫 | 利于静 | 覆盖"利客/利动"的信号 |
| 合格比格 | 宜动不宜静 | 覆盖"利主/利静"的信号 |
| 五不遇时 | 宜静不宜动 | 覆盖"利客/利动"的信号 |
| 时干入墓 | 宜静不宜动 | 覆盖一般主客判断 |
| 天网四张 | 不利群体，利个人 | 条件性覆盖 |
| 庚格 | 主阻隔但不一定不成 | 修正而非覆盖 |
| 悖格 | 主出乱子但不一定不成 | 修正而非覆盖 |

## 三、主客在断卦中到底多重要？

### 结论：分场景，不是"一刀切"（⚠️综合判断，非单一来源）

**1. 主客起源于军事/对抗场景，在此场景下是核心**

> "主客论的说法，本源于军事。大家都知道奇门遁甲，最早就是应用于军事上的排兵布阵的，所以特别讲求的是敌我双方的主客关系。对于交战双方来说，战争的胜败就取决于是否能分清主客关系。"

**2. "要把奇门通，先把主客论"——但这是策划层面，不是断卦层面**

多个来源强调主客是"策划"的关键：
> "判断主客动静，不仅是关系到事情的成败，而且这是我们预测师如何给人家求测人进行有效策划的关键所在。"

这里的关键区分是：
- **断卦**（判断事情好坏/成败）：主客不是第一优先级，用神分析、格局吉凶才是
- **策划**（指导行动策略）：主客是核心，决定"动还是不动、先动还是后动"

**3. 非对抗场景下的主客**

在求财、感情、健康等日常预测中：
- 主客关系降级为"策略建议"而非"吉凶判定"
- 例如测诉讼时要分主客（因为有对抗方），但测健康时主客概念弱化
- 卢师傅博客的断局13步中，主客判断在"完成用神分析之后"才出现，位于补充判断位置

**4. 没有找到任何名师明确说"主客不重要"**

搜索"不需要分主客""不用分""主客不重要"等关键词，未找到任何名师有此表态。最接近的是：
- 主客在不同场景下权重不同
- 策划（怎么做）比断卦（会怎样）更依赖主客
- 伏吟反吟在某些条件下会反转，说明不是死板规则

## 五、对排盘引擎的设计启示

### 主客判断应该怎么实现

1. **不做单一二值输出**：主客不是简单的"利主/利客"，而是多层信号的综合

2. **分层输出信号**：
   ```
   层级1: 大局（伏吟/反吟）→ 最高权重
   层级2: 八门天地盘生克 → 次高权重
   层级3: 奇仪天地盘生克 → 中等权重
   层级4: 八神（九天/九地等）→ 辅助权重
   层级5: 时辰（五阳时/五阴时）→ 辅助参考
   ```

3. **输出矛盾标记**：当不同层级给出相反结论时，应该标记"矛盾"而非强行统一

4. **输出事态阶段**：区分"事未开始"和"事已进行中"，因为伏吟/反吟的含义会反转

5. **特殊格局覆盖**：空亡、刑冲墓迫等应作为最终覆盖层

    },
    "eight_doors": {            # 八门天地盘
        "relationship": "生/克/比和",
        "signal": "利主/利客/中性"
    },
    "qiyi": {                   # 奇仪天地盘
        "relationship": "生/克/比和",

        "yang_hour": True/False,
        "signal": "利主/利客"
    },
    "overrides": [              # 覆盖层
        # 空亡、刑冲墓迫等特殊格局
    ],
