> 调研日期：2026-03-28

> 前置文件：`奇门遁甲-格局门权重与月令旺衰深挖调研.md`（已识别此矛盾，本次深挖）

## 一、两条古训的原文与出处

### 1.1 "旺凶减凶"派：《三元经》

**完整原文**：
> "大凶天蓬与天芮、小凶天柱及天英。更论五行旺相气，吉凶轻重自然分。**大凶旺相凶却小，小凶旺相号中平。吉星旺相吉无比，若还无气也中平。**"

**讨论对象**：✅ 明确是**九星**（天蓬、天芮、天柱、天英）

**核心逻辑**：凶星在旺相状态时，凶性反而减小；小凶星旺相时接近中平。吉星旺相则大吉，无气则中平。整个口诀是关于**九星**随五行旺衰变化的规律。

**《三元经》的地位**：
- ⚠️ "三元经"在奇门遁甲文献体系中不属于最顶级古籍（如《烟波钓叟歌》《遁甲符应经》等）
- 它被多个培训教材和专业网站引用（乾坤国学院、悟行阁等），被视为有参考价值的古籍诗诀
- 未找到独立的《三元经》全文在线版本，其原文主要通过引用流传
- 无法确认成书年代和作者，不排除是后人归纳的歌诀而非单独成书的经典
- 但其内容与其他古籍（《烟波钓叟歌》等）的旺衰思路一致，不是孤证

### 1.2 "旺凶更凶"派：《奇门遁甲鸣法》

**完整原文**：
> "九星蓬芮冲辅禽，心柱任英吉凶否。大凶大否蓬芮宫，次凶次否英柱度。辅禽心星上吉方，天冲天任小吉路。**凶逢生处旺更凶，吉至克宫喜转怒。吉得生乡大吉昌，凶归弱地凶不顾。凶星失地变吉神，吉宿非暗凶似蠹。**"

**讨论对象**：✅ 同样是**九星**（蓬芮冲辅禽心柱任英）

**核心逻辑**：凶星逢到生处（即被宫位所生，星得力），反而旺更凶；凶星失地（衰弱），则变为吉神。

**《鸣法》的地位**：
- ✅ 《奇门遁甲鸣法》是奇门遁甲的重要典籍，历来口传心授
- 包含入门、开端、用神、正格、辅格、奇格等多章
- 地位不低于《三元经》，同属古籍经典
- 有精校版在360doc等平台流传

### 1.3 《烟波钓叟歌》的说法（最权威古籍之一）

**原文**：
> "大凶篷芮不堪使，小凶英柱不精明。"
> "要识九星配五行，各随八卦考羲经：坎篷星水离英火，中宫坤艮土为营，乾兑为金震巽木，旺相休囚看重轻。与我同行即为相，我生之月诚为旺。废于父母休于财，囚于鬼兮真不妄。"
> "**小凶无气变为吉，大凶无气却平平。**"
> "吉宿更能逢旺相，万举万全功必成。若遇休囚并废没，劝君不必进前程。"

**讨论对象**：✅ 九星

**核心逻辑**：
- 只明确说了"无气"（衰弱）时的变化：小凶无气变吉，大凶无气则平平
- 说了吉星逢旺相则大吉，逢衰则不可用
- **但没有正面说"旺凶"怎样** -- 只说了旺衰决定"重轻"
- 相对中立，可被两派各自引用

**《烟波钓叟歌》的地位**：
- ✅ 奇门遁甲最核心的经典歌诀之一，北宋通人所作，后经明朝罗通增删
- 地位最高，各家必引
- 入公有领域，维基文库有完整收录

## 二、现代"旺凶更凶"派的来源分析

### 2.1 易先生（yixiansheng.com）

**八门旺衰（第20讲）**：
> "一个好的门，吉的能量很低，就好不了多少；一个凶的门，凶的能量很低，它也凶不了多少。同时，一个吉门如果很旺，那就好的很；**如果一个凶门很旺，就凶的很**。"

**讨论对象**：✅ 明确是**八门**

**九星旺衰（第23讲）**：
> "九星旺衰的吉凶判断，是根据事情的性质来决定的。如果这件事情是一件好事，那么越旺越好；如果这件事情是凶事，那么越旺就越凶。"

**讨论对象**：九星，但采用了"旺衰放大论"的解读

### 2.2 其他现代来源

- **网易/搜狐转载**："凶门受生，更加旺相，那就凶上加凶了。" -- 讨论对象是**八门**
- **博客园·占人生命运整理**："天芮本是黑星，得天时，性恶霸道" -- 讨论对象是**九星**但场景是**论命**（不是择吉）
- **知乎综合**："凶星处于旺、相的状态，其破坏性更大" -- 多个知乎文章持此观点

### 2.3 神机阁（shenjige.cn）的关键说法

> "凡遇五行在旺或相的时间，可使吉星更吉，**大凶星减凶**，小凶星则变得一般化，不凶不吉。反之，则可让**凶星更凶**，吉星也变得一般化，甚至转为凶兆。"

**讨论对象**：九星

**这个说法极其重要**：它同时包含了两种逻辑 --
- 旺相时：吉更吉，**大凶减凶**，小凶变中平 --> 与《三元经》一致
- 衰弱时：**凶更凶**，吉变中平 --> 反向推导

这意味着：**旺凶减凶、衰凶更凶**，而不是"旺凶更凶"。

## 三、核心发现：九星与八门的旺衰标准根本不同

### 3.1 九星旺衰口诀

> "我生为旺，同我为相，生我为废，克我为囚，我克为休"

**"我生为旺"的含义**（✅ 多源确认）：九星属性**去生**宫位属性时，星为旺。例如天蓬星属水落入木宫（震/巽），水生木，天蓬星此时为"旺"。

**关键洞察**："我生为旺"意味着星在生出能量、向外输出。这在五行学中叫**泄气** -- 星的能量在被宫位所泄。从一心乾元堂的解读来看："天英星属火落坤宫属土，火来生土，我生为旺，这里天英星落宫的能量是很旺的" -- 表面上"旺"但实际是在做功（输出能量）。

### 3.2 八门旺衰口诀

> "当令为旺，生我为相，我生为休，我克为囚，克我为死"

**"当令为旺"/"同我为旺"的含义**（✅ 多源确认）：门与宫位**五行相同**（或得月令）时为旺。这是标准五行旺衰，旺 = 力量最强。

### 3.3 根本区别

| 维度 | 九星旺衰 | 八门旺衰 |
|------|----------|----------|
| 旺的定义 | "我生为旺"（星生宫） | "当令为旺"/"同我为旺"（门与宫同行或得令） |
| 旺的本质 | 星在向外输出能量（泄气） | 门自身力量最大 |
| 相的定义 | "同我为相"（星与宫同行） | "生我为相"（宫生门） |
| 旺时状态 | 星虽活跃但在做功，能量有出口 | 门力量纯粹增强，无消耗 |

**这就是矛盾的根源**：
- 九星"旺"= 星在泄气输出 --> 凶星虽活跃但能量在被消耗 --> **凶性减弱**（与《三元经》一致）
- 八门"旺"= 门力量本身最大 --> 凶门力量增强 --> **凶性放大**（与易先生一致）

## 四、《鸣法》"凶逢生处旺更凶"的矛盾如何解释？

《鸣法》说的也是九星，但结论是"凶逢生处旺更凶"，这与《三元经》直接矛盾。

### 4.1 "生处"的歧义

"凶逢生处"可以理解为两种含义：
- 解释A：凶星**逢到生它的宫位**（宫生星，如水星落金宫）--> 星被宫所生，力量增强 --> "旺更凶"合理
- 解释B：凶星**去生宫位**（星生宫，如水星落木宫）--> 星在泄气

如果取解释A（"生处"= 星被宫所生），则《鸣法》说的"生处"不等于九星口诀中的"旺"（我生为旺），而是对应"废"（生我为废）或另一种关系。

### 4.2 与《三元经》的可能调和

| 情况 | 《三元经》 | 《鸣法》 | 是否矛盾 |
|------|-----------|---------|----------|
| 星生宫（我生为旺） | 大凶旺相凶却小 | 未明确讨论此情况 | 无矛盾 |
| 宫生星（生我，按口诀为"废"） | 未明确讨论此情况 | 凶逢生处旺更凶 | 无矛盾 |
| 星衰弱（失地） | 无气变吉/中平 | 凶星失地变吉神 | ✅ 一致 |

⚠️ 这个调和建立在"生处"= 宫生星（而非星生宫）的解读上。如果"生处"指星生宫，则两本古籍确实直接矛盾。此处文言文存在歧义，无法100%确认。

## 七、综合结论

### 7.1 最强判断

**两派说的很可能是不同层面的事情，而非真正的古今矛盾。** ✅ 高确信度

| 对象 | 旺时效果 | 依据 | 机制 |
|------|----------|------|------|
| **九星**旺相（星生宫，我生为旺） | 凶性减弱 | 《三元经》"大凶旺相凶却小" + 神机阁"大凶星减凶" | 星在泄气输出，凶力被消耗 |
| **八门**旺相（得令/门与宫同行） | 凶性放大 | 易先生"凶门很旺就凶得很" + 古诀"凶门得生祸难避" | 门力量纯粹增强，凶力放大 |
| 凶星/凶门**衰弱** | 凶性均减弱 | 所有来源一致，无争议 | 能量不足，凶力不显 |

**根本原因**：九星和八门使用了不同的旺衰判断口诀，导致"旺"在两个系统中的物理含义不同：
- 九星的"旺"= 我生为旺 = 星在做功泄气 = 有能量出口 = 凶力被分散
- 八门的"旺"= 同我为旺/当令为旺 = 门力量最大 = 纯粹增强 = 凶力放大

**验证结果**：✅ 与调研结论高度一致。Boss 的假设被多个独立来源佐证。

### 7.3 《鸣法》的遗留问题

《鸣法》"凶逢生处旺更凶"是唯一仍有歧义的古籍说法。如果"生处"指宫生星（而非星生宫），则可与《三元经》调和。如果"生处"指星生宫，则古籍内部本身就有分歧。⚠️ 文言文歧义，无法100%确认。

## 八、对排盘引擎的建议

### 8.1 九星评分

九星旺衰应采用"旺减凶"逻辑（《三元经》派）：

```python
# 九星旺衰系数 -- 旺相减凶，衰弱更凶
STAR_WANG_SHUAI_COEFF = {
    # 吉星：旺放大，衰减弱（无争议）
    "吉_旺": 1.3,   # 吉星旺相吉无比
    "吉_相": 1.1,
    "吉_休": 0.8,
    "吉_囚": 0.6,
    "吉_废": 0.4,   # 若还无气也中平
    # 凶星：旺减凶，衰更凶（三元经派）
    "凶_旺": 0.6,   # 大凶旺相凶却小
    "凶_相": 0.7,   # 小凶旺相号中平
    "凶_休": 1.0,   # 基准凶力
    "凶_囚": 1.2,   # 衰时凶力不减反增
    "凶_废": 0.3,   # 大凶无气变为吉 / 却平平
}
```

### 8.2 八门评分

八门旺衰应采用"旺更凶"逻辑（易先生/标准五行派）：

```python
# 八门旺衰系数 -- 旺放大一切，衰减弱一切（无争议）
DOOR_WANG_SHUAI_COEFF = {
    "旺": 1.3,  # 统一放大（吉更吉，凶更凶）
    "相": 1.1,
    "休": 0.8,
    "囚": 0.6,
    "死": 0.4,
}
```

在涉及凶星旺相的断语中，标注这个分歧但给出主判断：

```
"天蓬星落宫旺相 -- 按《三元经》'大凶旺相凶却小'，凶力有所减弱。"
```
