## 案例一：已婚男士测婚姻（含空亡/入墓/击刑 + 完整验证）

**来源**：知乎 https://zhuanlan.zhihu.com/p/609689338

### 1. 起卦时间
2015年7月20日 巳时

### 2. 问题
一位1973年（癸丑年）出生的已婚男士，咨询婚姻问题，未透露任何情况。

### 3. 盘面关键信息

**干支**：
- 年：乙未年（乙为太岁）
- 月：癸未月（癸为月干，未月土旺）
- 日：丁酉日（日干丁）
- 时：乙巳时（时干乙）

**四害标注**：
- 空亡：辰空、巳空、寅空、卯空
- 六仪击刑：戊六击、壬六击、癸六击
- 入墓：壬入墓、乙入墓
- 马星：巳时马星为亥，冲巳，壬入墓被冲掉

**用神体系**：
- 男士年命 = 癸（1973癸丑）
- 婚姻 = 六合
- 妻子 = 乙
- 丈夫 = 庚
- 女情人 = 丁
- 男情人 = 丙

### 4. 断卦分析过程

**(a) 夫妻感情（乙庚关系）**
- 妻子乙在坤土宫，丈夫庚在坎水宫
- 土克水 → 妻子对丈夫有所不满

**(b) 夫妻对婚姻态度（乙/庚与六合关系）**
- 六合落宫与庚落宫：土克水 → 婚姻拒绝丈夫，婚姻难维持
- 六合落宫与乙落宫：对冲但比和 → 有冲突但目前还和谐
- 结论：婚姻出现问题，丈夫行为让妻子不满

**(c) 第三者分析（丙丁与乙庚关系）**
- 丁（女小三）与庚落宫：金生水 → 有女性追求丈夫
- 乙（妻子）与丙（男小三）：对冲比和 → 不构成问题
- 结论：问题在丈夫方面，与其他女性过密交往

**问测者此时承认确有一个女相好，决定离婚，问妻子同意离婚的条件。**

**(d) 离婚条件 -- 财产**
- 癸为男方年命，戊癸合 → 戊代表女方及钱财
- 戊落震木宫，"戊六击"→ 破财
- 戊落宫克乙所在坤土宫 → 要破财解决问题

**(e) 离婚条件 -- 子女**
- 时干（代表子女）与妻子乙同宫 → 孩子愿意跟母亲
- 子女与男方关系为相克 → 不喜欢跟父亲

**(f) 赔偿数目**
- 戊落震木宫，数字3、4、8
- 阴局取大数4和8，逢空减半 → 2到4
- 结合财力情况 → 200万到400万

### 5. 实际结果验证

> 大约一个月后，双方协议离婚。一个女儿跟了母亲，丈夫给了妻子300万。

**验证要点**：
- 丈夫有外遇 ✅
- 问题出在丈夫方 ✅
- 女儿跟母亲 ✅
- 赔偿金额300万（在预测的200-400万区间内）✅
- 约一个月后离婚 ✅

**涉及的断卦规则**：乙庚关系（夫妻）、六合（婚姻）、丙丁（第三者）、戊癸合（年命对应）、空亡（逢空减半取数）、六仪击刑（破财）、入墓

## 案例二：奇门断单田芳去世原因（天芮 + 死门 + 完整验证）

**来源**：易先生 https://www.yixiansheng.com/article/1640.html

### 1. 起卦时间
2018年9月11日 15:30（北京时间）
- 八字：戊戌年、辛酉月、丙午日、丙申时
- 日空：寅卯
- 时空：辰巳

### 2. 问题
分析评书泰斗单田芳的死因（事后回溯分析）。

### 3. 盘面关键信息

**关键参数**：
- 太岁戊在兑金宫
- 月干辛在艮土宫（酉月旺金）
- 日干/时干丙在坎水宫
- 年命己在离火宫（甲戌年生人）
- **天芮病星：离火宫**
- 离火宫天干：己（临官之地、旺）、壬（胎地）、丁（临官之地、旺）
- 天心星（西医）：兑金宫，临辛金
- 死门：巽木宫（丁帝旺、庚长生）

### 4. 断卦分析过程

**(a) 病位判断**
- 己属胃，戊己是脾胃系统
- 己在临官之地 → 胃疾严重
- 丁属心脏，在离火宫 → 心血管问题加强

**(b) 治疗可能性**
- 天心星生天芮病星 → 西医可缓解但难根治
- 天心临辛金临惊门 → 西医需做手术

**(c) 死亡信息**
- 死门在巽木宫，丁帝旺 → 死于心脏问题
- 庚金长生伴随 → 呼吸系统也出问题
- 综合分析 → 血阻可能性大，心脏遇到重大阻力

### 5. 实际结果验证

**医学记录核实**：
- 2000年患胃癌，被切掉1/3的胃 ✅（对应己/戊脾胃系统）
- 2018年9月11日下午因心脏衰竭死亡 ✅（对应丁/死门分析）
- 有呼吸系统问题 ✅（对应庚金长生）

**涉及的断卦规则**：天芮星（病星）、死门、天心星（西医）、三奇六仪对应人体部位、宫位旺衰（临官/帝旺/长生/胎地）

## 案例三：寻找丢失钥匙（含空亡 + 入墓 + 完整验证）

**来源**：知乎 https://zhuanlan.zhihu.com/p/615417332

### 1. 起卦时间
2016年12月11日 19:18（庚戌时）

### 2. 问题
有位同学问测：某一个机要仓库的钥匙找不到了。

### 3. 盘面关键信息

**干支**：
- 年：丙申年（丙为太岁）
- 月：庚子月（庚为月干）
- 日：丁卯日（日干丁）
- 时：庚戌时（庚时干）

**四害标注**：
- 空亡：戌空、亥空、寅空、卯空
- 入墓：乙入墓、丙入墓
- 无六击、无伏吟反吟

**马星**：戌时马星在申，对着寅冲过去，乙入墓被去掉

**用神**：
- 失主 = 日干丁
- 失物 = 时干庚
- 小偷 = 玄武
- 钥匙类象 = 丁

### 4. 断卦分析过程

**(a) 排除偷盗**
- 玄武与时干不是克制关系，也非同宫 → 不是小偷偷盗

**(b) 失主与失物关系**
- 时干庚金在兑金宫，日干丁在乾金宫
- 二者比和 → 失物可以找到

**(c) 确定钥匙位置**
- 地盘丁在兑金宫，临休门 → 可能在休闲地方，周围有树林
- 天盘丁在乾金宫：
  - 乾金宫代表圆形的东西
  - 临天柱代表金属 → 可能在圆形金属桶里
  - 临九地 → 犄角旮旯处
  - **逢空 → 要看对宫信息**
- 对宫在东南方向：死、阴、己土、戊土 → 东南方向死角、脏乱杂物堆放处

**(d) 确定找到的应期**
- 戌空、亥空
- 预测时间为戌时（已填空）→ 亥时不空时可以找到

### 5. 实际结果验证

> 该名同学反馈：大约在21:50左右（亥时），在东南方向一层食堂的车库旁的一个纸箱里头，找到了这把钥匙。有可能清洁员见到钥匙以后就放在纸箱上面了。

**验证要点**：
- 方位：东南方向 ✅（对宫信息指向东南）
- 环境：食堂车库旁纸箱 ✅（对应"死角""脏乱杂物处"）
- 时间：亥时（21:50）✅（空亡应期准确）
- 非偷盗，是被人移动 ✅

**涉及的断卦规则**：空亡（逢空看对宫、空亡填实应期）、入墓（乙入墓被马星冲掉）、日干时干关系（比和可找到）、玄武（排除偷盗）、天盘地盘丁（确定位置）

## 案例四：李姓朋友测疾病（天芮 + 击刑 + 死门/白虎 + 28天死亡验证）

**来源**：诸夏周易文化研究院 http://www.zxzywh.com/1110.html

### 1. 起卦时间
文中未明确给出完整时间，但注明"从发病的第一时间起局"。患者为1962年壬寅年生人。

### 2. 问题
门市旁边的一个李姓朋友发病，预测其疾病情况及预后。

### 3. 盘面关键信息

**天芮星**：落坤宫（属土）
- 坤宫在体内主脾胃、食道、腹部等消化系统
- 死门临坤宫
- 白虎临死门
- 庚+庚（太白同宫）
- 庚+癸（大格）
- 癸+癸（天网四张）
- 值符空亡

**年命**：壬寅年生人
- 年命落巽宫
- 处于击刑
- 临杜门

**四干格局**：年月日时四干格局都有问题，四干两辛、两癸不吉之象

### 4. 断卦分析过程

**(a) 天芮星分析**
- 天芮落坤宫属土 → 主脾胃消化系统疾病
- 死门 → 病难愈
- 值符空亡 → 年命不保，"病空人空"
- 白虎临死门、庚 → 典型癌症信号

**(b) 格局分析**
- 庚代表阻隔压力，体内代表大肠、韧带，测病多代表癌症
- 癸代表神经、肾、血液、泌尿生殖系统
- 庚+庚太白同宫 → 体内严重阻隔
- 庚+癸大格 → 身体有大变化
- 癸+癸天网四张 → 行动受制约

**(c) 年命分析**
- 62年壬寅年生人，落巽宫
- **击刑** → 不吉征兆
- 临杜门 → 闭塞不通
- 木为肝，壬为膀胱、血液

### 5. 实际结果验证

> 从事发到死亡共计28天。事后证实是肝硬化，胃、肾都坏死了。事实是上呕血下拉血。最后因为胃出血，大肠拉血，失血过多导致昏迷离世。

**验证要点**：
- 肝硬化 ✅（年命落巽宫，木为肝）
- 胃坏死 ✅（天芮落坤宫主脾胃）
- 肾坏死 ✅（癸代表肾）
- 上呕血下拉血 ✅（庚+癸大格，身体大变化）
- 28天死亡 ✅（值符空亡、白虎临死门庚 = 极凶）

**涉及的断卦规则**：天芮星（病星落宫定病位）、死门（病难愈）、白虎（严重病灾）、击刑（年命受刑）、空亡（值符空亡年命不保）、庚+癸大格、癸+癸天网四张、庚+庚太白同宫

## 案例五：庚申年女士测疾病（天芮 + 空亡 + 详细验证）

**来源**：诸夏周易文化研究院 http://www.zxzywh.com/1110.html

### 1. 起卦时间
2016年4月20日早间，报"现在"时间。

### 2. 问题
一位庚申年生的女士预测疾病。

### 3. 盘面关键信息

**年命庚**：
- 落沐浴败地
- 乘白虎 → 疾病刑伤
- 临死门 → 身体疼痛
- 庚代表骨头，丙代表炎症

**天芮病星**：落兑宫（外盘）
- 兑为腰肋、胸部、肺、气管
- 戊为胃、鼻子
- 辛为肺、肋、足（辛在禄地，不算病）
- 癸为肾、神经、血液、四肢
- 对宫有丁（手术）、己（腹、脾脏）
- 九地 → 时间已经很长

**乙奇（药物）**：空亡，乘腾蛇 → 虚假的药，不起作用

**天心（西医）**：被病星所生 → 治疗不可取

### 4. 断卦分析过程

**(a) 症状判断**
- 年命庚+白虎+死门 → 疾病刑伤，身体疼痛
- 庚代表骨头、丙代表炎症 → 上身疼痛（头、肩）
- 天芮落兑宫 → 最大的病为癸（肾脏、神经、生殖疾病）和戊（胃）
- 兑宫在虚地 → 阴虚之感
- 九地 → 已是长期病症
- 起因为"癸+开" → 因生孩子开刀引起的后遗症

**(b) 治疗方案**
- 天心被病星所生 → 西医不可取
- **乙奇空亡**，乘腾蛇 → 当前中药无效
- 能治病的只有火 → 建议针灸、拔火罐、火性中药

### 5. 实际结果验证

反馈内容：
> - 头有时疼，有肩周炎 ✅（对应庚+丙上身疼痛）
> - 确实阴虚症，时常腰酸背痛、四肢疼痛 ✅（对应癸/兑宫虚地）
> - 胃由于长期吃药也不好了 ✅（对应戊为胃）
> - 她本人是中医，但就是不敢下药 ✅（对应乙奇空亡）
> - 听完建议后决定尝试火性中药治疗 ✅（对应"能治病的只有火"）

**涉及的断卦规则**：天芮星（病星定位）、死门+白虎（严重程度）、空亡（乙奇空亡=药物无效）、九地（病程长短）、三奇六仪对应人体部位

### 3. 盘面关键信息
