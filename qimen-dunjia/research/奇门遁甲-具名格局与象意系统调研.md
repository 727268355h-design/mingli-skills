## 一、具名格局

### 1. 吉格

#### 1.1 青龙返首
- **条件**：天盘甲子戊 + 地盘丙奇（即 **戊加丙**）
- **原理**：甲木为青龙，丙火为甲木之子，母子相顾；丙火克庚金，救护元帅甲木
- **吉凶**：大吉。宜就职、诉讼、迁移、求财、建造，百事皆吉
- **例外**：遇门克宫或地盘震三宫（子卯相刑）则吉变凶
- **确定性**：✅ 多源一致（阐微堂 + 知乎 + 乾坤国学院）

#### 1.2 飞鸟跌穴
- **条件**：天盘丙奇 + 地盘甲子戊（即 **丙加戊**，与青龙返首相反）
- **原理**：丙火为南方朱雀，回到母亲甲木身边，鸟归巢穴；木火相生
- **吉凶**：大吉。宜就职、求财、诉讼、建造、婚姻，百事吉
- **确定性**：✅ 多源一致

#### 1.3 玉女守门
- **条件**：天盘值使门 + 地盘丁奇所在之宫（即 **值使门加临丁奇**）
- **原理**：丁奇为玉女，值使门到达丁奇位置
- **吉凶**：吉。利于婚恋、相会、宴喜娱乐、经商、行军埋伏
- **确定性**：✅ 多源一致（阐微堂 + 知乎 + 乾坤国学院）

#### 1.4 三奇得使（仪奇得使）
- **条件**：天盘乙/丙/丁三奇加临地盘值使门所在之宫
  - 乙奇临地盘甲戌己或甲午辛
  - 丙奇临地盘甲子戊或甲申庚
  - 丁奇临地盘甲辰壬或甲寅癸
- **吉凶**：吉。逢贵人帮扶，事向理想方向发展。临吉门应吉大，临凶门应吉小
- **确定性**：✅ 多源一致

#### 1.5 三奇贵人升殿
- **条件**：
  - 乙奇到震宫 = 日出扶桑
  - 丙奇到离宫 = 月照端门
  - 丁奇到兑宫 = 星见西方
- **前提**：须有开/休/生三吉门之一，无门迫、奇无墓
- **吉凶**：大吉。宜远征、出行、婚嫁、建造、求财、迁移
- **确定性**：✅ 多源一致

### 1.6 九遁

#### 天遁
- **条件**：天盘丙奇 + 地盘丁奇 + 人盘生门，三者同宫
- **原理**：丙丁双奇会合生门
- **吉凶**：大吉。变无利为有利，把握机遇
- **确定性**：✅ 多源一致（百度百科 + 搜狐 + 知乎）

#### 地遁
- **条件**：开门 + 乙奇 + 地盘六己，三者同宫
- **原理**：开门合乙奇临地盘己
- **吉凶**：大吉。经商求学借助地力
- **确定性**：✅ 多源一致

#### 人遁
- **条件**：天盘丁奇 + 人盘休门 + 神盘太阴，三者同宫
- **原理**：丁奇合休门得太阴之助
- **吉凶**：大吉。谋职升迁展示才华
- **确定性**：✅ 多源一致

#### 风遁
- **条件**：乙奇 + 开/休/生三吉门之一 + 临巽宫
- **原理**：乙奇合吉门临风位（巽为风）
- **吉凶**：吉。以动制动，宜变更策略
- **确定性**：✅ 2 源确认（百度百科 + 搜狐）

#### 云遁
- **条件**：乙奇 + 开/休/生三吉门之一 + 地盘六辛
- **原理**：乙奇合吉门临辛位
- **吉凶**：吉，但运势不稳。宜稳中求胜
- **确定性**：✅ 2 源确认

#### 龙遁
- **条件**：天盘乙奇 + 人盘开/休/生三吉门之一 + 地盘坎一宫（或六癸）
- **原理**：乙木为龙，入坎水为龙入海
- **吉凶**：大吉。升学谋职前景明
- **确定性**：✅ 多源一致

#### 虎遁
- **条件**：乙奇 + 休门 + 地盘六辛 + 艮宫
- **原理**：虎退为隐，杀机更深
- **吉凶**：凶遁。暗藏危机，需虚张声势
- **确定性**：✅ 2 源确认

#### 神遁
- **条件**：天盘丙奇 + 九天 + 生门，三者同宫
- **原理**：丙合九天临生门
- **吉凶**：吉。升学打基础有利，不宜乔迁婚嫁
- **确定性**：✅ 2 源确认

#### 鬼遁
- **条件**：乙奇 + 杜门 + 九地，三者同宫（或生门 + 丁奇 + 九地临艮宫）
- **原理**：阴遁之象，暗中行事
- **吉凶**：凶遁。需用计谋
- **确定性**：⚠️ 条件说法有分歧（百度百科与搜狐版本略异），取交集
