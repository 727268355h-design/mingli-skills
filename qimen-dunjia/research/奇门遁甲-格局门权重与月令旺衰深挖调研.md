## 知识点一：十干克应格局 vs 八门 冲突时的权重

### 1.1 各来源具体说法汇总

#### 来源A：《奇门遁甲统宗》（古籍原文）

**核心原文**："奇门选择以会合为上吉，若有门无奇，犹为可用；有奇无门，则当另择矣。"

**解读**：门的优先级明确高于奇（三奇乙丙丁）。有吉门无三奇仍可用，有三奇无吉门则需另择。但此处讨论的是"奇"与"门"的关系，不是"十干克应格局"与"门"的关系。 ✅ 古籍原文可考

#### 来源B：《烟波钓叟歌》（古诀）

**原文一**："吉门偶尔合三奇，值此须云百事宜。更合从旁加检点，余宫不可有微疵。"
**原文二**："奇与门兮共太阴，三般难得总加临。若还得二亦为吉，举措行藏必遂心。"
**原文三**："三奇得使诚堪使，六甲遇之非小补。乙逢犬马丙鼠猴，六丁玉女骑龙虎。"
**原文四**："大凶无气变为吉，小凶无气亦同之。"

**解读**：烟波钓叟歌讨论的是门+奇+太阴三者的组合关系，核心思路是"能得几吉就取几吉"，并非简单的"格局>门"或"门>格局"。格局（十干克应）在歌诀中是作为事态的"状态描述"，而非独立于门之外的否决因素。 ✅ 古籍原文可考

#### 来源C：辰星网《奇门遁甲全集》教材

**原文**："诸吉中应以吉门为主，吉星次之，吉神辅之。得门而不得奇之方位，也可算吉；得奇而不得门之方位，则不为吉。奇、门均无，则应属于凶方了。"

**解读**：明确给出优先级——**门 > 奇 > 星 > 神**。但注意这里的排序是"门 vs 奇"，十干克应格局（天地盘干组合）是另一个维度，并未直接纳入这个序列。 ✅ 多处教材复述此原则

#### 来源D：阐微堂（郁知）

**关键论述**：
- "既不能遇到一个吉格就断吉，又不能遇到一个凶格就断凶，要综合判断，全面分析。"
- "三奇得使如所临值使门是吉门，则应吉大；临凶门，应吉小。"
- 引用《烟波钓叟歌》但强调"需全盘分析而非孤立判断"。

**解读**：格局是"定性参考因素"而非"绝对判据"。格局的效力被门的吉凶显著调节。 ✅ 原作者署名论述

#### 来源E：知乎文章（格局法深层详解）

**关键论述**：
- "格局是'表象'，生克是'本质'。"
- "吉格并非绝对吉，若临死门、玄武，吉事也可能暗藏隐患。"
- "凶格并非绝对凶，若得三奇生助，或临生门、开门，可逢凶化吉。"

**解读**：格局被明确定位为"表象层"，其吉凶需要门、神等"本质层"来修正。✅ 与来源D一致

#### 来源F：博客园（奇门遁甲占人生命运整理）

**关键原文**：
- "格局要分轻重，命运验其得失"
- "旺相得奇，且富且贵；死囚墓绝，极贱极贫"
- "天芮本是黑星，得天时，性恶霸道；失天时，牧隶庸工"

**解读**：这里的"格局"偏向"全盘格局"而非"十干克应"。核心逻辑是旺衰决定格局的实际表现——同一个凶星/凶格，得时和失时完全不同。⚠️ 整理稿，非原始古籍

#### 来源G：易德轩（三奇得使隐密）

**关键论述**：
- "三奇得使便是天盘三奇加地盘值使，虽未逢吉门，办事亦堪用"——三奇得使即使没有吉门也能用。
- "这里的'得使'二字尤其要注意。只有'得使'，才能为吉。"
- 凶格（如青龙逃走、荧入白、雀投江）的转化条件：**"如遇本旬直符同临其上，方可用之而吉也"**——需要直符同临才能化解。

**解读**：三奇得使的转化力有明确条件限制。转化凶格不是无条件的，需要直符的同时参与。这比其他来源更严格。⚠️ 一家之言，但论证逻辑清晰

### 1.2 综合分析：格局到底是"定性参考"还是"核心判据"？

**多源交叉验证后的结论**：

格局（十干克应）的定位是 **"重要的定性参考，但不是独立的核心判据"**。具体而言：

1. **格局是"状态描述层"** ✅（来源B/D/E一致）
   - 格局描述事态的运行趋势和特征（如"青龙返首"描述顺势而为的态势）
   - 但格局本身不直接决定最终吉凶

2. **门的优先级确实高于格局** ✅（来源A/C/D/E一致）
   - 古籍《奇门遁甲统宗》和《烟波钓叟歌》均将门置于奇之上
   - 现代教材普遍继承"吉门为主"的原则
   - 格局（十干克应）更接近"奇"的层面而非"门"的层面

3. **但格局不可忽略** ✅（全部来源一致）
   - 吉格临凶门，吉打折（"应吉小"）
   - 凶格临吉门，凶减轻（"可逢凶化吉"）
   - 格局与门的关系是"乘法修正"而非"加法叠加"

4. **一票否决格局例外** ✅（来源C明确）
   - 五不遇时、六仪击刑、三奇入墓等属于"绝对凶格"
   - 这些格局的否决权高于吉门——"百事皆凶，得奇得门也不可用"

### 1.3 "三奇得使"转化凶格的程度

**多源交叉验证后的结论**：

| 条件 | 转化效果 | 来源 |
|------|----------|------|
| 三奇得使 + 吉门 | 大吉（"应吉大"） | 来源D（阐微堂） |
| 三奇得使 + 凶门 | 小吉（"应吉小"，"不是大吉但至少没凶"） | 来源D |
| 三奇得使 + 直符同临 | 可化解本为凶的组合（如青龙逃走） | 来源G（易德轩） |
| 三奇得使 + 无直符 + 本身凶组合 | **不能化解**，仍为凶 | 来源G |

**确定性**：三奇得使能转化凶格，但转化程度有限且有条件。临吉门时效果显著，临凶门时仅小吉。对于本身就是凶组合的情况（如乙+辛=青龙逃走），需要直符同临才能化解。 ⚠️ 来源G的"直符同临"条件为一家之言，其他来源未提及此限制。

### 1.4 对排盘引擎的实现建议

基于调研结论，建议评分框架的处理逻辑为：

```
if 触发一票否决格局（五不遇时/六仪击刑/三奇入墓等）:
    最终 = -99  # 格局否决权 > 门
elif 格局为凶 and 门为吉:
    # 门修正格局：凶格减凶，但不完全翻转
    宫分 = 格局分 * 0.5 + 门分 * 1.0 + 星分 + 神分
elif 格局为吉 and 门为凶:
    # 门修正格局：吉格打折
    宫分 = 格局分 * 0.5 + 门分 * 1.0 + 星分 + 神分
elif 三奇得使:
    if 临吉门: 格局分 += 2  # "应吉大"
    elif 临凶门: 格局分 += 1  # "应吉小"
else:
    宫分 = 格局分 + 门分 + 星分 + 神分  # 正常相加
```

## 知识点二：月令旺衰能否"救"凶格

### 2.1 各来源具体说法汇总

#### 来源A：《三元经》（古籍诗诀） -- 最关键发现

**完整原文**：
> "大凶天蓬与天芮、小凶天柱及天英。更论五行旺相气，吉凶轻重自然分。**大凶旺相凶却小，小凶旺相号中平。吉星旺相吉无比，若还无气也中平。**"

**解读**：这是本次调研最重要的发现。《三元经》古诀明确说"大凶旺相凶却小"——大凶星（天蓬、天芮）在旺相状态时，凶性反而减小。这与"旺凶更凶"的说法直接矛盾。 ✅ 古籍原文可考

#### 来源B：《烟波钓叟歌》（古籍诗诀）

**原文**："大凶无气变为吉，小凶无气亦同之。"

**解读**：与《三元经》一致——凶格/凶星失去能量（无气=衰弱）时转为吉。反推：有气（旺相）时凶性应存在。但歌诀只说"无气变吉"，并未明说"旺相则凶更重"。 ✅ 古籍原文可考

#### 来源C：乾坤国学院培训教材

**原文**："大凶无气变为吉，小凶无气亦同之。" + "旺相休囚看重轻" + "以月令为主，有用神与有关事物各自的旺相休囚，从而决定事物是成是败。"

**解读**：与《烟波钓叟歌》一致。月令是"主要参数"，旺衰决定吉凶"重轻"（程度），但未明确说旺相凶格变更凶还是减凶。 ✅ 培训教材

#### 来源D：悟行阁（深圳）

**原文**："大凶旺相凶却小，小凶旺相号中平。吉星旺相吉无比，若还无气也中平。"（引自《三元经》）

另外补充："四时旺衰决定了事体外部环境的盛衰情况，外部的旺衰对局内的旺衰起到增减调节的作用。" + "时空旺衰一致时吉凶立验。"

**解读**：与来源A完全一致，明确认可"大凶旺相凶却小"。并补充了"时空一致"的概念。 ✅ 引用古籍一致

#### 来源E：易先生（八门旺衰判断方法）

**原文**：
- "一个好的门，吉的能量很低，就好不了多少；一个凶的门，凶的能量很低，它也凶不了多少。"
- "同时，一个吉门如果很旺，那就好的很；**如果一个凶门很旺，就凶的很**。"

**解读**：这是明确的"旺凶更凶"论述！与《三元经》的"大凶旺相凶却小"直接矛盾。易先生的逻辑是旺衰=能量放大器，放大原本的吉凶属性。 ⚠️ 现代教学者论述

#### 来源F：易先生（九星旺衰应用）

**原文**："九星旺衰的吉凶判断，是根据事情的性质来决定的。如果这件事情是一件好事，那么越旺越好；如果这件事情是凶事，那么越旺就越凶。"

**解读**：进一步确认易先生持"旺衰放大论"立场。旺衰不改变方向，只放大强度。 ⚠️ 现代教学者论述

#### 来源G：博客园（奇门遁甲占人生命运整理）

**原文**：
- "旺相得奇，且富且贵；死囚墓绝，极贱极贫"
- "天芮本是黑星，得天时，性恶霸道；失天时，牧隶庸工"
- "年月日时要旺相" + "以月令为纲来分析其旺衰"

**解读**：天芮（大凶星）得时后"性恶霸道"——明确是"旺凶更凶"的立场。得时的凶星权势更大、凶性更强。但此处讨论的场景偏向"论命"（占人生命运），而非"择吉"。 ⚠️ 整理稿，且场景有别

#### 来源H：网易/搜狐转载文章

**关键论述**："凶门受生，更加旺相，那就凶上加凶了。"

**解读**：明确的"旺凶更凶"论述。 ⚠️ 转载文章，出处不详

#### 来源I：辰星网《奇门遁甲全集》

**原文**："如惊门为凶门属金，值夏、月到离宫为囚时囚地，则凶者不能为凶。"

**解读**：凶门衰弱时凶性不显。这与所有来源一致（"衰凶减凶"无争议）。但未正面回答"旺凶"的问题。 ✅ 教材

### 2.2 核心矛盾：两个对立的古训

本次调研发现了一个此前未被明确识别的**根本性矛盾**：

| 立场 | 核心论述 | 来源 | 类型 |
|------|----------|------|------|
| **"旺凶减凶"派** | "大凶旺相凶却小" | 《三元经》（来源A/D） | 古籍诗诀 |
| **"旺凶更凶"派** | "凶门很旺，就凶得很" | 易先生（来源E/F）、博客园（来源G）、网易/搜狐（来源H） | 现代教学 |
| **"衰凶减凶"** （无争议） | "大凶无气变为吉" | 《烟波钓叟歌》（来源B）、辰星网（来源I） | 古籍+教材 |

### 2.3 尝试调和：两派可能在说不同的事

深入分析后，我认为两派的矛盾**可能**源于讨论对象和应用场景的不同：

**解释一：九星 vs 八门 的区别**

- 《三元经》说的是**九星**（天蓬、天芮为大凶星）旺相时"凶却小"
- 易先生说的是**八门**（凶门旺相）"凶得很"
- 可能九星和八门遵循不同的旺衰逻辑？

这个解释有一定道理：九星代表"天时"（外部环境、趋势），八门代表"人事"（具体行动、选择）。天时层面的凶星得旺，可能意味着"凶的态势虽大但已充分展现，不再隐蔽"（类似"明枪易躲"）；而人事层面的凶门得旺，则是"行动上的凶险被加强"。

**解释二：择吉 vs 断事 的区别**

- 择吉场景：选时选方用，关心"这个时间方位能不能用"。凶星旺相 = 凶的能量充足但已显化，可以识别和规避 --> "凶却小"（因为能看到和应对）
- 断事场景：分析已发生或将发生的事，关心"事态往哪个方向发展"。凶门旺相 = 凶的力量在增强 --> "凶更凶"

**解释三："旺"的含义不同**

- 《三元经》中的"旺相"可能指九星在当月/当季的五行状态（时令旺衰）
- 易先生的"旺"可能指门落入生扶之宫（空间旺衰）
- 时令旺衰和空间旺衰可能有不同效果

⚠️ 以上三个解释均为推测性调和，没有任何古籍或权威教材明确给出这种区分。

### 2.4 月令到底能不能"救"凶格？

**多源交叉验证后的结论**：

**不能单独救。** 这一点所有来源高度一致。但具体机制存在分歧：

| 场景 | "旺凶减凶"派的判断 | "旺凶更凶"派的判断 |
|------|-------|-------|
| 凶格 + 用神得月令旺相 | 凶性减弱（"凶却小"），月令在一定程度上缓解凶格 | 凶性增强，月令放大了凶的能量 |
| 凶格 + 用神月令休囚 | 凶性更弱（"无气变吉"），接近吉 | 凶性减弱（"能量低凶不了多少"） |
| 吉格 + 用神得月令旺相 | 大吉（"吉星旺相吉无比"） | 大吉（能量放大吉性） |
| 吉格 + 用神月令休囚 | 中平（"若还无气也中平"） | 吉性减弱 |

**唯一共识**：月令决定吉凶的"重轻"（程度），但不改变门和格局本身的吉凶属性。一票否决格局无论月令如何都不可救。

### 2.5 "旺凶更凶"在天芮星论命场景的特殊案例

来源G提供了一个有趣的具体案例视角：

> "天芮本是黑星，得天时，性恶霸道；失天时，牧隶庸工"

这说的是"论命"场景——天芮得时的人，成为霸道的权臣（如曹操、董卓）。这里的"旺凶更凶"不是说命运变差，而是说凶的**力量**变强——有能力行凶。这与择吉场景（"这个方位能不能用"）的逻辑完全不同。

**重要洞察**："旺凶更凶"可能更适用于论命/断事，而"大凶旺相凶却小"可能更适用于择吉。 ⚠️ 推测性结论
