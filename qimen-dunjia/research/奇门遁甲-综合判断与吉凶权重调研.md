## 一、五层竖看的优先级排序

### 1.1 总体优先级（从高到低）

| 优先级 | 层 | 代表 | 角色 | 说明 |
|--------|------|------|------|------|
| **最高** | 八门 | 人和 | 人事吉凶的核心参数 | "诸吉以吉门为主，吉星次之，吉神辅之" |
| 次高 | 十干克应（格局） | 天地盘干组合 | 事态具体走势 | 81格局直接定性吉凶 |
| 第三 | 九星 | 天时 | 机遇、形势、大环境 | "大事看星" |
| 第四 | 八神 | 神助 | 超自然辅助力量 | 起辅助参考作用 |
| 基础 | 宫位五行+旺衰 | 地利 | 能量基础 | 月令旺衰决定真假吉凶 |

### 1.2 关键原则

**"得门不得奇，也算吉；得奇不得门，不为吉"** -- 这是传统奇门最核心的优先级判断。吉门的重要性高于三奇（乙丙丁）。

**分事项调整优先级**：
- 测天时 --> 以九星为主（如测下雨看天蓬、天柱）
- 测人事 --> 以八门为主（如测工作看开门）
- 测地理风水 --> 以九宫为主
- 测六亲 --> 以年月日时干为主

### 1.3 旺衰是终极裁判

所有吉凶判断的前提是**旺衰**。教材原文："得时又得地，为旺相，才是真正的吉。如果生门临震3宫、巽4宫，木来克土，生门受制...则吉门也就不吉了。" 反之，"凶格若逢休囚死衰之时之地，则凶门也就不能逞凶了"。

## 二、五层冲突时的综合判定规则

### 2.1 "一票否决"规则（绝对凶格）

以下格局具有**绝对否决权**，无论其他层多吉，遇之皆凶：

| 否决格局 | 判定条件 | 效果 |
|----------|----------|------|
| **五不遇时** | 时干克日干（阳克阳、阴克阴） | "百事皆凶，得奇得门也不可用" |
| **六仪击刑** | 六仪落于相刑宫 | "极凶，即使六仪为直符，也不可用" |
| **三奇入墓** | 三奇落入墓库宫 | "百事不宜，谋事尽休" |
| **天网四张** | 特定格局条件 | "天网四张不可挡，此时行事有灾殃" |
| **八门伏吟** | 八门归原位不动 | "万事皆凶，得奇也不可用" |
| **九星反吟** | 九星全部反位 | "诸事皆凶" |

### 2.2 非绝对冲突时的综合判定方法

当不同层结果冲突但未触发一票否决时，采用**"因素计数+旺衰定性"**的方法：

**步骤一：竖看单宫，计数吉凶因素**
> "分析用神所落之宫各因素之间的生克情况。星、门、神、格局是吉是凶。吉的因素多则有利，否则为凶，不利。"

**步骤二：门宫关系定性**
门与所落宫的五行关系有明确吉凶歌诀：
- "吉门相生有大利" -- 吉门+宫生门 = 大吉
- "吉门被克吉不就" -- 吉门被宫克 = 吉事难成
- "凶门得生祸难避" -- 凶门+宫生门 = 祸难避免
- "凶门克宫事更凶" -- 凶门克宫 = 雪上加霜
- "吉门克宫吉不就" -- 门迫，吉门也打折

**步骤三：横看宫间生克**
用神落宫与相关宫（目标事物代表宫）之间的五行生克关系，决定事态走向。

**步骤四：月令旺衰最终裁决**
"以月令为主，看用神与有关事物的旺相休囚"——这是成败的终极裁判。

### 2.3 冲突场景示例

**星吉但门凶**：门的权重高于星，偏凶判断。但若该凶门落宫受克（"凶门被克凶不起"），凶性减弱。

**格局吉但门迫**：吉格打折。"吉门迫宫吉不成"，原本的吉格因门迫而无法发挥。

**三奇得使但临凶门**：有条件吉。"三奇得使能够把凶格转化成有小的帮助的吉格，当然不是大吉，但至少没凶。"临吉门则"应吉大"，临凶门则"应吉小"。

## 三、四害的权重与能量衰减系统

### 3.1 四害能量衰减表

| 四害 | 能量剩余 | 能量衰减 | 含义 |
|------|----------|----------|------|
| **击刑** | 50% | -50% | 事态别扭、拧劲、半途而废 |
| **门迫** | 50% | -50% | 窝里反，对环境不满，外部压力 |
| **入墓** | 20% | -80% | 被藏起来、被约束捆绑，力量大幅减弱 |
| **空亡** | 20% | -80% | 信息转移，80%信息不在原处 |

### 3.2 四害叠加计算规则

**核心规则：乘积法**

> "若一个宫既出现迫、刑又出现墓、空，则宫中最后剩余的能量为，各自剩余能量的'乘积'。"

**计算示例**：
- 击刑 + 门迫 = 50% x 50% = **25%** 能量
- 击刑 + 入墓 = 50% x 20% = **10%** 能量
- 门迫 + 空亡 = 50% x 20% = **10%** 能量
- 击刑 + 门迫 + 入墓 = 50% x 50% x 20% = **5%** 能量
- 四害全中 = 50% x 50% x 20% x 20% = **1%** 能量（近乎无效）

### 3.3 四害的影响范围

> "若宫出现门迫、击刑、入墓、空亡，则宫中的**所有符号**（奇仪、神、星、门）反映信息的能量都减少，而宫外干支、月将、建星、神将等的所暗示的信息能量不减少。"

例外：隐干入墓或击刑及月将空亡，仅影响自身，不影响宫内其他符号。

### 3.4 四害是否构成一票否决？

四害**不属于**一票否决级别，但严重削弱宫内能量：
- 击刑/门迫：降为50%，严重但不致命
- 入墓/空亡：降为20%，接近无效但仍有残余信息
- 多害叠加：通过乘积法可降至极低值，实质上接近否决

特殊情况："如果受到门迫击刑就不一样了，那就是受到国家政府机关的或者黑社会的制裁、拘役、羁押、限制，自己没有反抗的能力。" -- 入墓 + 门迫/击刑组合在实践中视为极凶。

### 3.5 空亡的特殊处理

空亡不完全是消亡，而是**信息转移**：
- 先看先天宫：如艮8宫空亡，翻到先天宫震3宫追查
- 先天宫也空：再翻到下一层（离9宫）追查
- 旺空 vs 衰空："旺空"仍有作为机会（"在生空、值空的日子里努力才行"），"衰空"则真空

## 四、系统化吉凶评分框架

### 4.1 传统体系中无标准化评分

经多源调研确认：传统奇门遁甲体系中**不存在**统一的量化评分标准（如大吉+2、吉+1、凶-1、大凶-2这种）。传统方法是定性判断 + 经验累积。

### 4.2 可提炼的量化框架（基于调研综合）

根据调研所得规则，可为引擎设计如下评分体系：

#### A. 格局层（十干克应）吉凶分级

| 等级 | 代表格局 | 建议分值 |
|------|----------|----------|
| 大吉 | 天遁、地遁、人遁、龙返首、鸟跌穴 | +3 |
| 吉 | 三奇得使（临吉门）、三奇贵人升殿 | +2 |
| 小吉 | 三奇得使（临凶门）、奇仪相生组合 | +1 |
| 平 | 无特殊格局、比和 | 0 |
| 小凶 | 悖格（得吉门可转）、三奇受刑 | -1 |
| 凶 | 庚格系列（天乙伏宫等）、飞宫格 | -2 |
| 大凶 | 大格、刑格、年格月格日格时格 | -3 |
| 否决 | 五不遇时、六仪击刑、三奇入墓 | -99（一票否决） |

#### B. 门层吉凶

| 门 | 基础分 | 说明 |
|-----|--------|------|
| 开门 | +2 | 三吉门 |
| 休门 | +2 | 三吉门 |
| 生门 | +2 | 三吉门 |
| 景门 | 0 | 中平门 |
| 杜门 | 0 | 中平门 |
| 伤门 | -1 | 三凶门（轻） |
| 惊门 | -1 | 三凶门（轻） |
| 死门 | -2 | 三凶门（重） |

门宫关系修正：
| 关系 | 修正 |
|------|------|
| 门生宫（和） | 基础分 x 1.0 |
| 宫生门（相生） | 基础分 + 1 |
| 门宫比和 | 基础分 x 1.0 |
| 宫克门（门被克） | 吉门打折（x 0.5），凶门减弱（x 0.5） |
| 门克宫（门迫） | 吉门打折（x 0.5），凶门加重（x 1.5） |

#### C. 星层吉凶

| 星 | 基础分 | 说明 |
|----|--------|------|
| 天心 | +2 | 大吉星 |
| 天任 | +1 | 吉星 |

| 天芮 | -2 | 大凶星 |

#### D. 神层吉凶

| 神 | 基础分 | 说明 |
|-----|--------|------|
| 值符 | +2 | 大吉神 |
| 九天 | +1 | 吉神 |
| 太阴 | +1 | 吉神 |
| 六合 | +1 | 吉神 |
| 九地 | 0 | 中平 |
| 玄武 | -1 | 凶神 |

#### E. 四害修正（乘法修正系数）

| 四害 | 修正系数 |
|------|----------|
| 击刑 | x 0.5 |
| 门迫 | x 0.5 |
| 入墓 | x 0.2 |
| 空亡 | x 0.2 |

多害叠加时修正系数相乘。

#### F. 综合评分公式（建议）

```
宫位原始分 = 格局分 + 门分 + 星分 + 神分
四害系数 = 各四害修正系数的乘积（无四害则为1.0）
旺衰系数 = 根据月令旺相休囚死（旺=1.2, 相=1.0, 休=0.8, 囚=0.6, 死=0.4）

最终分 = 宫位原始分 x 四害系数 x 旺衰系数

特殊：若触发一票否决格局，最终分直接为 -99
```

## 五、关键口诀汇总

**九步观察诀**：
> "一看时干二看刑，三看入幕四门迫；时地生克观旺相，阴阳开合分主客"

**门宫关系诀**：
> "吉门被克吉不就，凶门被克凶不起；吉门相生有大利，凶门得生祸难避。吉门克宫吉不就，凶门克宫事更凶。"

**六仪击刑口诀**：
> "戊三己二庚刑八，辛九壬四癸半八"

**四害叠加原则**：
> "迫、刑、墓、空好比数学中计算法则...用以上规定来量化宫内的能量数值"
> "若一个宫既出现迫、刑又出现墓、空，则宫中最后剩余的能量为各自剩余能量的乘积"
