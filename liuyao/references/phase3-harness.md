# Phase 3：审旺衰 — 执行指令

> 前置：Phase 0-2 卡片已完成
> 输入：卦象数据卡 + 用神体系卡
> 输出：旺衰评估卡（下方模板）
> 原则：**月建定旺衰，日辰定生克。** 月日是断卦的核心参照系。
> 推理强度：🔴重
>
> **来源共识**：「月建为万卦之提纲，日辰为六爻之主宰」——《增删卜易》
> 所有 12+ 来源一致认为月日审查是用神之后的第一步

## 3.1 月建审查 — 万卦之提纲

### Step 1：确定月建

从 Phase 0 数据卡读取月建地支和五行。

### Step 2：查旺相休囚死

以月建为准，对用神、原神、忌神、世爻逐一查表：

**旺相休囚死速查表**（必须查表）：

| 月令 | 旺 | 相 | 休 | 囚 | 死 |
|------|----|----|----|----|-----|
| 寅卯月（春） | 木 | 火 | 水 | 金 | 土 |
| 巳午月（夏） | 火 | 土 | 木 | 水 | 金 |
| 申酉月（秋） | 金 | 水 | 土 | 火 | 木 |
| 亥子月（冬） | 水 | 木 | 金 | 土 | 火 |
| 辰戌丑未月（季） | 土 | 金 | 火 | 木 | 水 |

**旺衰等级定义**：
- **旺**：月令当令，力量最强
- **相**：月令所生，力量次强
- **休**：生月令者，力量平
- **囚**：克月令者，力量弱
- **死**：被月令克，力量最弱

### Step 3：月建五种作用

对每个关键爻，判断月建与其关系：

| 月建作用 | 条件 | 效果 |
|---------|------|------|
| 月生 | 月建五行生爻五行 | 爻得令旺 |
| 月扶 | 月建五行与爻五行相同 | 爻得令旺（比和） |
| 月克 | 月建五行克爻五行 | 爻受制衰 |
| 月合 | 月建地支与爻地支六合 | 有用则增力/被绊住 |
| 月冲 | 月建地支与爻地支六冲 | **月破**（见3.4） |

> **来源**：《增删卜易》月建章/劝学网、《黄金策》"月建提纲"、《卜筮正宗》月建论、易阳子、易师汇、墨海六爻

## 3.2 日辰审查 — 六爻之主宰

### Step 1：确定日辰

从 Phase 0 数据卡读取日辰地支和五行。

### Step 2：日辰对每个关键爻的作用

| 日辰作用 | 条件 | 效果 |
|---------|------|------|
| 日生 | 日辰五行生爻五行 | 爻增力 |
| 日扶 | 日辰五行与爻五行相同 | 爻增力（比和） |
| 日克 | 日辰五行克爻五行 | 爻减力 |
| 日合 | 日辰地支与爻地支六合 | 绊住/增力 |
| 日冲 | 日辰地支与爻地支六冲 | **暗动或日破**（见下） |

### Step 3：日冲的两面性（最关键区分）

| 爻的月令状态 | 日冲效果 | 结果 |
|------------|---------|------|
| 旺相有气 | **暗动**（冲起） | 静爻升级为动爻，获得生克权 |
| 休囚无气 | **日破**（冲散） | 爻彻底破碎无用 |
| 旺相且旬空 | **冲实** | 空亡之爻被激活，恢复力量 |

**必须先看月令旺衰，再判日冲效果。顺序不可颠倒。**

### Step 4：日月协作判断

| 日月组合 | 爻的综合状态 |
|---------|------------|
| 日月双方都生扶 | **极旺**（力量最强） |
| 日生月克 或 月生日克 | **平相**（力量中等） |
| 日月双方都克 | **极弱**（无生克权） |

### 日月核心区别（必须牢记）

| 区别项 | 月建 | 日辰 |
|--------|------|------|
| 休囚之说 | ✅ 有（旺相休囚死） | ❌ 无 |
| 冲动（暗动） | ❌ 无 | ✅ 有 |
| 保护力 | 月合 > 日冲（月合之爻日冲不散） | — |

> **来源**：《增删卜易》日辰章/月建章/劝学网、《黄金策》、六爻详真 ab.newdu.com、刘虹言六爻、墨海六爻 ziweicn.com、易师汇、易阳子

## 3.4 空亡检查

### Step 1：确定旬空

从 Phase 0 数据卡读取旬空两个地支。

### Step 2：扫描全卦

检查六爻每个地支是否命中旬空。命中则标注。

### Step 3：判断真空 vs 假空

| 类型 | 条件 | 结论 | 应期含义 |
|------|------|------|----------|
| **真空** | 安静 + 休囚 + 无生助 + 受日月克 | 永无用处 | 无应期 |
| **假空（旺不为空）** | 月令旺相 或 得日月生扶 | 出空后有用 | 出空填实之日 |
| **假空（动不为空）** | 爻发动 | 处于待发状态 | 出空之日应验 |
| **假空（合不为空）** | 得月合/日合/动爻合 | 有空名无空实 | 仍有效力 |

### Step 4：特殊规则

- 用神空亡 → 事未成/事延后
- 忌神空亡 → 忌神暂失克力（吉信号）
- 原神空亡 → 原神暂无法生用神
- 世爻空亡 → 问事者心意不诚/犹豫不决

> **来源**：《增删卜易》旬空章/劝学网、《卜筮正宗》旬空论、易阳子、易师汇、易数圈 m.30725.com

## 3.5 月破检查

### Step 1：确定月破

月建地支六冲之支 = 月破。

| 月建 | 月破 |
|------|------|
| 寅月 | 申破 |
| 卯月 | 酉破 |
| 辰月 | 戌破 |
| 巳月 | 亥破 |
| 午月 | 子破 |
| 未月 | 丑破 |
| 申月 | 寅破 |
| 酉月 | 卯破 |
| 戌月 | 辰破 |
| 亥月 | 巳破 |
| 子月 | 午破 |
| 丑月 | 未破 |

### Step 2：扫描全卦

检查六爻每个地支是否命中月破。

### Step 3：判断真破 vs 假破

| 类型 | 条件 | 结论 |
|------|------|------|
| **真破** | 月冲 + 日辰无气 + 受克 | 永无生克权 |
| **假破** | 月冲 + 日辰有气 或 得生扶 | 出月可恢复 |

### Step 4：解救方式

- 出月不破（过了这个月就好）
- 逢生合不破（有动爻/日辰来合或生）
- 填实不破（遇到同一地支之日）

> **来源**：《增删卜易》月破章/劝学网、《黄金策》、易阳子、易师汇、易数圈、墨海六爻

## 3.6 入墓检查

检查用神/原神/世爻是否入墓：

**四墓库对应表**：

| 五行 | 墓库 |
|------|------|
| 水 | 辰（水库） |
| 火 | 戌（火库） |
| 金 | 丑（金库） |
| 木 | 未（木库） |

**三种入墓**：
1. 入日墓 — 日辰为爻之墓库（最强制）
2. 入动爻墓 — 动爻为爻之墓库
3. 入变爻墓 — 自身变出之爻为墓库（化墓）

**墓 vs 库**：
- 休囚入墓 = 凶（被埋葬）
- 旺相入库 = 中性（被储藏，非真凶）

> **来源**：《增删卜易》墓库章/随鬼入墓章、易阳子入墓详解、易师汇墓库专篇、国易堂、DEREK.WIKI
