## 6.1 核心十字诀

> **动待合，静待冲；衰待旺，旺待墓；**
> **冲待合，合待冲；空待出，墓待破；**
> **伏待引，吟待冲。**

这是应期推断的总纲。每一句对应一种具体情况。

> **来源**：《增删卜易》应期总论、《卜筮正宗》应期论、易阳子应期汇总、CSDN应期总结、好名字网

## 6.2 确定时间单位（近应 vs 远应）

### Step 1：判断事件时间跨度

| 事件特征 | 应期单位 | 判断信号 |
|---------|---------|---------|
| 当下即时之事（来人/等人） | 时辰 | 用神在内卦/动爻多 |
| 近期之事（数日内） | 日 | 用神旺动/忌神衰 |
| 中期之事（月内） | 日或月 | 常规情况 |
| 远期之事（跨年/大事） | 月或年 | 用神衰静/事关重大 |

### Step 2：辅助判断信号

| 信号 | 偏近应 | 偏远应 |
|------|--------|--------|
| 用神动/静 | 动（应速） | 静（应迟） |
| 用神旺/衰 | 旺（快应） | 衰（需等旺时） |
| 来人状态 | 站着/刚来 | 坐着/已久 |
| 用神内/外卦 | 内卦（近） | 外卦（远） |

> **来源**：《增删卜易》应期总论、好名字网应期时限性专文、易阳子

## 6.3 病药法推断（核心流程）

### Step 1：确定用神

从 Phase 2 用神体系卡读取。

### Step 2：查用神"病处"

从 Phase 3（旺衰评估卡）和 Phase 4（生克关系卡）汇总用神的问题：

| 病处 | Phase 来源 | 对应药方 |
|------|-----------|---------|
| 空亡 | Phase 3 | 出空/填实/冲实（仅假空适用，真空无应期——参见 Phase 3 判断） |
| 月破 | Phase 3 | 出月/填实/逢合（仅假破适用，真破无应期——参见 Phase 3 判断） |
| 入墓 | Phase 3 | 冲墓 |
| 被合住 | Phase 4 | 冲开合神 |
| 被冲散 | Phase 4 | 逢合 |
| 休囚无力 | Phase 3 | 逢生旺之期 |
| 被忌神克 | Phase 4 | 忌神衰/原神动 |
| 伏藏 | Phase 2 | 冲飞露伏/伏神当值 |
| 化退 | Phase 4 | 变爻旺相之期 |
| 反吟 | Phase 4 | 逢合/冲回之日 |
| 伏吟 | Phase 4 | 逢冲之日 |

### Step 3：找药方

"药方"就是解除"病处"所需的干支条件。

### Step 4："药到"之时 = 应期

将药方转化为具体干支日/月/年。

## 6.5 吉凶方向与应期的关系

| Phase 5 判断 | 应期逻辑 |
|-------------|---------|
| 吉事 | 找"吉应"——用神何时旺相/得生/出空/解合 |
| 凶事 | 找"凶应"——用神何时入墓/逢克/逢冲/空实 |
| 先凶后吉 | 两个应期：凶时+转吉时 |
| 先好后散 | 两个应期：好时+散时 |

---

## 6.6 应期防错清单

| 常见错误 | 正确做法 |
|---------|---------|
| 不管事件跨度一律用"日" | 先判近应/远应，选正确时间单位 |
| 独发就定应期 | 独发是参考，用神状态才是核心 |
| 月破就判"无望" | 区分真破/假破，假破出月可恢复 |
| 入墓就判"死" | 旺相入墓非真凶，休囚入墓才危 |
| 忽略原神 | 原神动时看原神的病处来定应期 |
| 不结合现实 | 必须结合现实时间约束（不可能3天后考上研究生） |
