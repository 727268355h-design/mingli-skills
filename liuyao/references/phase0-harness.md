# Phase 0：读盘提取 — 执行指令

> 输入：用户提供的排盘截图 + 问事内容
> 输出：结构化卦象数据卡（下方模板）
> 原则：此步骤不做任何判断，纯数据提取和标注
> 推理强度：🟢轻

---

## 0.1 识别排盘截图

从截图中识别并提取以下全部信息。逐项提取，不跳过。

### A. 日期信息

```
公历：YYYY年M月DD日
农历：...
日期干支：
  年柱：[干支]
  月柱：[干支]
  日柱：[干支]
  时柱：[干支]（若有）
旬空：[两个地支]（由日柱所在旬决定）
```

**旬空速查表**（必须查表，禁止推算）：

| 旬 | 空亡 |
|----|------|
| 甲子旬（甲子→癸酉） | 戌、亥 |
| 甲戌旬（甲戌→癸未） | 申、酉 |
| 甲申旬（甲申→癸巳） | 午、未 |
| 甲午旬（甲午→癸卯） | 辰、巳 |
| 甲辰旬（甲辰→癸丑） | 寅、卯 |
| 甲寅旬（甲寅→癸亥） | 子、丑 |

> **来源**：《增删卜易》、《卜筮正宗》旬空论、易师汇、易阳子、易数圈

### B. 卦象信息

```
主卦：[卦名]（[上卦]上 [下卦]下）
变卦：[卦名]（若有动爻）/ 纯静卦（无动爻）
所属宫：[X宫]
```

### C. 六爻逐爻提取

从**初爻到上爻**（底部到顶部），逐爻提取：

| 爻位 | 六神 | 六亲 | 地支 | 动/静 | 变爻六亲 | 变爻地支 | 世/应 |
|------|------|------|------|-------|---------|---------|-------|
| 上爻 | | | | | | | |
| 五爻 | | | | | | | |
| 四爻 | | | | | | | |
| 三爻 | | | | | | | |
| 二爻 | | | | | | | |
| 初爻 | | | | | | | |

**提取规则**：
- 六神：青龙/朱雀/勾陈/腾蛇/白虎/玄武（从初爻起按日干排列）
- 六亲：父母/官鬼/妻财/子孙/兄弟
- 地支：子丑寅卯辰巳午未申酉戌亥
- 动/静：动爻标 ○ 或 ×，静爻标 —
- 变爻：仅动爻有变爻，标注变爻的六亲和地支
- 世/应：标注哪一爻是世爻（世），哪一爻是应爻（应）

### D. 伏神信息

如果截图中标注了伏神，提取：

| 伏于爻位 | 伏神六亲 | 伏神地支 | 飞神六亲 | 飞神地支 |
|---------|---------|---------|---------|---------|
| | | | | |

如果截图未标注伏神，在此步暂不处理，Phase 2 取用神时按需查找。

## 0.3 基础校验（机械检查，不做判断）

逐项执行以下校验：

| 检查项 | 方法 | 异常处理 |
|--------|------|----------|
| 地支合法性 | 所有地支必须在子丑寅卯辰巳午未申酉戌亥之内 | 标注 ⚠️ |
| 六亲完整性 | 检查主卦六爻中五种六亲是否齐全 | 缺失的六亲标注（Phase 2 查伏神） |
| 世应位置 | 世爻和应爻必须相隔三位 | 不符标注 ⚠️ |
| 动变一致性 | 动爻必须有对应变爻 | 缺失标注 ⚠️ |
| 纳甲一致性 | 地支应符合所属宫的纳甲规则 | 不符标注 ⚠️（可能为识别错误） |

---

## 0.4 标注五行

为每个爻的地支标注五行属性（纯查表）：

| 地支 | 五行 |
|------|------|
| 寅、卯 | 木 |
| 巳、午 | 火 |
| 辰、戌、丑、未 | 土 |
| 申、酉 | 金 |
| 亥、子 | 水 |

同时标注月建五行和日辰五行。

> **来源**：通用五行基础，《增删卜易》、《卜筮正宗》、易师汇、易阳子、国易堂均一致
