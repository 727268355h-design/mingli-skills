| 怀孕/生育 | 子孙 | 妻财 | 官鬼 | 兄弟 |

### 特殊取用规则

**代占**（替别人问卦）：
- 代占父母 → 父母为用神
- 代占子女 → 子孙为用神
- 代占配偶 → 妻财/官鬼为用神
- 代占兄弟朋友 → 兄弟为用神
- **世爻代表问卦人，应爻代表被问之人**

**同一对象不同目的取不同用神**：
- 问父亲健康 → 父母为用神（看其旺衰）
- 问父亲财运 → 以父母为太极点，妻财为用神
- 问丈夫事业 → 以官鬼为太极点，官鬼本身看旺衰

## 2.3 标注四神关系链

用神确定后，完成四神关系链标注：

### 四神关系图

```
仇神 ──生──→ 忌神 ──克──→ 用神 ←──生── 原神
  ↑                                    |
  └────────────克────────────────────────┘
```

### 五行生克推导

根据用神的五行，推导其他三神：

| 用神五行 | 原神（生用神） | 忌神（克用神） | 仇神（生忌神） |
|---------|-------------|-------------|-------------|
| 金 | 土 | 火 | 木 |
| 木 | 水 | 金 | 土 |
| 水 | 金 | 土 | 火 |
| 火 | 木 | 水 | 金 |
| 土 | 火 | 木 | 水 |

逐一在卦中定位原神、忌神、仇神的爻位，标注动/静状态。

**注意**：某神可能有多个爻（如两个木五行的爻都是原神），全部标出。

## 2.4 用神两现处理

当同一六亲在卦中出现两次（如两个妻财爻），按以下**4 级优先级**取舍：

| 优先级 | 规则 | 取谁 |
|--------|------|------|
| 1 | 一动一静 | **取动爻** |
| 2 | 都动/都静，日月生扶 | **取得日月生扶者** |
| 3 | 都动/都静，旺衰同 | **取有特殊状态者**（空/破/暗动） |
| 4 | 以上都同 | **取离世爻最近者** |

标注：
```
用神两现：
  A = [六亲][地支]（[X爻]）[动/静]
  B = [六亲][地支]（[X爻]）[动/静]
  取用：[A/B]
  取舍理由：[引用上表规则]
```

> **来源**：《增删卜易》用神两现章、易阳子、易师汇、国易堂、新浪新闻

### Step 2：找伏神

在八纯卦中找到缺失六亲所在的爻位：
- 该爻 = **伏神**
- 主卦在同一爻位上的爻 = **飞神**

### Step 3：判断飞伏关系

| 飞伏关系 | 吉凶 | 含义 |
|---------|------|------|
| 飞生伏 | 吉 | 伏得长生，飞神帮助伏神 |
| 伏克飞 | 吉 | 伏神有力可出 |
| 飞伏比和 | 吉 | 和谐相助 |
| 伏生飞 | 不吉 | 伏神泄气给飞神 |
| 飞克伏 | 最凶 | 伏神被压制难出 |

### Step 4：判断伏神有用/无用

**有用条件**（满足任一即可）：
1. 得日月生扶
2. 旺相有气
3. 得飞神生（飞生伏）
4. 得动爻生扶
5. 日月冲克飞神（解放伏神）
6. 飞神休囚空破（压制力弱）

**无用条件**（满足任一即凶）：
1. 休囚无气
2. 被日月克制
3. 飞神旺相且克伏（压制力强）
4. 伏神墓绝于日辰

### Step 5：标注出伏条件

伏神何时可出：
- 冲飞露伏之日（最常见）
- 伏神当值之日
- 伏神得生旺之日
