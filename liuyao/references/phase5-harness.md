## 5.1 吉凶综合判断公式

从 Phase 3（旺衰）和 Phase 4（生克）汇总，套入以下判断矩阵：

### 判断矩阵

| 用神状态 | 原神 | 忌神 | 判断 |
|---------|------|------|------|
| 旺相 | 能生 | 不能克 | **大吉** |
| 旺相 | 能生 | 能克 | **吉**（有波折） |
| 旺相 | 不能生 | 不能克 | **吉** |
| 旺相 | 不能生 | 能克 | **平/小吉**（旺可抗） |
| 休囚 | 能生 | 不能克 | **小吉**（原神可救） |
| 休囚 | 能生 | 能克 | **平**（吉凶交战） |
| 休囚 | 不能生 | 不能克 | **小凶**（自身无力） |
| 休囚 | 不能生 | 能克 | **凶** |
| 真空/真破/入墓无解 | — | — | **大凶** |

### 修正因子

在基础判断上叠加修正：

| 因子 | 向吉修正 | 向凶修正 |
|------|---------|---------|
| 世爻 | 世爻旺相有力 | 世爻休囚空破 |
| 特殊卦象 | 六合卦、归魂卦 | 六冲卦、游魂卦 |
| 卦变 | 六冲变六合 | 六合变六冲 |
| 持世 | 子孙持世（利避灾） | 兄弟持世（不利财） |
| 六神 | 青龙临用神 | 白虎/腾蛇临用神 |

### 绝处逢生 / 克处逢生

- **绝处逢生**：用神将绝，得日月或动爻一处生扶 → 转吉
- **克处逢生**：用神受克，有原神解救 → 先凶后吉
- **合处逢冲**：卦本相合，后遇冲破 → 先好后散
- **冲中逢合**：卦本相冲，后遇合 → 先散后合

> **来源**：《增删卜易》断卦总法、《黄金策》总断千金赋、知乎 p/627027980, p/470660017, p/521636345；180 小驿站（5 步结构法）；易师汇（12 步法）

## 5.2 分类专项断法

根据 Phase 1 的问事类型，执行对应的专项分析。**每个专项必须引用前序 Phase 的结论。**

### 财运专项

| 维度 | 看点 |
|------|------|
| 核心看 | 妻财爻旺衰（Phase 3）+ 子孙爻状态 |
| 吉象 | 财持世/生合世、子孙财同旺、财化进神 |
| 凶象 | 兄弟动克财、世空破、财化退、兄临玄武=被骗 |
| 求财方式 | 子孙持世=经营生产、妻财持世=正当投资、官鬼持世=靠地位权力 |

### 婚姻专项

| 维度 | 男测 | 女测 |
|------|------|------|
| 用神 | 妻财 | 官鬼 |
| 成婚象 | 世应相合+用神旺 | 世应相合+用神旺 |
| 不成象 | 六冲卦/财官空破墓 | 六冲卦/财官空破墓 |
| 第三者 | 财/官多现+伏藏 | 财/官多现+伏藏 |
| 对方条件 | 应爻旺衰+六神 | 应爻旺衰+六神 |

### 事业求职专项

| 维度 | 看点 |
|------|------|
| 核心看 | 官鬼（职位）+ 父母（单位）+ 妻财（薪资）三者旺衰 |
| 成事象 | 三者皆旺+世爻有力 |
| 不成象 | 子孙发动克官（不利功名）、官空破墓 |
| 面试/笔试 | 加看父母爻（文书） |

### 疾病专项

| 维度 | 看点 |
|------|------|
| 世爻 | 代表病人本身（旺=体质好，衰=体质差） |
| 官鬼 | 代表病症（旺=病重，衰=病轻） |
| 子孙 | 代表医药（旺=有药可治，空破=药无效） |
| 近病 | 逢冲则愈，逢合则凶 |
| 远病 | 逢冲则死，逢合则安 |
| 爻位 | 初=足，二=腿，三=腰腹，四=胸背，五=耳鼻喉，六=头 |

### 官司专项

| 维度 | 看点 |
|------|------|
| 核心看 | 世爻vs应爻旺衰力量对比 |
| 世旺应衰 | 我方胜 |
| 应旺世衰 | 对方胜 |
| 间爻 | 证人（空=无证人，动克谁不利谁） |
| 六冲卦 | 官司了结快 |
| 六合卦 | 官司拖延 |

### 考试专项

| 维度 | 看点 |
|------|------|
| 普通考试 | 父母为用神 |
| 高等考试 | 父母+官鬼双用神（文凭+功名） |
| 父母旺动 | 利考试 |
| 妻财动克父母 | 不利（财星坏印） |

### 出行专项

| 维度 | 看点 |
|------|------|
| 用神 | 世爻 |
| 子孙持世 | 最吉（一路平安） |
| 官鬼动 | 凶险（灾祸信号） |
| 用神逢合 | 有事绊住走不了 |
| 应爻旺生世 | 目的地利好 |

> **来源**：《增删卜易》各分类章、《黄金策》27 类分占、易阳子分类断法（财运/婚姻/事业/疾病/官司/考试/出行）、易师汇、加拿大风水命理研究中心
