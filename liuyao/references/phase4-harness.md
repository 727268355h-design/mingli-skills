# Phase 4：析生克 — 执行指令

> 前置：Phase 0-3 卡片已完成
> 输入：卦象数据卡 + 用神体系卡 + 旺衰评估卡
> 输出：生克关系卡（下方模板）
> 原则：**内因决定论** — 卦体生克关系是决定性因素，月日只是引发因素。
> 推理强度：🟡中
>
> **优先级链**（12+ 来源一致）：
> 回头生克/化进退 ＞ 动爻生克 ＞ 日月作用
> 先生后克，先动后静，无生则克，无克则生

## 4.1 动爻逐一分析

对 Phase 0 标记的每个动爻，逐一完成以下分析：

### A. 变爻关系判断

| 类型 | 条件 | 效果 | 对该爻影响 |
|-----|------|------|-----------|
| 回头生 | 变爻五行生动爻五行 | 动爻增力 | 有生克权，力量增强 |
| 回头克 | 变爻五行克动爻五行 | 动爻减力 | **无生克权**，被束缚 |
| 回头合 | 变爻地支与动爻地支六合 | 动爻被绊 | 暂无力，被合住 |
| 回头冲 | 变爻地支与动爻地支六冲 | 事反复 | 反吟信号 |
| 伏吟（回自） | 变爻地支与动爻地支相同 | 停滞呻吟 | 伏吟信号 |

### B. 进神退神判断

**进神 8 组**（必须查表）：

| 动爻 | 变爻 | 方向 |
|------|------|------|
| 寅 | → 卯 | 木进 |
| 巳 | → 午 | 火进 |
| 申 | → 酉 | 金进 |
| 亥 | → 子 | 水进 |
| 丑 | → 辰 | 土进 |
| 辰 | → 未 | 土进 |
| 未 | → 戌 | 土进 |
| 戌 | → 丑 | 土进 |

**退神 8 组**（进神的反向）：
卯→寅、午→巳、酉→申、子→亥、辰→丑、未→辰、戌→未、丑→戌

**含义**：
- 用神/原神化进 → 吉（事物发展前进）
- 用神/原神化退 → 凶（事物倒退衰败）
- 忌神/仇神化进 → 凶（忌神力量增强）
- 忌神/仇神化退 → 吉（忌神力量减弱）

### C. 化空/化墓/化绝/化破判断

| 类型 | 条件 | 效果 |
|------|------|------|
| 化空 | 变爻地支命中旬空 | 旺不为空出空有用，休囚则无力 |
| 化墓 | 变爻地支为动爻之墓库 | 受困被锁（冲墓之日解） |
| 化绝 | 变爻地支为动爻之绝地 | 旺相短期微弱，休囚几乎无用 |
| 化破 | 变爻被月建冲破 | 变爻失去回头生克能力 |

**五行绝地对照**：金绝在寅、木绝在申、水绝在巳、火绝在亥、土绝在巳

### D. 禁止克规则（5 组）

| 动爻 | 变爻 | 规则 |
|------|------|------|
| 兄弟 | 化妻财 | 兄化财不劫（不克财） |
| 父母 | 化子孙 | 父化子不克 |
| 官鬼 | 化兄弟 | 官化兄不克 |
| 妻财 | 化父母 | 财化父不克 |
| 子孙 | 化官鬼 | 子化鬼不克 |

### E. 动爻对用神的作用

该动爻五行与用神五行的生克关系 → 标注是"生用神"还是"克用神"。

> **来源**：《增删卜易》进退章/动变章/劝学网、《黄金策》、易海东方 zhouyiss.com（回头生克）、易阳子（进退神/化空绝墓）、国易堂（进退神）、搜狐（化空化绝化破）、六爻详真（层次/生克权）

## 4.2 原神能否生用神

综合 Phase 3 旺衰 + Phase 4.1 动变分析，判断原神是否真正能生到用神：

### 原神能生（满足任一）

| 条件 | 说明 |
|------|------|
| 动而旺相 | 原神发动且月令旺相 |
| 临日月 | 原神得日建或月建直接生 |
| 动化回头生 | 原神发动且变爻回头生 |
| 不化绝/墓/空 | 变爻未入绝/墓/空 |
| 不贪合忘生 | 原神未被其他爻合住（合住则无法脱身生用神） |
| 不逢日月合住 | 原神未被日月合住（合住至冲开之日方能生） |
| 静卦旺相 | 原神虽静但旺相，仍可缓生 |

### 原神不能生（满足任一）

| 条件 | 说明 |
|------|------|
| 休囚无气 | 月令休囚且无日辰生扶 |
| 动入变墓 | 变爻为原神之墓 |
| 化退神 | 变爻为退神 |
| 化绝 | 变爻为原神之绝地 |
| 被日月克 | 日月同时克原神 |
| 旬空真空 | 安静+休囚+空亡 |
| 贪合忘生 | 原神被其他爻合住 |

> **来源**：《增删卜易》原神忌神篇、易阳子、易师汇、新浪新闻、DEREK.WIKI

## 4.3 忌神能否克用神

### 忌神能克（满足任一）

| 条件 | 说明 |
|------|------|
| 日月生扶忌神 | 忌神得日月生扶有力 |
| 日月比扶忌神 | 忌神与日月同五行（比和增力） |
| 化进神 | 忌神变爻为进神 |
| 得变爻生 | 变爻回头生忌神 |
| 与仇神同动 | 仇神动来生忌神 |
| 日月为忌神长生地 | 忌神得长生有气有力 |

### 忌神不能克（满足任一）

| 条件 | 说明 |
|------|------|
| 休囚静爻 | 忌神安静且休囚 |
| 被日月克 | 日月克制忌神 |
| 月破 | 忌神月破 |
| 空亡 | 忌神空亡 |
| 化退 | 忌神化退神 |
| 遇绝化绝 | 忌神遇绝地或变爻为绝地 |
| 入墓 | 忌神入墓 |
| 贪生忘克 | 原神忌神同动→忌神被牵制（见下） |

### 特殊规则：贪生忘克

当原神和忌神同时发动时：
- 原神动来生用神，忌神虽动但被原神牵制
- 忌神贪图与原神的相生关系，无暇去克用神
- 结论：**用神安全**（忌神被牵制，克力大减）

> **来源**：《增删卜易》忌神篇、DEREK.WIKI（贪生忘克）、易阳子、易师汇、国易堂

## 4.4 多爻齐动处理

当动爻数 ≥ 3 时，需分层处理：

**现代原则（优先）**：始终以**用神为核心**，按爻的层次分析各动爻对用神的生克。

**古法参考**（辅助）：

| 动爻数 | 古法取断 |
|--------|---------|
| 两爻动 | 一阴一阳取阴爻；同阴同阳取上动爻 |
| 三爻动 | 取中间一爻 |
| 四爻动 | 取下方静爻 |
| 五爻动 | 取独静爻 |
| 六爻皆动 | 乾坤用"用九/用六"，余卦以变卦彖辞 |

**层次优先级**：日月建 > 变爻 > 动爻 > 静爻

**合冲互解规则**：
- **合处逢冲** → 以冲为最终结果（先合后散）
- **冲中逢合** → 以合为最终结果（先散后合）

> 当同一爻同时存在合和冲时，后发作用决定最终状态。
