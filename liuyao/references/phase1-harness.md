## 1.1 问事类型分类

根据 Phase 0 提取的问事关键词，分类到以下标准类型：

| 类型 | 关键词 | 用神方向预告 |
|------|--------|-------------|
| 财运 | 求财、投资、生意、收入、买卖 | → Phase 2 取妻财 |
| 婚姻 | 感情、结婚、对象、恋爱、复合 | → Phase 2 男取妻财/女取官鬼 |
| 事业 | 工作、求职、升职、辞职、跳槽 | → Phase 2 取官鬼 |
| 疾病 | 生病、健康、身体、手术、康复 | → Phase 2 世爻+官鬼双看 |
| 官司 | 诉讼、纠纷、仲裁、打官司 | → Phase 2 世应对比 |
| 考试 | 考试、学业、面试、笔试、资格证 | → Phase 2 取父母 |
| 出行 | 出差、旅行、搬家、出国 | → Phase 2 取世爻 |
| 天气 | 下雨、晴天、天气 | → Phase 2 取父母(雨)/子孙(晴) |
| 失物 | 丢失、找东西、寻人 | → Phase 2 按物品取用 |
| 其他 | 不属于以上 | → Phase 2 按六亲类象取用 |

> **来源**：《增删卜易》分类章、《黄金策》27 类分占、易阳子分类断法、易师汇、国易堂

**六合卦**（8 个）：

| 卦名 | 内外卦 |
|------|--------|
| 天地否 | 乾上坤下 |
| 地天泰 | 坤上乾下 |
| 雷风恒 | 震上巽下 |

**标记**：
- 🔴 主卦六冲 — 事散/冲突/不稳
- 🔴 变卦六冲 — 结局散
- 🟢 主卦六合 — 事和/顺利
- 🟢 变卦六合 — 结局合
- 🔴🔴 六合变六冲 — 先好后散
- 🟢🟢 六冲变六合 — 先散后合

> **来源**：《增删卜易》六冲六合章、《黄金策》、易阳子、气功学、中国预测网

### B. 游魂卦 / 归魂卦

- **游魂卦**：八宫各第七卦（上卦回头内卦不变） → 🟡 变动不定、心神不安
- **归魂卦**：八宫各第八卦（下卦归本宫） → 🟡 稳定不变、难启新事

> **来源**：《增删卜易》、易师汇、易阳子、百度百科、国易堂

### C. 反吟 / 伏吟（卦级别）

- **卦变反吟**：主卦变卦互为六冲关系 → 🔴 事反复无常
- **卦变伏吟**：主卦变卦相同（全部化回本位） → 🟡 事停滞哭泣

> **来源**：《增删卜易》反吟伏吟章、易阳子、易师汇、国易堂、DEREK.WIKI

## 1.3 持世爻分析

标注世爻的六亲身份及含义（纯标记，不判断吉凶）：

| 持世六亲 | 标记含义 |
|---------|---------|
| 父母持世 | 主辛劳操心，利文书合同，不利子女/六畜 |
| 官鬼持世 | 事有压力，利功名升迁，有忧虑 |
| 妻财持世 | 利财利妻，不利父母长辈，不利升学 |
| 子孙持世 | 事无忧平安，利医药解灾，不利功名 |
| 兄弟持世 | 有竞争，不利求财，男占不利妻 |

> **来源**：《增删卜易》诸爻持世章、《黄金策》、易阳子、易师汇、国易堂
