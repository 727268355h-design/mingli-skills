## 输入格式说明

紫微斗数命盘数据来自「文墨天机」排盘工具，格式为树状结构文本，包含以下层级：

```
├基本信息
│ ├性别
│ ├地理经度
│ ├钟表时间 / 真太阳时
│ ├农历时间
│ ├节气四柱 / 非节气四柱
│ ├五行局数
│ └身主 / 命主 / 子年斗君 / 身宫
│
├命盘十二宫（每宫包含）
│ ├宫名[天干地支]
│ ├主星 : [星名][庙旺得利平不陷][生年四化][自化]
│ ├辅星 : [星名][庙旺得利平不陷][自化]
│ ├小星 : [星名][庙旺得利平不陷]
│ ├神煞
│ │ ├岁前星
│ │ ├将前星
│ │ ├十二长生
│ │ └太岁煞禄
│ ├大限 : N~M 虚岁
│ ├小限 : 列表
│ ├流年 : 列表
│ └限流叠宫 : 有/无
│
├大限流年信息（如有）
│
└备注
```

### 关键标记说明

| 标记 | 含义 |
|------|------|
| `[庙]` `[旺]` `[得]` `[利]` `[平]` `[不]` `[陷]` | 星曜在该宫的庙旺状态 |
| `[生年禄]` `[生年权]` `[生年科]` `[生年忌]` | 生年四化 |
| `[↑禄]` `[↑权]` `[↑科]` `[↑忌]` | 宫干飞化入（其他宫飞入本宫的四化） |
| `[↓禄]` `[↓权]` `[↓科]` `[↓忌]` | 宫干飞化出（本宫飞出去的四化） |
| `[来因]` | 来因宫标记 |
| `[身宫]` | 身宫所在位置 |

## 分析框架

### 第一部分：命盘总论

1. **基本格局判定**
   - 命宫主星格局定性
   - 身宫主星与命宫的关系
   - 命主、身主的意义
   - 五行局数对命运的影响
   - 来因宫的特殊意义

2. **生年四化分析**
   - 四化星落宫位置及其含义
   - 四化与各宫的交互关系
   - 生年四化构成的格局（如双禄、禄忌交战等）

3. **飞宫四化网络**
   - 命宫干飞四化路径
   - 各宫间飞化的连锁效应
   - 重要的化忌路径追踪

### 第二部分：十二宫逐宫分析

对每个宫位分析：
- 主星组合的基本含义
- 辅星、小星的修正作用
- 该宫的生年四化和飞化影响
- 庙旺陷落对星曜力量的影响
- 神煞的辅助判断
- 十二长生位的含义

### 第三部分：主题专项分析

按以下主题逐项深入：

1. **性格与天赋** — 命宫、福德宫、身宫三角
2. **健康** — 疾厄宫为主，参看命宫、福德宫
3. **学业** — 父母宫（学业）、官禄宫（考试）、命宫（才智）
4. **事业与财运** — 官禄宫、财帛宫、田宅宫、迁移宫联动
5. **人际关系** — 交友宫、兄弟宫、父母宫
6. **婚姻感情** — 夫妻宫、子女宫、福德宫、桃花星分布
7. **家庭与置产** — 田宅宫、子女宫、兄弟宫
