## Step 1：大运地图（一生总览）

**目的**：一张图看完一生的大运走向，识别高峰和低谷。
**方法**：基础排列 + 星情定性 + 生年四化/大限四化快速叠合。
**产出**：大运总览表 + 运势曲线标注。

### 1.1 大限排列

```
起运年龄 = 五行局数
行运方向：阳男/阴女顺行，阴男/阳女逆行
第1限命宫 = 本命命宫，之后按方向逐宫移动
每限10年（起运年龄 ~ 起运年龄+9）

⚠️ 幼年命宫期：0岁至起运年龄前，命主尚未进入第一步大限。
此阶段以原局命宫论断，不套大限四化。
例：木三局3岁起运 → 0~2岁为幼年命宫期，纯看原局。
```

### 1.2 每步大限快速定性（6 字段，不展开）

```

```

**评级速判规则**（三合派星情法）：
- 大限命宫主星庙旺 + 三方四正吉多煞少 → 吉
- 大限命宫主星陷 + 三方四正煞多 → 凶
- 大限忌冲命宫/冲财帛/冲官禄 → 降一级
- 大限走到生年忌宫 → 降一级
- 大限走到来因宫/身宫 → 升一级

## Step 2：当前大限深入

**目的**：深入分析命主正在走的这步大限，这是流年分析的"天花板"。
**方法**：三级相扣的"中盘定型"逻辑 + 飞宫四化 + 循环忌检查。
**核心原则**（令东来）：大运有的，流年才会有。大运没有的，流年有了程度也不大。

### 2.1 大限四化详解

```
大限宫干[X]：
化禄[星]→[宫] — 此限得利方向
化权[星]→[宫] — 此限掌控方向
化科[星]→[宫] — 此限名声/贵人方向
化忌[星]→[宫]，冲[对宫] — 此限最大困扰
```

### 2.2 大限四化与原局叠合（三级相扣·中盘定型）

逐颗检查大限四化与原局的关系：

```
大限禄[星]→[宫]：
  原局此宫有什么？[主星+飞化入情况]
  叠合效应：[禄叠禄/禄解忌/新增禄/平]

大限忌[星]→[宫]（冲[对宫]）：
  原局此宫有什么？[主星+飞化入情况]
  原局此宫已有忌？→ 忌形态判定[八种之一]
  忌冲对宫：原局对宫状态？→ 冲击程度
  一六共宗：冲宫的交友宫=[宫] → 牵连影响
```

### 2.2a 飞宫忌形态细分（八种）

双忌/三忌不可笼统论"大凶"，须辨别形态：

| # | 形态 | 定义 | 凶险 | 典型应验 |
|---|------|------|------|----------|
| 1 | **重叠忌** | 本命+大限化忌汇入同宫（同星或异星） | 极凶 | 该宫事项十年内必有重大损失 |
| 2 | **冲命宫** | 任何盘的化忌冲命宫（含对宫忌入迁移） | 极凶 | 本人直接受冲 |
| 3 | **互冲忌** | 本命忌入A宫，大限忌在A宫对宫，形成对宫线互冲 | 高 | 两宫事项同时受损 |
| 4 | **反弹忌** | A盘忌冲B盘某宫，B盘忌冲A盘某宫 | 高 | 两领域互相拖累 |
| 5 | **同类相冲** | 三盘同类宫化忌互冲（如三盘财帛互冲） | 高 | 该类事项反复受打击 |
| 6 | **三合忌** | 本命+大限化忌呈三合角度（120°/240°） | 中高 | 凶性较弱但影响面更广 |
| 7 | **忌入同类** | 忌星入同类宫（如大限财帛忌入本命财帛） | 中 | 同类宫互忌不凶（见下方规则） |
| 8 | **单忌** | 不呈现上述任何组合 | 低 | 庙旺可压制 |

⚠️ 三忌同宫（生年+大限+流年）→ 超凶。"三忌反不忌"规则实战验证不足，暂不纳入。

### 2.2b 庙旺对四化效力调节

| 组合 | 效果 |
|------|------|
| 化禄/化权 + 庙旺 | 力量完整发挥 |
| 化禄/化权 + 陷落 | 力量打折，约六七成 |
| 化忌 + 庙旺 | 忌的破坏力被压制，约六七成 |
| 化忌 + 陷落 | 忌的破坏力完全释放 |
| 化科 + 任何状态 | 受庙旺影响较小 |

在 2.2 叠合分析中，标注四化星庙旺后再论效力。

### 2.2c 同类宫互忌/互冲规则

- **同类宫互忌不凶**：大限财帛化忌入本命财帛宫 → 不作凶论（忌入同类=自家事，内耗而非破坏）
- **同类宫互冲才凶**：大限财帛化忌冲本命财帛宫 → 必有破财（冲=外力介入，破坏性释放）
- 判断时先确认两宫是否为同类宫（同名宫位），再看忌星是入还是冲。

### 2.3 循环忌在此限是否激活

```
从 Phase 2 循环忌记录中检查：
循环忌1：[A宫]↔[B宫] — 大限命宫是否踩在A或B上？→ [是(激活)/否]
循环忌2：[C宫]↔[D宫] — 同上
```

### 2.4 大限定型结论

```
此限主题：[一句话]
此限天花板：吉事上限=[方向]，凶事上限=[方向]
此限出路：[基于大限禄/权/科的正面方向]
此限红线：[基于大限忌+循环忌的危险方向]
```

## Step 3：当前大限逐年扫描

**目的**：快速扫描当前大限内每一年，识别出需要深入分析的关键年。
**方法**：速断七步法（简化版）+ 关键年筛选标准。
**注意**：此步不做深度分析，只做快速筛选。深度留给 Step 4。

### 3.1 逐年快速扫描表

对当前大限内的每一年，填写以下 6 列：

```
| 年份 | 干支 | 虚岁 | 流年命宫=[原局宫] | 流年忌[星]→[宫](冲[宫]) | 快判 |
```

**快判规则**（速断法简化）：
- 流年命宫叠原局强宫 → +1
- 流年命宫叠原局弱宫 → -1
- 流年禄入命宫/财帛/官禄 → +1
- 流年忌冲命宫/财帛/官禄 → -1
- 流年天干=生年天干 → 标记"四化全引动"
- 流年命宫=生年忌宫 → 标记"走到忌宫"
- 流年忌与生年/大限忌同宫 → 标记"重叠忌"
- **大限天花板约束**：快判得分不得超过 Step 2 定型结论的天花板。吉事上限受大限出路限制，凶事下限受大限红线限制。例：大限定型为"财运受限，事业有出路"，则流年财运吉信号降权，事业吉信号保留。

### 3.2 关键年筛选

从扫描表中筛选满足以下**任一条件**的年份为"关键年"，进入 Step 4：

```
关键年条件（任一即选）：
1. 流年天干 = 生年天干（四化全引动）
2. 流年命宫 = 原局命宫 或 大限命宫（格局激活）
3. 流年命宫 = 生年忌所在宫（走到忌宫）
4. 流年忌与生年忌/大限忌同宫（重叠忌）
5. 流年天干 = 大限天干（大限四化重新引动）
6. 命主指定要看的年份
```

**至少选出 2 个关键年**。如果条件筛出超过 4 个，优先选当前年和最近的未来年。

### 4.2 三级相扣补充规则（在 Layer 2 上传下应中执行）

在五层漏斗的 Layer 2 中，额外执行以下检查（来自紫微精成三级相扣论断法）。

> **"走到"精确定义**：以下三种情况均算"走到"——
> 1. **大限命宫落在**某原局宫位（该宫事项在此十年被激活）
> 2. **流年命宫落在**某原局宫位（该宫事项在此年被激活）
> 3. **大限/流年的同类宫**与原局宫位重叠（如流年财帛宫 = 原局财帛宫所在地支）
>
> **注意**：十年大限内，真正被三级相扣触发的年份通常只有 1-3 个。**多数年份不触发是正常的**——不要为了"找到关键年"而降低触发标准。

**天人合一检查**：
```
原盘四化 → 在当前大限的什么宫位？→ 该宫成为流年的什么宫位？
= 原盘兆示 → 大限定型 → 流年应验 的完整链路

具体操作：
1. 取生年忌所在宫 → 在大限十二宫中是什么宫职？
2. 该宫职在流年中落在哪个原局宫上？
3. 若流年的该同类宫被冲 → 凶事在此年发生
4. 若流年的该同类宫被叠（禄入）→ 吉事在此年发生
```

**同类相冲/叠合检查**：
```
大限忌冲的宫位 = [X宫]
流年的[X宫名]宫是否走到原局的[X宫]？
- 是 → 同类相冲，此年该宫凶事发生
- 否 → 此年不应验该凶

大限禄入的宫位 = [Y宫]
流年的[Y宫名]宫是否走到原局的[Y宫]？
- 是 → 同类叠合，此年该宫吉事发生
- 否 → 此年不应验该吉
```

## Step 5：流月定位（当前年/下一年）

**目的**：对命主最关心的年份，定位关键月份。
**方法**：斗君法推流月 + 上传下应月级验证。
**仅对当前年和/或下一年执行，不对所有年份都做。**

### 5.1 斗君推算

```
1. 以流年太岁宫(=流年地支)起正月
2. 逆数到出生月
3. 从该宫起子时，顺数到出生时辰
4. 到达的宫位 = 斗君 = 正月流月命宫
5. 之后每月顺移一宫

输出：12 个月的流月命宫位置表
```

### 5.2 关键月份筛选

```
关键月条件：
- 流月命宫 = 流年忌所在宫 → 该月应凶
- 流月命宫 = 流年禄所在宫 → 该月应吉
- 流月命宫 = 大限忌所在宫 → 大限凶事在该月应验
- 流月命宫 = 流年命宫(同宫) → 流年能量集中释放月
- 流月命宫 = 原局命宫 → 格局激活月
```

### 5.3 流月输出


## 执行节奏建议

### Context Anxiety 防护（必读）

本 Phase 是全流程分析量最大的阶段，必须遵守以下 5 条防止草率收尾：

1. **分段执行，不要一口气输出**：按 Phase 5a/5b 分段，每段完成后自检质量。
2. **后面的步骤不得比前面粗糙**：若发现 Step 4 以后的分析密度明显低于 Step 2-3，停下来重新展开。
3. **质量优先于完成度**：宁可只分析 2 个关键年但每年深入，也不要 5 个关键年但后 3 个敷衍。
4. **关键年不得集中在一个方向**：吉凶都要有，不要因为凶年分析复杂就只选吉年。
5. **自检信号**：如果开始用"类似前述""不再赘述"等省略语，立即停下——这是 context anxiety 的症状。每个关键年都要独立展开分析。
