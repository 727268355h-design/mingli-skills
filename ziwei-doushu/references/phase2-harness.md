## 2.2 飞化标记溯源

Phase 0 排盘数据卡中，文墨天机标注了各宫的飞化标记（上行标记和下行标记）。此步逐一溯源每个标记的来源。

### 上行标记（化入本宫）溯源

对每个上行标记，执行以下操作：

```
标记：[目标宫] 有 ↑[化X]
溯源方法：
  1. 在 2.1 的 12 宫飞化路径表中搜索：哪个宫的化[X]路径指向[目标宫]？
  2. 找到源宫
  3. 记录：[源宫] 飞 化[X] 入 [目标宫]
  4. 解读：[源宫含义] 对 [目标宫含义] 的影响

输出格式：
  ↑[化X] @ [目标宫]：[源宫]([源宫干][化X]) → [目标宫] = [一句话解读]
```

**解读要点：**
- 化禄入：源宫给目标宫带来福禄、缘分、增添
- 化权入：源宫给目标宫带来掌控、成就、压力
- 化科入：源宫给目标宫带来名声、贵人、解厄
- 化忌入：源宫给目标宫带来执着、纠缠、阻碍（同时冲照对宫）

### 下行标记（化出/自化）说明

对每个下行标记，先判断类型再解读：

```
判断方法：
  Step 1: 取本宫宫干，查十天干四化表
  Step 2: 化[X]对应的星是否在本宫？
    - 是 → 自化（本宫能量自我散发）
    - 否 → 飞化出（本宫能量流向他宫）

自化的四种类型及含义：
  自化禄 = 火山初步活跃，得而复失。热心热情但可能虚饰，不持久
  自化权 = 火山猛烈爆发，遇事强硬主观。能量强但消耗大
  自化科 = 缓缓冒出熔岩，内敛理性有真本事。临危不乱但不积极
  自化忌 = 大量喷发火山灰，最难搞且影响最广。失去还有麻烦，猜疑畏缩

输出格式：
  ↓[化X] @ [本宫]：
    类型 = [自化 / 飞化出]
    若自化：[本宫干][化X][星名] 在本宫 → 自化[X]，[一句话解读]
    若飞化出：[本宫干][化X][星名] → [目标宫]，[一句话解读]
```

**自化的特殊规则：**
- 自化是离心力，能量从内向外散发，代表不持久
- 飞禄逢自化禄有抵消效应——"我对你好，你觉得理所当然"
- 离心忌会冲对宫，向心忌不冲发射宫
- 自化不改变星曜本身的性质，只影响持续性

## 2.3 飞化路径分类

将 2.1 中全部 48 条飞化路径（12宫 x 4化）归入以下四大类。每条路径只归入一类，以最显著特征为准。

### 四大分类标准

**第一类：化入我宫（吉利流向，能量汇聚）**

```
判断标准：化禄/化权/化科 飞入"我方四宫"（命宫/财帛宫/官禄宫/田宅宫）
含义：三吉化入本命三合方为"真吉"，代表环境和他人对"我"的利好
收集方法：从 12 宫路径表中，筛选目标宫为命/财/官/田 且 化类为禄/权/科 的路径
```

**第二类：化出外宫（付出型，能量外散）**

```
判断标准：某宫的四化全部飞往"他方宫位"（兄弟/夫妻/子女/疾厄/迁移/仆役/福德/父母）
含义：此宫能量外散，付出多、收回少。化出为"虚吉"
收集方法：检查每个宫的 4 条路径，若禄/权/科/忌全部指向他宫，标注为"全化出"
特别关注：化禄化权全部化出的宫位，说明此宫主动付出倾向强
```

**第三类：化忌冲击线（最需警惕的凶象路径）**

```
判断标准：任何宫位飞出的化忌路径
含义：化忌飞入的宫位 = 直接阻碍；化忌冲照的对宫 = 破坏力最大
核心规则——忌冲：
  忌入（飞入宫位）= 阻力、执着、纠缠，不算大凶
  忌冲（冲照对宫）= 破坏力最大，"凶事必然发生"
  凡主星落陷 + 加煞 + 忌冲之宫 = 最薄弱之宫 = "破格"

收集方法：
  1. 从 12 宫路径表中提取所有化忌路径
  2. 标注每条忌的飞入宫位和冲照对宫
  3. 标记高危线路：忌冲命宫 / 忌冲官禄 / 忌冲财帛 / 多忌叠加

影响力比例参考：
  自宫 vs 对宫 = 70 : 30
  对宫/自宫 vs 三合双宫 = 60 : 40
```

**第四类：自化泄气（不持久，能量自耗）**

```
判断标准：2.2 中判定为自化的路径
含义：本宫能量不持久，有时效性。自化忌最严重（失去还有麻烦）
收集方法：从 2.2 的下行标记溯源结果中，筛选类型为"自化"的条目
```

### 分类汇总输出

```
化入我宫（吉利流向）：
  - [源宫] 飞 化[X] 入 [命/财/官/田]：[一句话]
  - ...

化出外宫（付出型）：
  - [宫位]：四化全部化出，付出于 [目标宫列表]
  - ...

化忌冲击线（警惕线路）：
  - [源宫] 化忌 → [飞入宫]（冲 [对宫]）：[影响描述]
  - [标记高危线路]
  - ...

自化泄气（不持久）：
  - [宫位] 自化[X]：[影响描述]
  - ...
  （若无自化，标注"无自化"）
```

### 分类特别关注

完成分类后，检查以下模式：
1. **禄权科三吉大量化入我宫** → 先天资源丰厚，但也可能安逸不思进取
2. **化忌集中冲击某一宫** → 此宫为人生最大弱点，后续 Phase 必须重点分析
3. **多宫全化出** → 付出型人格，容易劳碌
4. **化忌冲命宫或冲迁移** → 最凶象之一，必须标红

### 2.4 化忌链追踪与循环发现

**标准操作（必做）：忌转忌两步追踪**

追踪每个化忌的下一步走向：
- A宫化忌入B宫 → B宫天干再化忌入C宫
- 记录：A → B → C 的忌转忌链
- 这是飞星派标准技法（搬山道人、华阳、紫微府等多源确认）

**标准操作（必做）：两宫循环忌检查**

检查是否存在 A宫化忌入B宫 且 B宫化忌入A宫 的互忌模式：
- 含义：两宫之间"有德报德，有仇报仇"，能量来回拉扯
- 多源确认的标准概念

**延伸探索（选做）：三宫以上闭环**

> 注意：以下是基于忌转忌和循环忌的逻辑延伸，非传统命理技法。
> 传统上忌转忌追踪止于两步，三步以上的追踪在公开资料中未找到先例。
> 数学上12宫最多12步必成环（鸽巢原理），但远端追踪的命理意义可能衰减。

若需要做三步以上追踪：
- A宫化忌→B宫化忌→C宫化忌→...→回到已访问宫位
- 发现闭环时描述路径和可能的行为模式
- 标注为"延伸分析，非传统技法"

### 闭环行为模式解读（适用于两宫循环忌和延伸探索）

| 闭环类型 | 典型表现 | 破解方向 |
|---------|---------|---------|
| 命宫-迁移互忌 | 内外拉扯，安全感缺失 | 稳定一端，另一端自然缓解 |
| 官禄-财帛-疾厄三角 | 工作压力→花钱消灾→身体受损→影响工作 | 从疾厄（健康）入手打断 |
| 命宫-夫妻-福德三角 | 自我需求→感情纠葛→精神内耗→自我怀疑 | 福德宫（精神修行）为突破口 |
| 田宅-子女-兄弟 | 家产→子嗣→手足之间的能量循环 | 视具体化星决定破解宫 |

### 输出格式

```
忌转忌链：
  - [A宫]([干]化忌[星]) → [B宫]([干]化忌[星]) → [C宫]
  - ...

两宫循环忌：[是 / 否]
  若是：[A宫] ←→ [B宫] 互忌，行为模式：[描述]
  若否：标注"已检查，未发现两宫循环忌"

三宫以上闭环（延伸分析）：[是 / 否 / 未做]
  若是：路径 + 行为模式 + 标注"延伸分析，非传统技法"
  若否：最长忌链终止于[宫位]
```

### 禄转忌与忌转忌（进阶链追踪）

在闭环检查之外，还需关注两种特殊传导模式：

**禄转忌：**
```
A宫 飞化禄 → B宫，然后 B宫 飞化忌 → C宫
含义：A对B的好意/资源，经由B转化为对C的压力/阻碍
解读模板：[A宫义]给予[B宫义]的[好处]，转化为[B宫义]对[C宫义]的[负面影响]
```

**忌转忌：**
```
A宫 飞化忌 → B宫，然后 B宫 飞化忌 → C宫
含义：A的执着/阻碍传导到B，B再传导到C，形成"连环冲击"
危险度：忌转忌 > 单忌，因为能量叠加放大
解读模板：[A宫义]的[困扰]引发[B宫义]的[困扰]，进而冲击[C宫义]
```

标注所有发现的禄转忌和忌转忌路径。

## 常见错误提醒

1. **查表错误** -- 十天干四化表是最容易出错的环节。丙干化忌是廉贞不是天同，庚干化忌是天同不是太阴。完成后必须逐行回查。
2. **星曜定位错误** -- 某星在哪个宫，必须以 Phase 0 排盘数据卡为准，不可凭记忆。
3. **遗漏自化** -- 宫干化出的星恰好在本宫，容易被当作"飞化出"而忽略自化属性。
4. **忘记标注冲照对宫** -- 化忌不只影响飞入宫，对宫受冲才是最大破坏力。
5. **忌转忌追踪不完整** -- 每条化忌路径都要追踪下一步走向，两宫循环忌是必查项。三宫以上闭环为选做延伸。
6. **宫干来源混淆** -- 宫干由生年天干通过五虎遁诀确定，不是随意排列。若 Phase 0 数据卡已提供宫干则直接使用，不必重算。
