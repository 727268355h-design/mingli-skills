# Phase 0：排盘预检 — 执行指令

> 输入：用户提供的排盘数据（文墨天机文本 / iztro JSON / 截图 / 生辰信息）
> 输出：结构化排盘数据卡（下方模板）
> 原则：此步骤不做任何命理判断，纯数据提取和校验

---

## 0.0 输入识别

用户输入四种情况，按优先级处理：

| 输入类型 | 处理方式 |
|---------|---------|
| **A. 文墨天机树状文本** | 直接解析十二宫、主星、辅星、四化、大限等结构化数据 |
| **B. iztro JSON** | 直接解析 JSON 字段，映射到十二宫数据结构 |
| **C. 排盘截图** | 用视觉识别读取截图中的宫位、星曜、四化信息 |
| **D. 仅提供生辰** | 提示用户排盘后提供数据 |

**D 的处理**：若用户只给了生辰信息，告知：
```
请提供排盘数据。推荐使用文墨天机（微信小程序搜索"文墨天机"）排盘后，
截图或复制文本粘贴给我。也可使用 iztro 等工具输出 JSON。
仅凭生辰我无法确保排盘精度，建议以专业排盘工具为准。
```

## 0.2 真太阳时校验

**仅当用户提供了钟表时间和出生地经度时执行。**

```
真太阳时 = 北京时间 + (出生地经度 - 120度) x 4分钟 + 时差方程修正
```

经度修正：东经 120 度以东为正（时间加快），以西为负（时间延后）。

**关键检查**：修正后若在时辰交界附近（前后 15 分钟内），标注：
```
真太阳时校验：待确认
风险说明：真太阳时在 [X时] 和 [Y时] 交界附近（修正后 HH:MM）。
时辰不同将导致命宫位置和全盘星曜布局改变。
建议命主确认精确出生时间，或对两个时辰分别排盘比对（古法"三时断"）。
```

**夏令时自动扣回**：1986-1991 年中国实行夏令时（具体起止日期见下表，钟表拨快1小时）。
若出生日期在对应年份的夏令时区间内，钟表时间须减 1 小时后再做经度修正。

| 年份 | 夏令时起止 |
|------|----------|
| 1986 | 5月4日 - 9月14日 |
| 1987 | 4月12日 - 9月13日 |
| 1988 | 4月10日 - 9月11日 |
| 1989 | 4月16日 - 9月17日 |
| 1990 | 4月15日 - 9月16日 |
| 1991 | 4月14日 - 9月15日 |

处理规则：出生日期在夏令时区间内 → 钟表时间 - 1小时 = 实际标准时间。

**子时日期归属（早子时 vs 晚子时）**：
- 23:00-00:00（晚子时/夜子时）：日期归**当日**，天干用当日日干
- 00:00-01:00（早子时）：日期归**次日**（已过零点），天干用次日日干
- ⚠️ 此规则影响日柱天干，进而影响日干四化。务必确认排盘工具的子时处理方式

**西部地区特别提醒**：
新疆/西藏/青海/四川西部/贵州/云南等地区经度偏差大（可差 2+ 小时），必须做经度修正。
若用户称出生在上述地区但未提供经度，提示补充或按省会经度近似修正。

**若排盘工具已做修正**：标注「真太阳时校验：已校验（工具内置）」。

**若用户未提供出生地**：标注「真太阳时校验：无需校验（未提供出生地，使用排盘原始时辰）」。

- 宫名：命宫/兄弟/夫妻/子女/财帛/疾厄/迁移/交友/官禄/田宅/福德/父母
- 宫位干支：[天干][地支]

- 辅星：左辅/右弼/天魁/天钺/文昌/文曲/禄存/天马等
- 煞星：擎羊/陀罗/火星/铃星/地空/地劫等
- 小星及神煞：（若排盘数据提供）
- 大限范围：N~M 岁
- 流年列表：（若排盘数据提供）

**四化标注**：
- 生年四化：在对应宫位标注 化禄/化权/化科/化忌
- 飞化标记：若排盘工具提供了宫干飞化信息，标注 ↑入（化入本宫）/ ↓出（化出到他宫）/ 自化

**来因宫标注**：
- 来因宫 = 生年天干所在的宫位（即年干落在哪个宫的宫干位置）
- 在对应宫位标注「来因宫」

## 0.6 四化版本确认

十天干中，甲乙丙丁己辛癸七干四化全派一致，无需选择。
戊庚壬三干有流派分歧，必须确认使用哪个版本：

| 天干 | 争议项 | 全集/钦天门 | 全书/闽派 | 中州派 | 北派 |
|------|--------|-----------|----------|--------|------|
| 戊 | 化科 | 右弼 | 右弼 | 太阳 | 右弼 |
| 庚 | 化科/忌 | 太阴科/天同忌 | 天同科/太阴忌 | 天府科/天同忌 | 天同科/天相忌 |
| 壬 | 化科 | 左辅 | 天府 | 天府 | 左辅 |

处理规则：
1. 若排盘工具已标注版本（如文墨天机支持多流派切换），使用工具版本
2. 若排盘工具未标注，默认使用全集版（最古版本，iztro 默认）
3. 在排盘数据卡中标注使用的四化版本，后续所有 Phase 以此为准
4. 若命主的命盘涉及庚干宫位的飞化分析，在报告中标注"庚干四化存在流派分歧，本分析使用[版本]"

**庚干"五化"现象**（令东来实战发现）：
庚干除标准四化（太阳禄/武曲权/太阴科/天同忌）外，天相在庚干时也会同时表现出化忌特征。
即庚干实际存在"五化"：太阳化禄、武曲化权、太阴化科、天同化忌 + **天相隐性化忌**。
- 此现象在实战中有验证（天同化忌+天相化忌同时应验）
- 处理方式：在庚干飞化分析中，天同化忌为主忌，天相标注为"隐性化忌（令东来实战验证，非标准四化）"
- 在大限/流年遇到庚干宫干时，检查天相所在宫位是否也出现化忌特征
