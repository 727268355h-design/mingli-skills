# Phase 6：综合评判 — 执行指令

> 前置：Phase 5 大限流年卡已完成
> 输入：Phase 0-5 所有卡片数据
> 输出：最终分析报告（下方模板）
>
> **Context Anxiety 防护**：本 Phase 是最终交付，不得因为是"最后一步"而草率。
> 关键事件时间线 >= 8 个、确定性标注 >= 5 个、建议必须具体到飞化依据。
> 如果发现自己在赶工，停下来重新展开。最终报告的质量决定整个分析的价值。
> 原则：**交叉验证，飞化依据贯穿始终，结论必须有明确倾向。** 禁止空泛骑墙。

---

## 6.1 关键事件时间线

从 Phase 5 提取信号强度为"高"或"极高"的年份，汇总为时间线。

**提取规则**：
- 仅取 Phase 5 中明确标注为"高信号强度"或"极高信号强度"的年份
- 每个事件必须追溯到具体的飞化路径（原局/大限/流年三盘叠合）
- 吉凶判定基于飞化吉凶分类（Phase 2）和三盘叠合结果（Phase 5）
- 确定性基于多盘交叉验证的一致程度

**输出表格**：

```
| 年份/虚岁 | 事件类型 | 描述 | 吉凶 | 飞化依据 | 确定性 |
|-----------|---------|------|------|---------|--------|
```

**字段要求**：
- 年份/虚岁：公历年份 + 对应虚岁
- 事件类型：事业变动/财运转折/感情变化/健康风险/搬迁/学业/贵人/是非/意外等
- 描述：一句话概括事件方向（如"大限化禄入官禄叠流年化权，事业跃升期"）
- 吉凶：大吉/吉/平/凶/大凶
- 飞化依据：列出关键飞化路径（如"大限官禄飞禄入命宫，流年化权入迁移"）
- 确定性：✅ 确定 / ⚠️ 不确定 / ❌ 存疑

**数量要求**：至少 8 个事件，涵盖命主主要人生阶段。

## 6.2 核心优势与短板

### 优势提取

从以下来源交叉提取：
1. **飞化吉利路径**（Phase 2）：化禄/化权/化科的吉利落宫和流通路径
2. **格局加分项**（Phase 1）：命宫格局中的有力组合
3. **庙旺主星**（Phase 0/3）：关键宫位的庙旺主星

每条优势格式：
```
优势 N：[一句话概括]
  来源：[格局/飞化路径/主星庙旺]
  表现宫位：[宫位]
  实际影响：[在现实中如何表现]
```

**数量要求**：至少 2 条优势。

### 短板提取

从以下来源交叉提取：
1. **化忌冲击线**（Phase 2）：化忌的落宫及冲破路径
2. **飞化闭环**（Phase 2）：若存在负面闭环，必须列出
3. **陷落星**（Phase 0/3）：关键宫位的陷落主星

每条短板格式：
```
短板 N：[一句话概括]
  来源：[化忌路径/闭环/主星陷落]
  影响宫位：[宫位]
  风险方向：[在现实中如何表现]
  化解线索：[是否有飞化路径可缓解]
```

**数量要求**：至少 1 条短板。

## 6.3 阶段策略表

基于 Phase 5 大限分析，为命主各人生阶段制定策略。

**输出表格**：

```
| 大限 | 年龄段 | 运势基调 | 策略建议 | 需注意 |
|------|--------|---------|---------|--------|
```

**字段要求**：
- 大限：大限宫位名称 + 天干地支
- 年龄段：虚岁范围
- 运势基调：用一句话定性（如"事业上升期，财运随之改善"），必须有明确倾向，禁止"好坏参半"式骑墙
- 策略建议：基于运势基调给出可操作的方向建议
- 需注意：该大限中化忌/煞星带来的具体风险点

**数量要求**：至少 4 个大限，必须包含命主当前所在的大限。

**策略建议原则**：
- 吉利大限：指出主攻方向，利用哪些飞化路径
- 凶险大限：指出防守重点，规避哪些冲击线
- 平淡大限：指出蓄力方向，为下一步大限做准备

## 6.4 趋吉避凶建议

每条建议必须具体到飞化依据，禁止废话建议（如"注意身体""多学习"）。

**每条格式**：
```
- **[方向]**：[具体建议]
  - 理论依据：[星曜/飞化路径]
  - 现实表现：[不执行会怎样，执行会怎样]
  - 可执行动作：[具体做什么]
```

**方向分类**：
- 事业方向：基于官禄宫飞化路径和格局特点
- 财务策略：基于财帛/田宅宫飞化路径
- 人际关系：基于交友/夫妻宫飞化路径
- 健康管理：基于疾厄宫星曜组合和飞化冲击
- 时机把握：基于大限/流年三盘叠合的吉凶窗口

**数量要求**：3-5 条。每条必须可执行，不接受"保持乐观""与人为善"等泛泛而谈。

**禁止项**：
- 禁止没有飞化依据的建议
- 禁止"注意身体""多储蓄"等废话
- 禁止推荐风水物品、改名等非命理范畴的建议

## 6.5 确定性总标注

对全报告的核心判定逐项标注确定性，让命主清楚哪些结论可信度高，哪些仅供参考。

**标注标准**：

| 等级 | 标记 | 条件 |
|------|------|------|
| 确定 | ✅ | 多个 Phase 交叉验证一致（如格局判定 + 飞化路径 + 逐宫分析均指向同一结论） |
| 不确定 | ⚠️ | 单一 Phase 支撑，或 Phase 间有轻微分歧 |
| 存疑 | ❌ | Phase 间有矛盾，或信息不足以下判断 |

**逐项标注**：

```
[判定 1]：[确定性标记] — [支持来源]
[判定 2]：[确定性标记] — [支持来源]
...
```

**数量要求**：至少标注 5 个核心判定。

**标注范围**（优先标注以下判定）：
1. 格局层次判定
2. 事业方向判定
3. 财运层次判定
4. 婚姻/感情判定
5. 最凶险的时间窗口
6. 最有利的时间窗口
7. 核心优势判定
8. 核心短板判定

**处理矛盾**：
- 发现 Phase 间矛盾时，不掩盖，标注 ❌ 并简述矛盾内容
- 矛盾是有价值的信号，提示命理判断的边界

## 6.6 禁止项自检

- [ ] 无废话建议（每条建议有飞化依据和可执行动作）
- [ ] 关键事件都有飞化依据（时间线中每个事件的飞化依据列已填写）
- [ ] 未断生死（命理伦理底线）
- [ ] 未编造未验证的事件（只描述趋势和概率，不编造具体事件）
- [ ] 未推荐风水物品/改名等非命理建议
- [ ] Phase 0-5 的结论在报告中被正确引用（无张冠李戴）
