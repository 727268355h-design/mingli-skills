# Phase 1：命盘总论 -- 执行指令

> 前置：Phase 0 排盘数据卡已完成
> 输入：Phase 0 输出的排盘数据卡（含十二宫主星、生年四化、宫干、庙旺状态）
> 输出：命盘总论卡（下方模板）
> 推理强度：🟡中 -- 格局判定需推理，四化落宫为查表+模式匹配
> 原则：**先定性后细化，先格局后四化**。这一步是"定调"，确立命盘的基本走向。

## 1.1 命宫主星格局定性

从 Phase 0 排盘数据卡中提取命宫主星，判定所属格局类型。

**Step 1：识别命宫主星**

- 从排盘数据卡的命宫栏读取主星（甲级主星）
- 若命宫无主星（空宫），执行借星规则：借对宫（迁移宫）主星论断，力量按 Phase 3 借星折扣规则评估（庙旺太阳/太阴约八九成，其他约五七成）
- 记录主星的庙旺状态（庙/旺/得/利/平/不/陷）

**Step 2：格局速查表逐项匹配**

| 格局 | 核心星曜 | 成格条件（已验证） | 破格条件 | 格局含义 |
|------|---------|-------------------|---------|---------|
| 杀破狼 | 七杀、破军、贪狼 | 三星之一**坐命宫**，其余二星在财帛、官禄（三方四正）。需吉星辅助 | 四煞空劫冲破且缺吉星 | 开创型，一生多变动，先破后立 |
| 机月同梁 | 天机、太阴、天同、天梁 | 四星在命宫三方四正会照，**且必须有文昌或文曲三合加会** | 落陷+六煞入破格 | 稳定型，适合公职/专业技术/幕僚 |
| 紫府同宫 | 紫微、天府 | 紫微天府在**寅或申**同宫坐命，**加会禄存/科权禄/左右/昌曲等吉星** | 空劫羊陀火铃入则减等 | 帝王格，主导力强，甲年生人最佳 |
| 日月并明 | 太阳、太阴 | 太阳在寅/卯/辰/巳/午（庙旺）且太阴在酉/戌/亥/子/丑（庙旺），二星同在三方四正 | 煞忌冲破 | 阴阳调和，贵人多，事业顺遂 |
| 日月反背 | 太阳、太阴 | 太阳在酉/戌/亥/子（落陷）且太阴在卯/辰/巳/午（落陷），在命财官三方会照 | 无吉星加会则为凶格 | 阴阳失调，早年辛苦，有吉星辅助可反转 |
| 火贪格 | 贪狼、火星 | 贪狼坐命（最佳辰戌丑未），火星**同宫**（最优）或三合会照（次之），二星皆庙旺 | 擎羊陀罗冲破（不忌空劫） | 爆发型财运，有禄存/化禄更佳 |
| 铃贪格 | 贪狼、铃星 | 同火贪格，铃星替代火星 | 同火贪格 | 同火贪但力量稍弱，渐进积累型 |
| 七杀朝斗 | 七杀 | 七杀在寅/申/子/午**独坐命宫**，庙旺。对宫紫微天府 | 四煞空劫加会则暴发暴败 | 武将格，事业有大成。需辅弼魁钺加会 |
| 君臣庆会 | 紫微+六吉 | 紫微坐命+**天府天相+左辅右弼+文昌文曲**在三方四正会合，缺一不成格 | 六煞化忌入则"臣蔽君" | 大贵格，加魁钺禄存更佳。魁钺非必须 |
| 马头带箭 | 擎羊、贪狼/天同太阴 | **丙年或戊年**生人，擎羊在午宫与主星同宫坐命。丙年天同太阴+化禄最优，戊年贪狼+火铃次之 | 三方四正煞星过多；无主星则为"马头带剑"（凶格） | 武贵格，镇御边疆。福不耐久 |

**Step 3：判定格局强弱**

格局成立后，评估强弱等级（三级判定，替代二元成/破）：

| 等级 | 条件 | 输出标注 |
|------|------|---------|
| **全成格** | 主星庙旺 + 吉星（左辅/右弼/天魁/天钺/禄存）会照 + 无煞无忌 | 格局：[名称]（全成格） |
| **半破格（降级）** | 煞星在三方四正但不同宫 → 格局降一等；或吉凶混合 | 格局：[名称]（半破格，降为中/弱格）+ 破格原因 |
| **全破格** | 煞忌同宫或冲照命宫 → 格局不成立 | 格局：[名称]（全破格）+ 破格原因 |

**灰色地带判断规则**（最容易出错的判定）：
- 擎羊在三合位但不同宫 → **半破格**（不同命理师意见不一，取保守判定降级处理）
- 火星在三合而非同宫 → 火贪格仍成立但降为中格
- 贪狼不在辰戌丑未但火星同宫 → 勉强成格，标注"非标准格局，降级处理"
- 贪狼化忌 + 文曲同宫 → **全破格**（双重破格信号）

**输出要求**：破格判定必须输出破格原因和破格程度，不可仅写"成立/不成立"

**Step 4：空宫借星特别处理**

若命宫为空宫：
1. 查对宫（迁移宫）主星 -- 借入论断
2. 标注"空宫借星"，按 Phase 3 借星力量折扣规则评估实际力量：
   - 庙旺太阳/太阴约八九成，其他主星约五七成
   - 本宫有煞忌时吉星降至约四成、凶星倍增
   - 分析优先级：本宫小星 > 对宫借星 > 三方四正 > 夹宫
3. 具体六种空宫组合及折扣比例见 Phase 3 section 3.1 checkbox 1（子午八九成、丑未/寅申/辰戌六七成、卯酉视太阳庙陷、巳亥五六成）
4. 借星后仍按上述格局表匹配，但格局强弱评级降一等

## 1.2 身宫位置与命宫关系

**Step 1：确认身宫位置**

身宫只可能出现在以下六个宫位之一：

| 身宫位置 | 后天行为重心 |
|---------|------------|
| 命宫 | 自我驱动极强，命身合一，先天后天一致 |
| 夫妻宫 | 感情/婚姻是此生行为核心，配偶影响大 |
| 财帛宫 | 一生围绕财务运转，赚钱欲望强 |
| 迁移宫 | 外出发展为主，不宜守成，适合异地/海外 |
| 官禄宫 | 事业心极强，以工作成就定义自我 |
| 福德宫 | 注重精神生活，追求内心满足 |

**Step 2：身宫主星与命宫主星互动**

- 身宫主星是什么？庙旺状态？
- 身宫主星与命宫主星是否同系（如都属杀破狼系） -- 同系则先后天方向一致
- 若身宫主星与命宫主星属性冲突（如命宫杀破狼开创+身宫机月同梁保守），标注"内在矛盾"

**Step 3：判断后天行为重心**

结合身宫位置和身宫主星，一句话总结后天行为导向。例：
- "身宫在官禄+武曲坐守 → 后天以事业财务为重心，实干型"
- "身宫在福德+天同坐守 → 后天追求安逸享乐，精神满足优先"

## 1.4 来因宫分析

来因宫是飞星派的核心概念，决定命盘的"体用关系"。

**Step 1：确认来因宫位置和宫干**

- 从 Phase 0 排盘数据卡中读取来因宫标记
- 记录来因宫的宫位名称和天干

**Step 2：判定命运主导权**

| 条件 | 判定 | 含义 |
|------|------|------|
| 来因宫干 = 生年天干 | 命运自主型 | 此生吉凶主要由自己决定，主观能动性强 |
| 来因宫干 != 生年天干 | 受他影响型 | 此生受环境/他人影响较大，需借力而行 |

**Step 3：体用关系**

- 来因宫 = 体（本体，内在根基）
- 来因宫的对宫 = 用（外显，实际运用）
- 体用关系决定了命主"从哪里出发"和"往哪里发力"

举例：
- 来因在命宫 → 体在自我，用在迁移 → 以自身条件为根基，在外部世界施展
- 来因在官禄 → 体在事业，用在夫妻 → 事业为根基，人际/婚姻为发力点
- 来因在财帛 → 体在财务，用在福德 → 财务能力为根基，精神追求为目标

## 1.5 生年四化逐颗分析

生年四化是与生俱来的能量配置，是整张命盘的"基调"。

**Step 1：从排盘数据卡提取四化信息**

逐颗记录：

| 四化 | 化星 | 落宫 | 庙旺状态 | 基本含义 |
|------|------|------|---------|---------|
| 化禄 | [星名] | [宫名] | [庙/旺/得/利/平/不/陷] | 此生最大资源和机会所在 |
| 化权 | [星名] | [宫名] | [庙/旺/得/利/平/不/陷] | 此生最强掌控力和竞争力所在 |
| 化科 | [星名] | [宫名] | [庙/旺/得/利/平/不/陷] | 此生名声、贵人、学识所在 |
| 化忌 | [星名] | [宫名] | [庙/旺/得/利/平/不/陷] | 此生最大课题、执念、阻碍所在 |

**Step 2：庙旺对四化效力的影响矩阵**

这是关键判断 -- 同一颗四化，庙旺和陷落的效果天差地别：

| 四化 | 庙旺（庙/旺/得） | 平和（利/平） | 陷落（不/陷） |
|------|----------------|-------------|-------------|
| 化禄 | 完整发挥，资源充沛 | 资源有但需努力获取 | 资源受限，禄的好处打折 |
| 化权 | 掌控力强，权威稳固 | 有权但需争取 | 权力不稳，容易越权或被夺权 |
| 化科 | 名声显著，贵人有力 | 有名但不显 | 虚名，贵人力弱 |
| 化忌 | 破坏力被压制，问题可控 | 问题中等，需注意 | 破坏力完全释放，此宫为重灾区 |

**Step 3：逐颗四化含义展开**

对每颗四化，结合落宫和庙旺，写出一句定性判断。例：
- "化禄在财帛宫[庙] → 天生财运佳，赚钱有门路，财源稳定"
- "化忌在夫妻宫[陷] → 感情为此生最大课题，婚姻波折概率高"

## 1.6 四化构成的格局判定

检查生年四化的分布是否构成以下格局：

**格局速查表：**

| 格局 | 成立条件 | 含义 |
|------|---------|------|
| 双禄 | 化禄+禄存在同一宫或三方会照 | 财运极佳，但也可能过于贪恋 |
| 禄权科会 | 化禄+化权+化科在三方四正会照 | 大吉格局，才华+权力+名声兼备 |
| 禄忌交战 | 化禄与化忌在对宫或同宫 | 成败同源，得失并存 |
| 忌冲对宫 | 化忌冲对宫（化忌所在宫的对宫受冲） | 对宫事项受损，需特别关注 |
| 科忌同宫 | 化科与化忌同宫 | 问题虽有但有贵人化解 |
| 权忌同宫 | 化权与化忌同宫 | 争执强烈，有能力但阻力也大 |
| 禄忌同宫 | 化禄与化忌同宫 | 财来财去，好坏参半 |

**四化分布模式判定：**

- 集中型：三颗以上四化落在同一三方 → 能量集中，该领域是人生主战场
- 分散型：四化分布在不同三方 → 人生面向多元，不会一面倒
- 内宫型：四化多落在命/兄/夫/子（命宫三方） → 内在/家庭导向
- 外宫型：四化多落在财/疾/迁/官（财官三方） → 外在/事业导向
