# Phase 4：主题专项断事 — 执行指令

> 前置：Phase 3 十二宫逐宫分析卡已完成
> 输入：Phase 0-3 所有卡片数据（排盘数据卡、命盘总论卡、飞化全景卡、十二宫卡）
> 输出：专项断事卡（下方模板，按主题分块）
> 原则：**动态立极 + 飞化佐证。** 不同问题立不同太极宫位，每条结论必须有飞化路径支撑。

**执行指令**：根据命主提问选择对应专项。未明确提问时，默认执行全部 7 项。最低必做 4 项：性格天赋、事业、财运、婚姻感情。

---

## 4.0 动态立极原则

来自星理 12 步 SOP："以来人问事之角度掌握体用。"

**核心概念**：不同问题需要立不同的太极宫位。问事业以官禄宫为太极点，问婚姻以夫妻宫为太极点。立极之后，该宫的三方四正、飞化路径才具有直接意义。

**立极步骤**：
1. 确定问题对应的太极宫位
2. 以该宫为中心重新定义六亲宫位关系
3. 查看该宫的三方四正组合
4. 追踪该宫的飞化入和飞化出路径
5. 结合生年四化在该宫及三方的分布

**体用关系**：
- 体宫 = 太极宫位本身（问题的主体）
- 用宫 = 三方四正及关联宫位（影响主体的外部因素）
- 体宫看状态，用宫看助力与压力

对每个专项，按以下模板逐步执行：

- 煞星影响：[擎羊/陀罗/火铃/空劫等对主星的冲击]

### 步骤 3：三方四正综述

```
三方四正组合：
- [正方宫1]：[主星][庙旺] — [简述]
- [正方宫2]：[主星][庙旺] — [简述]
- [正方宫3]：[主星][庙旺] — [简述]

```

### 步骤 4：飞化佐证（必须至少 1 条）

从 Phase 2 飞化全景卡中，追踪与当前太极宫位相关的飞化路径：

```
飞化佐证：
- [源宫] 飞 [化禄/化权/化科/化忌] 入 [目标宫] → [对此专项的影响]
- [源宫] 飞 [化X] 入 [目标宫] → [影响说明]
（至少列出 1 条，优先列出与太极宫直接相关的飞化）
```

**飞化读法速查**：
- 化禄入太极宫：该领域有缘分、有机会、顺畅
- 化权入太极宫：该领域有掌控力、有成就感、但也有压力
- 化科入太极宫：该领域有名声、贵人、学识助力
- 化忌入太极宫：该领域有执着、有付出、有纠缠，需关注
- 太极宫飞忌出到某宫：主体的能量外泄到该宫，主体受损

**校验**：引用的每条飞化路径必须能在 Phase 2 飞化全景卡中找到对应记录。若找不到，触发回溯条件（见 4.4）。

### 步骤 5：三宫联动验证（令东来技法）

> 三宫联动验证主要适用于**健康/体质判断**场景（命/身/疾三宫交叉指向同一结论）。
> 对于事业、财运、婚姻等其他主题，应使用该主题对应的三方四正组合做交叉验证，
> 而非固定使用命/身/疾三宫。这是令东来的方法论，底层的"多宫交叉验证"原则是全派共识。

选取与当前专项最相关的三个宫位，检查它们的指向是否一致：

```
三宫联动验证：
- [宫A]（[角色说明]）指向：[积极/消极/中性]，依据：[主星+飞化简述]
- [宫B]（[角色说明]）指向：[积极/消极/中性]，依据：[主星+飞化简述]
- [宫C]（[角色说明]）指向：[积极/消极/中性]，依据：[主星+飞化简述]
- 一致性：[强/中/弱/矛盾]
```

**一致性判定**：
- 强：三宫指向完全一致，结论可信度高
- 中：两宫一致、一宫中性或略偏，结论基本可靠
- 弱：两宫一致、一宫反向，结论需谨慎表述
- 矛盾：三宫指向不一致，需如实说明矛盾，不强行统一

### 步骤 6：参看宫位补充

## 4.3 七个专项执行指引

### 4.3.1 性格天赋

**太极宫位**：命宫

**分析要点**：
- 命宫主星定基本性格底色（紫微=领导欲、天机=聪敏善变、太阳=热情直率...）
- 命宫庙旺定性格外显强度
- 福德宫看内心世界、精神追求、思维方式
- 身宫看后天修为方向（身宫所在宫位即后天着力点）
- 三方四正（命+迁+财+官）看社会互动中的性格表现

**飞化关注**：
- 命宫飞化出到哪里：命主精力外放的方向
- 哪些宫飞化入命宫：外界对命主的影响来源
- 生年四化中落命宫者：先天赋予的特质

**三宫联动**：命宫 + 福德宫 + 身宫

---

### 4.3.2 事业

**太极宫位**：官禄宫

**分析要点**：
- 官禄宫主星定事业类型和格局高低
- 命宫看个人能力是否匹配事业格局
- 财帛宫看事业的收益转化能力
- 迁移宫看外部环境和社交资源对事业的助力

**飞化关注**：
- 官禄宫飞禄/权入命宫或财帛宫：事业与个人、财运联动良好
- 官禄宫飞忌入某宫：事业上的执着方向或压力来源
- 命宫飞化入官禄宫：个人主动投入事业的倾向
- 田宅宫与官禄宫的飞化关系：事业是否有根基/平台

**三宫联动**：官禄宫 + 命宫 + 财帛宫

---

### 4.3.3 财运

**太极宫位**：财帛宫

**分析要点**：
- 财帛宫主星定赚钱方式和财运基调
- 官禄宫看赚钱途径（正职收入源头）
- 田宅宫看守财能力和不动产运
- 福德宫看消费观和理财心态

**飞化关注**：
- 财帛宫飞禄入田宅宫：赚来的钱能存住
- 财帛宫飞忌入某宫：花钱的方向或财务压力点
- 官禄宫飞禄入财帛宫：工作确实能带来收入
- 福德宫飞忌入财帛宫：因享乐或心态问题花钱

**三宫联动**：财帛宫 + 官禄宫 + 田宅宫

---

### 4.3.4 婚姻感情

**太极宫位**：夫妻宫

**分析要点**：
- 夫妻宫主星定配偶特质和婚姻互动模式
- 福德宫看感情中的精神需求
- 子女宫看桃花和性缘（夫妻宫的对宫）
- 迁移宫看在外的人际关系是否影响婚姻

**飞化关注**：
- 夫妻宫飞禄入命宫：配偶对命主好
- 夫妻宫飞忌入命宫：配偶对命主纠缠、管束
- 命宫飞忌入夫妻宫：命主对感情执着
- 夫妻宫自化：婚姻关系的不稳定信号
- 福德宫与夫妻宫的飞化关系：内心期待与实际婚姻的匹配度

**三宫联动**：夫妻宫 + 福德宫 + 子女宫

---

### 4.3.5 健康

**太极宫位**：疾厄宫

**分析要点**：
- 疾厄宫主星定体质底色和易患病症方向
- 命宫看先天体质强弱
- 福德宫看精神健康和心理状态
- 疾厄宫的三方四正煞星聚集度判定健康风险等级

**飞化关注**：
- 疾厄宫飞忌入命宫：疾病影响整体运势
- 命宫飞忌入疾厄宫：先天体质有薄弱环节
- 福德宫飞忌入疾厄宫：精神压力转化为身体疾病
- 疾厄宫自化忌：慢性病或反复发作的信号

**三宫联动**：疾厄宫 + 命宫 + 福德宫

**星曜疾病对应速查**：

| 天机 | 肝胆、神经、四肢 |
| 太阳 | 眼目、心脏、血压 |
| 武曲 | 肺、呼吸系统、筋骨 |
| 天同 | 肾、泌尿、耳疾 |
| 廉贞 | 心脏、血液、精神 |
| 天府 | 脾胃、消化系统 |
| 太阴 | 眼目（左眼）、肾、妇科 |
| 贪狼 | 肝、肾、生殖泌尿 |
| 巨门 | 脾胃、口腔、喉咙 |
| 天相 | 皮肤、面部、泌尿 |
| 天梁 | 脾胃、肝胆 |
| 七杀 | 呼吸系统、外伤、手术 |
| 破军 | 泌尿、生殖、外伤 |

### 4.3.6 学业

**太极宫位**：父母宫

**分析要点**：
- 父母宫主星定学业资质和考试运
- 官禄宫看学以致用的能力
- 命宫看个人学习意愿和天赋
- 文昌、文曲、天魁、天钺的位置和庙旺直接影响学业

**飞化关注**：
- 父母宫飞禄/科入命宫或官禄宫：学业成就可转化为事业优势
- 父母宫飞忌入某宫：学业上的障碍来源
- 官禄宫飞化入父母宫：事业发展需要继续深造

**三宫联动**：父母宫 + 官禄宫 + 命宫

### 4.3.7 家庭置产

**太极宫位**：田宅宫

**分析要点**：
- 田宅宫主星定家庭环境和不动产运
- 子女宫看家庭和睦程度（田宅的邻宫）
- 兄弟宫看家族助力和合作置产的可能
- 田宅宫也代表命主的"库"——积蓄能力

**飞化关注**：
- 田宅宫飞禄入财帛宫：家产生财，不动产有收益
- 财帛宫飞禄入田宅宫：赚来的钱能转化为资产
- 田宅宫飞忌入某宫：家庭或置产方面的困扰来源
- 田宅宫自化：家宅不稳或搬迁频繁的信号

**三宫联动**：田宅宫 + 子女宫 + 兄弟宫

## 4.4 回溯条件

**触发条件**：Phase 4 分析中引用的飞化路径在 Phase 2 飞化全景卡中不存在。

**判定方式**：
1. 执行步骤 4（飞化佐证）时，检查每条引用的飞化路径
2. 对照 Phase 2 卡片中的飞化路径表
3. 若存在引用但 Phase 2 未记录的路径，标记为"飞化路径缺失"

**执行流程**：
```
发现飞化路径缺失
    |
    v
标注：BACKTRACK Phase 4 -> Phase 2
原因：[具体哪条飞化路径在 Phase 2 中缺失]
    |
    v
回溯到 Phase 2，补全遗漏的飞化路径
    |
    v
Phase 2 Gate Check 重新校验
    |
    v
Phase 3 重新推进（用补全后的飞化数据更新逐宫分析）
    |
    v
Phase 4 继续
```

**回溯上限**：1 次。若回溯后仍有缺失，标注为存疑项继续推进。

## 4.5 硬约束

1. **每条结论必附飞化依据**：不能只看主星庙旺就下结论，必须有飞化路径支撑
2. **禁止极端断语**：禁止使用"必定大富大贵""一生悲惨""绝对不能"等断言
3. **每条建议三层**：理论依据 -> 现实表现 -> 可执行动作，缺一不可
4. **无孤证**：每个结论至少 2 个宫位联动佐证，单一宫位的单一信息不能独立成为结论
5. **确定性标注**：每个专项的结论必须标注确定性等级
6. **矛盾如实说明**：三宫联动出现矛盾时如实表述，不强行统一为一个结论
7. **引用可溯源**：所有飞化引用必须能回溯到 Phase 2 卡片
