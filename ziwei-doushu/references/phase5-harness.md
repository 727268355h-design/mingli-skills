## 5.1 生年四化三阶分析法（定象→定时→定细节）

> 方法论来源：刘金府"象数心学"体系的"在数因缘"断法。
> 底层逻辑（生年四化定先天格局 + 自化定变动倾向）是飞星派共识，
> 三阶封装（初阶定象/二阶定时/三阶定细节）为刘金府独创。

三阶依次递进，不可跳步。

### 第一阶：在数（定象）

找出"这辈子一定会发生的事"，不问时间。

```
操作：
1. 取生年四化，逐颗确认其"使命"：
   - 生年化禄在[宫] → 此宫领域是一生的机会所在
   - 生年化权在[宫] → 此宫领域是一生的主控欲望
   - 生年化科在[宫] → 此宫领域是一生的声名/贵人来源
   - 生年化忌在[宫] → 此宫领域是一生的执念/业力
2. 生年化忌冲对宫 → 被冲宫的人事注定受损
3. 飞化闭环涉及的宫位 → 此生必有此环上的反复循环事件
```

输出：在数定象表

```
| 生年四化 | 落宫 | 定象（一生使命） |
|---------|------|----------------|
| 化禄[星] | [宫] | [一句话定象] |
| 化权[星] | [宫] | [一句话定象] |
| 化科[星] | [宫] | [一句话定象] |
| 化忌[星] | [宫] | [一句话定象] |
| 化忌冲 | [对宫] | [一句话定损] |
```

### 第二阶：因缘（定时）

找出事件发生的时间窗口。

```
操作：
1. 找自化变量 → Phase 2 飞化全景卡中标注的自化记录
2. 自化挂钩生年四化：
   - 自化禄挂钩生年禄 → 机会的引爆点
   - 自化忌挂钩生年忌 → 灾厄的引爆点
   - 自化权/科挂钩对应生年四化 → 能力/名声的引爆点
3. 大限命宫踩到哪个宫 = 该事件的十年时间窗口
   - 大限命宫走到自化所在宫 → 该限内引爆
   - 大限命宫走到生年化忌所在宫 → 该限内业力显现
   - 大限命宫走到飞化闭环宫 → 该限内循环事件启动
```

输出：因缘定时表

```
| 自化变量 | 所在宫 | 挂钩生年四化 | 事件踩宫时间（第N大限） |
|---------|--------|-------------|---------------------|
| [自化X] | [宫] | 生年[X] | 第[N]大限（[X~Y]虚岁） |
| ... | ... | ... | ... |
```

### 第三阶：时节（定细节）

在定时基础上，加入大限变量和星辰用法丰富论断。

```
操作：
1. 大限四化补充论断：
   - 大限化禄飞入哪宫 → 此限得利方向
   - 大限化忌飞入哪宫 → 此限受困方向
2. 大限主星+庙旺 → 此限的人格面具和处事风格
3. 星辰用法丰富内容：
   - 大限走到的宫位原有哪些星 → 星辰特质加持/拖累
   - 大限辅星（左辅/右弼/天魁/天钺）是否入三方 → 此限贵人运
   - 大限煞星（擎羊/陀罗/火铃/空劫）是否入三方 → 此限风险点
```

## 5.4 流年命宫推算

**流年命宫推算：太岁法（主流方法）**

流年命宫 = 流年地支所在宫位。所有人同年份的流年命宫相同。
例如：丁酉年，所有人的流年命宫都在酉宫。

**流月推算：斗君法（辅助方法）**

斗君用于推算流月正月位置，不用于推流年。
起法：以流年太岁宫起正月，逆数到出生月，再从该宫起子时，顺数到出生时辰止。

两者是层级关系（太岁定年，斗君定月），不是择一关系。

### 流年十二宫排列

以流年命宫为起点，逆时针排列十二宫（与大限十二宫排法一致）：

```
流年命宫 → (逆时针) 流年兄弟 → 流年夫妻 → 流年子女 →
流年财帛 → 流年疾厄 → 流年迁移 → 流年交友 →
流年事业 → 流年田宅 → 流年福德 → 流年父母
```

### 流年四化确定

```
流年四化 = 流年天干 → 查天干四化表（5.2 节同一张表）
```

## 5.5 三盘叠合规则

三盘叠合是 Phase 5 的核心技法。每个宫位同时承载三重身份和三套四化。

### 天地人三才论

| 层级 | 角色 | 权重 | 功能 |
|------|------|------|------|
| 原命盘（天盘） | 民 | 基础 | 先天格局，终生不变 |
| 大限盘（地盘） | 臣 | 次重 | 十年转折，承上启下 |
| 流年盘（人盘） | 君 | 最重 | 具体事件，应期 |

### 三盘叠合信号速查表

| 叠加情况 | 效果 | 信号强度 |
|----------|------|---------|
| 流年化禄 解 原局化忌（同宫） | 可减轻原局化忌的凶性 | 吉（需确认无煞星冲破） |
| 流年化忌 叠 大限化忌（同宫） | 重叠忌，见下方形态细分表 | 极凶 |
| 流年化忌 叠 生年化忌（同宫） | 生年业力引爆，见下方形态细分表 | 极凶 |
| 三忌叠加（生年+大限+流年同宫） | 超凶，该宫事项几乎必然重大变故 | 超凶 |
| 流年天干 = 生年天干 | 生年四化全面重新引动 | 关键年 [综合推论] |
| 流年命宫 = 本命命宫 | 原局格局原样激活 | 关键年 [综合推论] |
| 流年干支 = 大限干支 | 大限四化双重引动 | 此限最重要一年 [综合推论] |
| 化吉叠化吉（禄叠禄/权叠权） | 好上加好 | 吉 |
| 化忌冲化吉 | 由吉变凶 | 凶 |
| 化吉冲化忌 | 化解或减轻 | 吉 |

### 飞宫忌形态细分（八种）

双忌/三忌不可笼统论"大凶"。不同叠加形态的凶险等级和应验方式不同，必须逐一辨别。

| 序号 | 形态 | 定义 | 凶险等级 | 典型应验 |
|------|------|------|----------|----------|
| 1 | **重叠忌** | 本命+大限化忌汇入同一宫（同星或异星） | 极凶 | 该宫事项十年内必有重大损失 |
| 2 | **冲命宫** | 任何盘的化忌冲命宫（含对宫化忌入迁移） | 极凶 | 本人直接受冲，健康/事业/婚姻均可能出事 |
| 3 | **互冲忌** | 本命某宫忌入A宫，大限某宫忌在A宫对宫，形成对宫线互冲 | 高 | 两宫事项同时受损，常见于财帛-福德线 |
| 4 | **反弹忌** | 两盘互相冲击（A盘忌冲B盘某宫，B盘忌冲A盘某宫） | 高 | 两个领域互相拖累，解一个另一个恶化 |
| 5 | **同类相冲** | 天地人三盘的同类宫位（如三盘的财帛宫）化忌互冲 | 高 | 该类事项反复受打击，同类宫互冲才凶 |
| 6 | **三合忌** | 本命+大限化忌呈三合角度（120度/240度） | 中高 | 凶性较重叠忌弱，但影响面更广 |
| 7 | **忌入同类** | 忌星入同类宫位（如大限财帛忌入本命财帛） | 中 | 同类宫互忌不凶（5.3 规则），需结合其他因素 |
| 8 | **单忌** | 不呈现上述任何组合 | 低 | 单忌影响有限，庙旺可压制 |

**三忌叠加特别处理**：

生年+大限+流年三个化忌汇入同一宫 → 超凶信号。具体处理：

1. 三忌同宫同星（如三个廉贞化忌）→ 该宫事项几乎必然发生重大变故
2. 三忌同宫异星（如天同忌+武曲忌+廉贞忌在同宫）→ 多重压力叠加，该宫全面受创
3. "三忌反不忌"规则 → ⚠️ 实战验证不足，暂不纳入主分析。若遇此情况，标注"存在高级规则'三忌反不忌'，需结合调和星曜判定，可靠性存疑"

**分析模板中的输出要求**：

当检测到双忌/三忌叠加时，必须标注：
```
忌形态：[八种之一]
涉及盘：[原局/大限/流年的组合]
涉及星：[化忌星名]
凶险等级：[极凶/高/中高/中/低]
```

### 上中下三级相扣论断法

核心逻辑：上级兆示 -> 中级定型 -> 下级为应期。

```
第一级（命盘兆示）：
  命盘禄忌飞入哪个宫 → 一生的焦点领域

第二级（大限定型）：
  大限走到命盘禄忌所在宫线时 → 该限内吉凶将发生

第三级（流年应期）：
  吉事 → 应于流年同类宫与原命盘禄入宫相叠之年
  凶事 → 应于流年同类宫与原命盘忌冲宫相冲之年
```

体用冲击方向的轻重：
- 上级化忌冲下级（体冲用）→ 凶事较轻，主迹象
- 下级化忌冲上级（用冲体）→ 凶事必然发生，程度重

| 原局 | 大限 | 流年 | 综合结果 |
|------|------|------|----------|

| 吉 | 吉 | 凶 | 小凶，有底子 |
| 吉 | 凶 | 凶 | 凶（大限定型为凶） |
| 凶 | 凶 | 吉 | 平运（格局限制） |
| 凶 | 凶 | 凶 | 大凶 |
| 凶 | 吉 | 吉 | 中吉（先天不足后天补） |

## 5.8 庙旺对四化效力的影响

在大限/流年四化叠合分析中，需考虑星曜庙旺状态对四化效力的调节：

| 组合 | 效果 |
|------|------|
| 化禄/化权 + 庙旺 | 力量完整发挥 |
| 化禄/化权 + 陷落 | 力量打折，约六七成 |
| 化忌 + 庙旺 | 忌的破坏力被压制，约六七成 |
| 化忌 + 陷落 | 忌的破坏力完全释放 |
| 化科 + 任何状态 | 科名/贵人，受庙旺影响较小 |

## 5.9 常见分析陷阱

| 陷阱 | 正确做法 |
|------|---------|
| 看到化忌就说凶 | 化忌庙旺时破坏力有限；化忌也代表执着，有时是动力 |
| 流年只看流年四化 | 必须叠加大限四化+原局四化，三层叠加才是真实效果 |
| 大限只看大限宫主星 | 大限宫干的四化飞往哪里，才决定这十年的核心主题 |
| 把三盘当三次独立分析 | 三盘是有层级的整体，上级兆示下级应期 |
| 忽略同类宫规则 | 同类宫互忌不凶，同类宫互冲才凶 |
| 不标注庙旺就论四化效力 | 庙旺直接影响四化的实际效力 |
| 给出绝对时间断语 | 只说"此限/此年有较强信号"，不说"X年必定发生Y" |
