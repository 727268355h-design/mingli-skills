# Phase 3：十二宫逐宫分析 -- 执行指令

> 前置：Phase 2 飞化全景卡已完成
> 输入：Phase 0 排盘数据卡 + Phase 1 命盘总论卡 + Phase 2 飞化全景卡
> 输出：十二宫卡（下方模板）
> 推理强度：🟡中 -- 按模板逐宫分析，允许适度推断，但不展开长篇论证

---

## 3.0 分析原则

1. **不单星论命，必须综合全盘** -- 孤立看一颗星毫无意义，主星+辅星+神煞+飞化联动才构成完整宫位判断
2. **参考 Phase 1 总论和 Phase 2 飞化全景** -- 生年四化落宫来自 Phase 1，飞化入/出/路径来自 Phase 2，不要重新推导
3. **每宫完成 9 项 checklist 才算完成** -- 缺项标注"无"或"不适用"，不可跳过
4. **庙旺利陀，落陷利辅** -- 主星庙旺时辅星锦上添花，主星落陷时辅星雪中送炭更显重要
5. **三方四正联动** -- 分析某宫时需注意其三方四正的星曜配置，但详细联动留给 Phase 4 专项分析

---

## 3.1 每宫 9 项 Checklist

对十二宫中的每一宫，必须逐项完成以下 9 项检查：

```
□ 1. 主星组合 + 庙旺状态 → 基本含义
     - 双主星注意组合效应（如紫杀、武贪、廉贞天府等经典组合）
     - 空宫借对宫主星，标注"借星安宫"，并按以下折扣规则评估力量：
       **空宫借星力量折扣规则**（令东来体系）：
       a) 庙旺太阳/太阴借入 → 约八九成力量
       b) 其他主星借入 → 约五七成力量（视对宫庙旺状态）
       c) 本宫有煞忌时：借入吉星力量降至约四成，借入凶星力量倍增
       d) 分析优先级（严格按此顺序）：先看本宫小星 → 对宫借星 → 三方四正 → 夹宫
       e) 空宫六种典型组合：
          子/午空宫借天同太阴(八九成) | 丑/未借武曲贪狼(六七成) |
          寅/申借天机太阴(六七成) | 卯/酉借太阳巨门(视太阳庙陷) |
          辰/戌借天机天梁(六七成) | 巳/亥借廉贞贪狼(五六成)
     - 庙旺平陷直接影响星曜发挥

□ 2. 辅星（六吉六煞 + 禄存）
     六吉：左辅、右弼、文昌、文曲、天魁、天钺
     六煞：擎羊、陀罗、火星、铃星、地空、地劫
     禄存：独立标注（与擎羊陀罗必夹）
     - 吉星多寡影响主星发挥上限
     - 煞星不一定凶，庙旺之星见煞反激发（如贪狼见火铃）

□ 3. 小星/杂曜 → 补充判断
     天刑、天姚、天马、红鸾、天喜、华盖、咸池、天空等
     - 小星不改大势，但可微调判断方向
     - 桃花星（天姚/红鸾/咸池）在夫妻/子女宫特别留意

□ 4. 神煞（岁前星/将前星/十二长生/太岁煞禄）
     岁前十二星：岁建、晦气、丧门、贯索、官符、小耗、大耗、龙德、白虎、天德、吊客、病符
     将前十二星：将星、攀鞍、岁驿、息神、华盖、劫煞、灾煞、天煞、指背、咸池、月煞、亡神
     - 神煞为参考，不可凌驾主星判断

□ 5. 生年四化是否落此宫 → 来自 Phase 1
     - 直接引用 Phase 1 命盘总论卡中的生年四化落宫
     - 化禄入宫：此宫得缘、有资源
     - 化权入宫：此宫有掌控力、竞争优势
     - 化科入宫：此宫有声名、贵人、文采
     - 化忌入宫：此宫为执念所在、问题焦点

□ 6. 飞化入（↑）→ 从哪来？什么含义？← 来自 Phase 2
     - 引用 Phase 2 飞化全景卡中的上行标记
     - 格式：↑[化星] 从[源宫]来
     - 解读：源宫对本宫的态度和能量输送

□ 7. 飞化出（↓）→ 飞去哪？什么含义？← 来自 Phase 2
     - 引用 Phase 2 飞化全景卡中的下行标记
     - 格式：↓[化星] 到[目标宫]
     - 解读：本宫对目标宫的影响和能量流出

□ 8. 本宫飞化四化路径 → 此宫主动影响了哪些宫？← 来自 Phase 2
     - 本宫宫干引发的四化分别飞入哪四个宫
     - 格式：禄→[宫] 权→[宫] 科→[宫] 忌→[宫]
     - 特别关注：忌飞入哪宫 = 此宫最大隐患方向

□ 9. 潜在问题和建议
     - 综合 1-8 项，判断此宫是否存在结构性问题
     - 有问题写具体：问题是什么，根源在哪，建议怎么应对
     - 无问题写"无明显问题"，不强行找问题
```

## 3.2 十二宫分析顺序

按以下固定顺序逐宫分析，不可跳跃：

```
1.  命宫    -- 命主核心，全盘枢纽
2.  兄弟宫  -- 手足、同事、合作伙伴
3.  夫妻宫  -- 配偶、婚姻、亲密关系
4.  子女宫  -- 子嗣、下属、桃花、合伙
5.  财帛宫  -- 进财方式、理财能力
6.  疾厄宫  -- 健康、体质、灾厄
7.  迁移宫  -- 外出、社会互动、外在表现
8.  交友宫（仆役宫）  -- 部属、朋友、社交圈层
9.  事业宫（官禄宫）  -- 职业方向、社会成就
10. 田宅宫  -- 不动产、家庭环境、内心世界
11. 福德宫  -- 精神状态、享受、思维模式
12. 父母宫  -- 长辈、教育、文书、因果
```

## 3.3 每宫输出格式

每宫严格按以下格式输出，字段不可缺省：

```
### [宫名]（[天干][地支]）

**主星**：[星名][庙旺] — [一句话基本含义]
**辅星**：[星名列表] — [影响]
**小星**：[关键杂曜] — [补充]
**神煞**：[关键神煞] — [含义]
**生年四化**：[有/无] — [具体，如"化禄在此宫"]
**飞化入**：[↑X 从 Y 宫] — [解读] / 无飞化入
**飞化出**：[↓X 到 Z 宫] — [解读] / 无飞化出
**本宫飞化路径**：禄→[宫] 权→[宫] 科→[宫] 忌→[宫]
**潜在问题**：[有则写，无则"无明显问题"]
```

注意事项：
- 主星庙旺状态用 [庙/旺/得/利/平/不得/落陷] 标注
- 辅星列表按六吉、六煞、禄存的顺序排列
- 飞化入/出若有多条，逐条列出
- 生年四化直接从 Phase 1 卡片引用，不重新推导
