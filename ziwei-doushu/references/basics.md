## 一、紫微斗数概述

紫微斗数为中国五大神数之首，源自北宋，根据人的出生年月日时（农历）排出十二宫构成命盘，参看各宫位中星曜的组合，预测一生运程和吉凶祸福。明清时期由钦天监用以辅佐帝王朝政。

紫微斗数的前身是"十八飞星"，以紫微为主星，设立十八颗虚星，制定一套安星程序用以推算禄命。其兴起背景是当时"五星"与"子平"的竞胜。

**核心特点：**
- 以星曜为工具，以宫位为场景，以四化为动态变化
- 命盘 = 十二宫位 + 星曜（主星/辅星/杂曜） + 四化星
- 论命层次：原局（先天） → 大运（十年） → 流年（一年） → 流月/流日

## 二、十二宫位系统

### 排列顺序（逆时针）

命宫 → 兄弟宫 → 夫妻宫 → 子女宫 → 财帛宫 → 疾厄宫 → 迁移宫 → 交友宫（奴仆宫） → 事业宫（官禄宫） → 田宅宫 → 福德宫 → 父母宫

### 六内宫 vs 六外宫

| 六内宫（自身造化） | 六外宫（六亲外部） |
|---|---|
| 命宫、财帛宫、疾厄宫 | 兄弟宫、夫妻宫、子女宫 |
| 事业宫、田宅宫、福德宫 | 迁移宫、交友宫、父母宫 |

### 各宫含义

| 宫位 | 核心含义 |
|------|---------|
| **命宫** | 命盘核心。性格、品德、容貌、才华、一生总体走向 |
| **兄弟宫** | 与母亲/兄弟/深交朋友/合伙人的相处、缘分、遗传 |
| **夫妻宫** | 伴侣个性、缘分深浅、感情状况、婚姻心态 |
| **子女宫** | 子女多寡与才华、亲子互动、性生活态度、桃花 |
| **财帛宫** | 金钱观念、理财观、花钱态度、赚钱能力、财运 |
| **疾厄宫** | 身体健康、先天体质、大灾大病、情绪表达、内在心态 |
| **迁移宫** | 外出发展际遇、人际关系、旅游移民搬家、外出相关 |
| **交友宫** | 平辈关系、人缘好坏、交往对象类型、互动关系 |
| **事业宫** | 事业职业状况、富贵程度、发展方向、创业能力、社会地位 |
| **田宅宫** | 不动产、居住环境、祖业、风水、财库、家庭相处 |
| **福德宫** | 福气道德、品性修养、精神物质享受能力、宗教信仰、阴德 |
| **父母宫** | 与父母/长辈/上司的互动关系、文书、政府、大环境 |

### 身宫（辅助宫位）

身宫代表后天运势，后天努力可改造命运。身宫必寄附于命宫、夫妻宫、财帛宫、迁移宫、官禄宫、福德宫六宫之一。

### 三方四正

- **三方**：以任一宫为主宫，顺时针第五宫 + 逆时针第五宫 + 对宫 = 三方
- **四正**：本宫 + 三方 = 四正
- 例：子宫的三方为辰宫、申宫、午宫（对宫）

## 三、十四主星系统

### 来源与分类

十四主星源于北斗七星（去第三禄存、第四文曲）+ 南斗六星 + 北极星 + 太阳太阴。

| 分类 | 星曜 |
|------|------|
| **北斗主星** | 紫微（北帝）、贪狼、巨门、廉贞、武曲、破军 |
| **南斗主星** | 天府（南帝）、天相、天梁、天机、七杀、天同 |
| **中天主星** | 太阳、太阴 |

### 两大星系运行

- **紫微系**：紫微带领天机、太阳、武曲、天同、廉贞 → 逆转命盘排兵布阵
- **天府系**：天府带领太阴、贪狼、巨门、天相、天梁、七杀、破军 → 顺转命盘排兵布阵

### 个性分型

| 类型 | 星曜 | 特点 |
|------|------|------|
| **领导型** | 紫微、天府、武曲、天相 | 动静搭配，天生管理者 |
| **开创型** | 贪狼、七杀、廉贞、破军 | 快速直接，天生冒险家 |
| **支援型** | 太阳、巨门、天机 | 热情爽朗，幕僚人才 |
| **合作型** | 太阴、天梁、天同 | 温和稳健，注重和谐 |

### 十四主星性格详解

#### 1. 紫微星（帝王星/尊）
- 五行：己土（阴土）
- 关键词：尊贵、领导、高傲、耳根软、力求完美
- 优点：坚定认真、中庸敦厚、有领导力
- 缺点：自寻烦恼、感情用事、霸气强势

#### 2. 天机星（智慧星/机）
- 五行：乙木（阴木）
- 关键词：智慧、善变、机敏、计较
- 优点：心思灵敏、擅长分析、口才好、乐于助人
- 缺点：不够沉稳、投机取巧、博而不精

#### 3. 太阳星（贵星/贵）
- 五行：丙火（阳火）
- 关键词：贵气、发光发热、声名、代表男性
- 优点：开朗正义、保护弱者、光明磊落
- 缺点：过分热心、精神压力重、不主富

#### 4. 武曲星（财星/将星/刚）
- 五行：辛金（阳金）
- 关键词：财星、刚正、果断、将星
- 优点：眼神坚定、动作敏捷、重视秩序、心地善良
- 缺点：刚愎自用、倔强好胜、操心

#### 5. 天同星（福星/福）
- 五行：壬水（阳水）
- 关键词：福星、平和、知足常乐
- 优点：平易近人、心地宽宏、不记仇、贵人运强
- 缺点：畏首畏尾、感情用事、耐性不足、想多做少

#### 6. 廉贞星（杀星/囚星/傲）
- 五行：丁火（阴火）
- 关键词：高傲、暴烈、双重性格、次桃花星
- 优点：直率、精明、阳刚、有魅力
- 缺点：冲动、划地自限、一意孤行

#### 7. 天府星（财库星/库）
- 五行：戊土（阳土）
- 关键词：禄库、权、稳定、守成
- 优点：才华出众、兴趣广泛、独立自主、聪明好学
- 缺点：爱面子、自负骄傲、不能生财只能守

#### 8. 太阴星（月亮星/富）
- 五行：癸水（阴水）
- 关键词：富、月亮、代表女性、计划理财
- 优点：温柔细腻、善于理财、有耐心
- 缺点：多愁善感、优柔寡断

#### 9. 贪狼星（桃花星/欲）
- 五行：甲木（阳木）
- 关键词：桃花、欲望、多才多艺、豪爽
- 优点：才艺丰富、交际广、适应力强
- 缺点：贪欲重、反复无常、好投机

#### 10. 巨门星（暗星/暗）
- 五行：癸水（阴水）
- 关键词：是非、口才、多疑、暗
- 优点：精明善辩、耿直、直言不讳
- 缺点：苛薄、多口舌是非、言多必失

#### 11. 天相星（印星/印）
- 五行：壬水（阳水）
- 关键词：印花、谨慎、正义感、文雅
- 优点：踏实周全、讲诚信、路见不平拔刀相助
- 缺点：容易过份相信别人

#### 12. 天梁星（寿星/阴星/寿）
- 五行：戊土（阳土）
- 关键词：长者风范、造福人群、寿星
- 优点：温和爽朗、耿直公正、原则性强
- 缺点：主观固执、不肯接纳意见

#### 13. 七杀星（将星/杀）
- 五行：庚金（阳金）
- 关键词：肃杀、伶俐、冲劲、孤独
- 优点：果敢决断、执行力强、独立
- 缺点：好斗、孤僻、情绪起伏大

#### 14. 破军星（耗星/耗）
- 五行：癸水（阴水）
- 关键词：耗、大开大合、侠客、变革
- 优点：机智胆大、有魄力、好奇心强
- 缺点：独断专行、自以为是、多疑多变

### 速记口诀

> 潇洒莫过贪狼，严肃莫过紫微，英勇莫过破军，温柔不过月亮，享受天同化禄，廉贞内心倔强，善变自是天机，口舌牵在巨门，古典多在府相，伶俐必是七杀，急取便是武曲，阳光还是太阳，长者自在天梁。

## 四、辅星系统

### 六吉星

| 星曜 | 取法 | 关键词 | 特质 |
|------|------|--------|------|
| **文昌** | 日参数 | 文学 | 文质彬彬，主科甲，阴金 |
| **文曲** | 日参数 | 口才 | 口才佳，桃花星，属水 |
| **左辅** | 月参数 | 忠厚 | 敦厚慷慨，能文能武，阳土 |
| **右弼** | 月参数 | 豁达 | 乐观好文学，桃花星，阴水 |
| **天魁** | 年参数 | 正直善良 | 积极机敏，贵人星 |
| **天钺** | 年参数 | 自重好义 | 积极上进，贵人星 |

六吉星代表正面能量。三方四正吉星越多，一生运气较顺。

### 六煞星

| 星曜 | 关键词 | 特质 |
|------|--------|------|
| **擎羊** | 权威/冲动 | 事业心强，主意外，阳金 |
| **陀罗** | 推理/顽固 | 容忍心强，宜守不宜攻 |
| **火星** | 刚毅/暴烈 | 心慈善良但个性耿直，创意佳 |
| **铃星** | 刚毅/暴烈 | 类似火星，挫折多但能破局 |
| **地空** | 损失/创意 | 思想独特，主钱财中断 |
| **地劫** | 损失/被劫 | 钱财感情事业挫折，不宜经商 |

**重要原则：**
- 吉凶善恶是相对的，不是绝对的
- 辅星好坏必须和主星一起判断
- 六煞星庙旺时可转化为吉星
- 有煞星激发才能独树一格

### 其他重要辅星

- **禄存**：化气为财，主财禄，必与擎羊陀罗同宫或相邻
- **天马**：主驿马、变动、奔波

## 五、四化星系统

### 四化总论

四化 = 化禄 + 化权 + 化科 + 化忌

"化"是动的现象，由诸星性质演化而来，是主星的余气，表现变化的倾向。四化星在斗数中是断事的基础和直接原因。

### 四化含义

| 化星 | 五行 | 方位 | 季节 | 核心含义 |
|------|------|------|------|---------|
| **化禄** | 金 | 西方 | 秋（收成） | 忙碌、收获、得利、财运、人缘、贵人 |
| **化权** | 火 | 南方 | 夏（生长） | 权力、掌控、职责、主观能动性 |
| **化科** | 木 | 东方 | 春（萌芽） | 功名、名声、善缘、文墨、考试 |
| **化忌** | 水 | 北方 | 冬（死绝） | 是非、困扰、纠缠、波折、阻滞、执着 |

### 季节循环

禄（春/起头） → 权（夏/生长） → 科（秋/收成） → 忌（冬/归藏） → 循环

### 四化同宫效应

- 化禄 + 化忌：金生水，加强化忌力量
- 化权 + 化忌：水克火，自我冲突
- 化科 + 化忌：水生木，利于研究学问

### 重要原则

四化并无绝对好坏。不要看到化禄就高兴，化忌就害怕。要看庙陷、宫位、星曜组合综合判断。

## 八、排盘方法（安星法）

### 排盘八步

1. **确定出生时间**：公历转农历（太阴历），需要万年历
2. **安命宫与身宫**
   - 口诀："斗柄建寅起正月，数至生月顺流行，子时起数生时止，逆回安命顺安身"
   - 寅宫起始，顺数生月，逆数生时 → 命宫
   - 寅宫起始，顺数生月，顺数生时 → 身宫
3. **安十二宫**：从命宫逆时针排列其余十一宫
4. **起寅首（定宫干）**：根据生年天干遁出寅宫天干，顺排各宫天干
5. **定五行局**：命宫地支对应纳音五行
   - 水二局、木三局、金四局、土五局、火六局
6. **安紫微星**：出生日数 ÷ 五行局数 → 定紫微位置
7. **安十四正曜及辅星**：根据紫微位置排布所有星曜
8. **安四化星**：根据天干四化表确定四化

### 安大限

五行局数字 = 起大限岁数（虚岁）
- 水二局 → 2岁起
- 木三局 → 3岁起
- 金四局 → 4岁起
- 土五局 → 5岁起
- 火六局 → 6岁起

### 安长生十二神

依五行局起：金在巳、木在亥、水土在申、火在寅。
阳男阴女顺行，阴男阳女逆行。
顺序：长生 → 沐浴 → 冠带 → 临官 → 帝旺 → 衰 → 病 → 死 → 墓 → 绝 → 胎 → 养

## 九、解盘方法

### 核心原则：由大到小，层层递进

1. **看原局**（先天命盘）→ 定格局高低
2. **看大运**（十年运）→ 看环境吉凶
3. **看流年**（一年运）→ 细查四化与星曜冲起
4. 三盘合参（天盘如民、大限如臣、太岁如君）

### 解盘步骤

1. 先以命宫为中心，确定主星格局
2. 看三方四正（命宫、事业宫、财帛宫、迁移宫）的星曜组合
3. 查看六吉星、六煞星分布
4. 分析四化星的落宫与影响
5. 配合身宫判断后天运势
6. 逐步进入大运、流年的动态分析

### "主星-辅星-四化"三层分析法

1. **看主星定基调**：命宫主星决定基本性格和命运基调
2. **看辅星增细节**：六吉六煞增加吉凶细节
3. **看四化断变化**：四化星揭示动态变化趋势

### 命与运的关系

- "命"的格局高低 = 一生承受富贵福寿的最高限额
- "运"= 外部因素、后天环境影响力
- 原局无官格局 → 大运扶助也难有官，最多事业顺利
- 原局吉多星旺 → 逢恶运落魄，恶运过后能恢复

### 大运断法要点

- 大限落何宫，先应该宫的类神（宫名含义）
- 北斗星吉凶多应于前五年，南斗星多应于后五年
- 看大运命宫三方四正 + 大运四化落宫

### 流年断法要点

- 先以大运为基础，再看流年三方四正
- 看流年四化在哪里，影响哪些宫位
- 小星在流年中作用不可低估

## 十二、主要流派与区别

### 三大流派对比

| 对比项 | 三合派（南派） | 飞星派（北派） | 中州派 |
|---|---|---|---|
| **用星数量** | 100+颗全星系 | 14-18颗主星+辅星 | 介于两者之间 |
| **核心方法** | 星情格局、三方四正 | 四化飞化、逻辑推演 | 星情格局+四化兼用 |
| **四化比重** | 较低（锦上添花） | 极高（核心工具） | 中等偏高 |
| **学习特点** | 考验记忆力 | 考验逻辑推理 | 兼重星情与推理 |
| **主流地区** | 内地、东南亚 | 台湾 | 港澳、内地 |

### 三合派（南派）
- 斗数发源的开始，从《紫微斗数全书》和《紫微斗数全集》发展而来
- 所有星星都要用，包括甲级星32颗、乙级星34颗、丙丁戊级星48颗，共114颗
- 重视星曜本质解释与星曜间组合变化
- 主要利用星性、星群、星系组合为骨干，配以三方四正互动关系断吉凶
- 类似统计运筹分析，适合记忆力好、知识面广的人

### 飞星派（北派/四化派）
- 由"十八飞星策天紫微斗数"衍生，以四化为论命核心
- 仅看14颗主星与4颗辅星（文昌、文曲、左辅、右弼），其他杂曜不参考
- 命盘简易但四化碰撞产生的变化系统更庞大复杂
- 结合先后天八卦、河洛数、五行生克等逻辑推理
- 类似解方程式，适合逻辑推理能力强的人
- 代表人物/支派：蔡明宏（钦天四化）、梁若瑜（梁派飞星）、方外人/翁富裕（河洛飞星）

### 中州派
- 起源洛阳，祖师白玉蝉和吴景鸾，每代只收一弟子口述相传
- 王亭之公开所学后命名为中州派
- 着重星曜赋性和星情格局，论命要求推理能力
- 属三合派大类，但有独特之处：庚、戊、壬天干四化与传统不同
- 流年诸星运用别树一格

### 进阶技法：飞星与宫干

**宫干飞化原理：**
- "天干"带有"干涉"意思，用宫位天干飞出四化星，观察该宫位想要干涉什么
- 宫干飞出的四化遇到生年禄权科忌时，产生吉凶作用力
- 未遇到生年四化，仅表示"关注"、"想要干涉"

**飞宫与自化：**
- 自化 = 本宫或本对宫的直线力量（力量更大）
- 飞宫 = 抛物线的力量（力量较小）
- 飞宫遇生年 → 现象一定发生
- 飞宫遇自化 → 现象往往发生

**气数位：**
- 任何一宫逆数第九位 = 该宫的气数位
- 以气数位宫干所化出的禄权科忌论本宫吉凶

## 十三、常见格局

### 格局的三大分类

1. **富贵格局**：主富贵，做官厅级以上，经商千万以上
2. **凶险格局**：身体差/重病/残疾/生活落魄/债台高筑
3. **杂格（不入格）**：介于两者之间，看大运走势

### 格局强弱判断
- 主星庙旺 + 吉多无煞 → 格局强而有力
- 主星失陷 + 加煞忌星 → 格局弱甚至破格

### 常见富贵格局

| 格局名 | 条件 | 含义 |
|--------|------|------|
| **紫府同宫格** | 紫微天府在寅宫同宫，三方四正遇吉 | 主富贵，巨万之资 |
| **日照雷门格** | 太阳天梁在卯宫同宫 | 富贵荣华 |
| **金灿光辉格** | 太阳在午宫，加吉星三吉化 | 贵格，主权力 |
| **雄宿乾元格** | 廉贞七杀在未宫坐命 | 大气魄，白手取富贵 |
| **双禄朝垣格** | 禄存+化禄分坐官禄/财帛宫 | 一生富贵可期 |
| **机月同梁格** | 天机、太阴、天同、天梁会于命/三方 | 适合公职/大企业 |
| **府相朝垣格** | 天府天相夹命或在三方 | 富贵稳重 |
| **君臣庆会格** | 紫微与左辅右弼/天魁天钺同宫或加会 | 极贵 |
| **三奇加会格** | 化禄+化权+化科在命宫三方四正会齐 | 大富大贵 |
| **贪铃朝垣格** | 贪狼与铃星同宫于四墓地 | 突发富贵 |
| **三合火贪格** | 贪狼与火星在三合宫 | 横发资财 |
| **禄马交驰格** | 禄存与天马同宫或三合 | 财源滚滚 |

### 常见凶险格局

| 格局名 | 条件 | 含义 |
|--------|------|------|
| **日月反背格** | 太阳太阴皆落陷 | 少小离家，奔忙劳碌 |
| **劫空临财福** | 地劫地空守财帛/福德宫 | 一生贫贱 |
| **马落空亡** | 天马逢空亡 | 奔波无功 |
| **羊陀夹忌格** | 擎羊陀罗夹化忌星 | 多灾多难 |
| **泛水桃花格** | 特定条件下的桃花煞格 | 感情困扰 |
| **铃昌陀武** | 铃星+文昌+陀罗+武曲 | 古书称"限至投河" |

### 格局辩证观

- 吉格越多，富贵层次越高
- 吉格遇四煞劫空忌星会照 → 破格，秀而不实
- 命盘只占三分之一（天地人各三分之一），后天努力同样重要
- 能成格者十有三四，可得富贵者百不存一

## 十四、名人命盘案例

### 案例1：奥巴马
- 生于1961年8月4日
- 命宫无主星，看迁移宫：天机+巨门
- 巨门善口才 + 天机聪明多变 → 擅长演说、词辩、谋略
- 事业宫有太阳星（主贵）→ 为人民服务的工作
- 适合律师、议员、领导等角色

### 案例2：李小龙
- 生于1940年11月27日辰时
- 命宫天相星 → 忠诚可靠、正义感
- 迁移宫紫微+破军 → 开创截拳道，提升华人地位
- 福德宫武曲+七杀 → 武打与杀气腾腾的内心
- 事业宫借对宫廉贞+贪狼 → 电影娱乐事业

### 案例3：毛泽东
- 生于1893年12月26日辰时
- 贪狼在申宫坐命
- 辰宫破军化禄 + 子宫禄存 → "双禄朝垣格"
- 迁移宫铃星 → "贪铃朝垣格"
- 多重吉格叠加 → 极贵之命

### 解盘实战方法论

**许铨仁三步法：**
1. 先看来因宫
2. 再看生年四化
3. 看自化（对生年四化的使用），凭自化定时间

**炎一整体把握法：**
1. 拿到命盘先不急于四化飞星飞宫
2. 整体看一遍：命宫主星 → 三方四正吉煞星配置
3. 静态判断整体格局（如双禄交流、有无煞星等）
4. 再进入动态分析

**东派经验：**
- 同年月日时同性别的"五同生人"不会都在同一年结婚/生子
- 案例是既定事实，用来锻炼技法
- 可从任何角度入手（星情、飞宫、飞星），紫微斗数任何角度都能分析

## 十七、桃花星系统

### 桃花星分类

| 星曜 | 级别 | 性质 | 核心特点 |
|------|------|------|---------|
| **贪狼** | 甲级主桃花 | 情欲之星 | 多才多艺，化气为桃花 |
| **廉贞** | 甲级次桃花 | 双重性格 | 阳刚中带阴柔 |
| **红鸾** | 乙级正桃花 | 婚姻喜庆 | 端正桃花，主早婚 |
| **天喜** | 乙级喜庆桃花 | 活泼消灾 | 永在红鸾对宫，主消灾添喜 |
| **天姚** | 乙级交际桃花 | 放纵享受 | 人缘佳，易沉迷感情 |
| **咸池** | 乙级欲望桃花 | 桃花煞 | 主邪淫，永居子午卯酉 |
| **沐浴** | 乙级不端正桃花 | 欲望 | 主色欲 |

### 红鸾星详解
- 五行癸水，主婚姻喜庆，星性开朗直爽
- 安星法：起于卯，依生年地支逆行十二宫
- 入庙最吉，男招美妻，女配贤夫
- 早年逢之主婚姻之喜；中年逢之主添丁喜庆；六十以后逢之易有丧偶之痛
- 红鸾+天姚 → 当年结婚离婚，或因色惹祸
- 红鸾+擎羊 → 血光之灾

### 天喜星详解
- 阳水，永在红鸾对宫
- 安星法：起于酉，依生年地支逆行
- 最大本事是消灾，其次是姻缘
- 与红鸾不同：红鸾可能表示同居，天喜只表示青梅竹马

### 咸池星详解
- 癸水，又名"桃花煞"
- 永居子午卯酉四宫（四败之地）
- 喜与天德、天空、截空同度 → 化解煞气转为才艺

## 二十一、行限体系（小限/流月/流日/流时）

### 完整行限层次

| 层次 | 管辖范围 | 用途 |
|------|---------|------|
| **原局** | 一生 | 定先天格局高低 |
| **大限** | 十年 | 十年运势方向 |
| **小限** | 一年（生日到生日） | 个人内在状态变化 |
| **流年** | 一年（正月到年末） | 年度外部环境吉凶 |
| **流月** | 一个月 | 月度运势细化 |
| **流日** | 一天 | 日运（细节事件） |
| **流时** | 一个时辰 | 极细微时段 |

### 小限 vs 流年
- **流年**：农历正月初一到下一年正月初一，看外部环境
- **小限**：农历生日到下一个农历生日，聚焦个人内在状态
- 两者同时显示某方面压力时，该年风险较高

### 小限排法
口诀："不论阴阳男俱顺数，不论阴阳女俱逆数"
- 寅午戌人起辰宫
- 申子辰人起戌宫
- 巳酉丑人起未宫
- 亥卯未人起丑宫

### 流月排法（定斗君）
1. 以流年太岁所在宫起正月
2. 逆数到出生月宫上
3. 以该宫起子时，顺数到出生时辰止 → 此宫为斗君（正月流月命宫）
4. 之后每月顺时针走一宫

### 流日排法
以流月所在宫起初一，顺行十二宫，一日一宫

### 推断逻辑（层层递进）
1. 看原局隐患/祸根
2. 看大限是否增凶
3. 看流年/小限定灾厄年份
4. 看流月、流日、流时定具体时间点

### 叠象原则
- 叠的越多越准，单象不足以断命
- 某宫位三方四正遇吉星汇聚 + 被大限流年吉化冲起 → 主吉
- 某宫位三方四正遇煞星较多 + 被化忌冲起 → 主凶险

## 二十二、纳音五行与五行局

### 五行局对照

| 五行局 | 局数 | 起大限岁数（虚岁） |
|--------|------|-------------------|
| **水二局** | 2 | 2岁起 |
| **木三局** | 3 | 3岁起 |
| **金四局** | 4 | 4岁起 |
| **土五局** | 5 | 5岁起 |
| **火六局** | 6 | 6岁起 |

### 确定方法
1. 根据命宫的天干地支（不是出生年的天干地支！）
2. 查六十甲子纳音表，取纳音五行
3. 纳音属水 → 水二局，属木 → 木三局，以此类推

### 掌诀速算
知道命宫宫干，只可能是三种五行局：
- 甲己 → 金四局/火六局/水二局
- 乙庚 → 金四局/火六局/水二局
- 丙辛 → 水二局/木三局/土五局
- 丁壬 → 木三局/火六局/土五局
- 戊癸 → 火六局/土五局/木三局

再根据命宫地支确定具体哪一种。

### 五行局在排盘中的核心作用
1. **定紫微星位置**：出生日数 / 局数 = 商数和余数 → 定位紫微
2. **起大限**：局数 = 起运岁数
3. **同主星不同五行局层次不同**：如紫微在子宫，水二局与土五局的人层次大小不一样

## 二十三、紫微斗数 vs 八字对比

| 对比项 | 紫微斗数 | 八字（四柱） |
|--------|---------|-------------|
| **基础** | 星曜 + 宫位 | 五行 + 天干地支 |
| **排盘复杂度** | 高（100+颗星，12宫） | 低（8个字，4柱） |
| **分析精度** | 极细（可到流日流时） | 中等（流年为主） |
| **六亲分析** | 强（专门宫位） | 弱（十神推导） |
| **性格分析** | 强（多维度） | 中等 |
| **运势判断** | 复杂但精细 | 直观高效 |
| **学习难度** | 高 | 中等 |
| **流传地区** | 港澳台、东南亚 | 全球华人 |
| **体系成熟度** | 明清曾失传，近代复兴 | 几千年未断代 |
| **辩证性** | 强（变量多，指导性意义） | 弱（更多直断结果） |

### 核心差异
- **八字胜在**：简洁高效、体系成熟、宏观判断力强、取象直接
- **紫微胜在**：精细多维、六亲分析详尽、辩证指导性强、可推极细时间点

### 使用建议
- 两者并不矛盾，角度和侧重不同
- 八字在运势判断方面更精准
- 紫微斗数在性格分析方面更精准
- 结合使用效果最佳

## 二十四、星曜组合原则

### 双星同宫判断法
1. 双主星同宫 → 以庙旺的一个为主，另一个辅助
2. 吉凶同时存在 → 相互抑制但不抵消，两种性质意义都在
3. 辅星成对同宫（如昌曲、左右）→ 力量最大
4. 辅星单星 → 力量较弱，陷于名义

### 辅佐配比原则
同宫最佳 → 两夹为次 → 三方再次 → 四正辅参

### 十四主星六合关系
- 太阳 ↔ 天府：光明与黑暗

- 太阴 ↔ 破军：柔静与变革
- 天相 ↔ 七杀：文雅与肃杀

### 重要双星组合示例
- **紫微+左辅右弼** → 上品之命，领导才能增强
- **紫微+昌曲** → 才华横溢，文官之命

- **廉贞+贪狼** → 娱乐事业

## 二十五、空宫（命宫无主星）处理

### 空宫的定义
- 宫位内无十四主星，并非一颗星都没有（辅星、杂曜仍可能存在）
- 约六分之五的人命盘有空宫，17%的人命宫是空宫
- 空宫本身不具备凶性，不等于地劫地空

### 空宫的特质
- 像空杯子：可以装水也可以装毒药 → 可塑性高
- 强力吸收海绵 → "近朱者赤、近墨者黑"
- 容易学好也容易学坏
- 内心不安、不踏实、自信不足（但不等于穷困潦倒）

### 空宫分析步骤（有先后顺序）
1. **先看本宫小星**：辅星、煞星、四化（空宫时这些星影响力相对增大）
2. **再借对宫主星**：只借"性质"非"本质"，力量打折扣
3. **再看三方四正**：财帛宫、官禄宫的星曜组合
4. **再看两夹宫**：空宫时两夹的重要性尤为突出

### 借星力量折扣
- 庙旺太阳/太阴借入 → 约八九成力量
- 其他星借入 → 约五到七成
- 本宫有煞忌时 → 对宫吉者约四成，凶者反而倍增

### 争议观点
部分学者认为不应随意借星：
- 排盘设计时特意预留空宫位置，有其存在意义
- 应以三方四正组合呈现的现象来论，而非机械借星
- "借了就要还，空者其实没有东西可以还"

## 二十六、化忌深入解读

### 化忌的本质
- 字面义："己心" → 自己的心思、心魔、执念
- 化忌入哪宫 → 对该宫执着、痴情、亏欠
- 五行壬水，北方坎宫，象冬天万物死绝

### 化忌为何是四化中最重要的？
- 化忌是所有事情的**终极结果**，命运的最后一站
- 决定成败的关键：无论过程多顺利，最终落到化忌的结果上
- 不仅是"坏"，更是"终结" → 可能是转折、停顿、新的开始

### 化忌的两面性

| 负面 | 正面/中性 |
|------|---------|
| 困扰、纠缠、波折、阻滞 | 执着→对某事极度负责认真 |
| 失败、受打击、嫉妒 | 化忌入官禄→工作肯钻、积极 |
| 亏欠、自卑、无缘 | 贪狼化忌→反为收敛（非全坏） |
| 灾咎、破坏 | 阶段性结束→推动重新审视 |

### 化忌的作用方式
- **忌入**：化忌落入某宫，该宫事务受困扰
- **忌冲**（最凶）：化忌在对宫冲照，本宫有煞恶则引爆灾厄
- **流出忌**：本宫化忌到对宫 → 本宫留不住（财、人、缘）
- **循环忌**：两宫相互化忌 → 平等对待的关系
- **叠忌**：多重化忌叠加 → 凶性倍增

### 化忌的解救
1. **化科**（最佳解药）→ 聪慧平和，可与化忌抵消
2. **化权**（次佳）→ 积极向上，可与消极对抗
3. **四化俱全**（禄权科忌皆有）→ "春夏秋冬四象俱全"，反而有大气象

## 二十七、禄存 vs 化禄

| 对比项 | 禄存（明禄） | 化禄（暗禄） |
|--------|-------------|-------------|
| **财富类型** | 固定资产、具体物质（金银珠宝） | 流动资产、抽象价值（才华、人脉） |
| **特性** | 收藏、保守、积蓄（"存"） | 发散、分享、流动（"化"） |
| **得财方式** | 辛劳而得，细水长流 | 忙碌中得，源源不断 |
| **心态** | 偏自私、守财 | 偏博爱、分享 |
| **解厄功能** | 可解厄制化 | 不能解厄 |
| **被破时** | 逢盗窃破坏 | 怀才不遇 |
| **附带条件** | 必被擎羊陀罗所夹 | 无此限制 |
| **力量** | 打折扣（因被羊陀夹） | 较纯粹 |

### 两禄同见的吉格
- 禄存+化禄同宫 → "禄合鸳鸯格"
- 禄存+化禄在三方四正拱照 → "双禄朝垣格"
- 化禄+禄存+天马 → "禄马交驰"，极吉

## 二十九、常见误区与注意事项

### 初学者六大误区

1. **混用门派技法**
   - 三合派用星情，飞星派用四化，混着用会时准时不准
   - 应先选定一个门派系统学习，建立完整断盘逻辑

2. **基础不牢就急于断盘**
   - 星曜的表层和深层解释、组合解释都要掌握
   - 四化的单独含义和组合含义都要理解
   - 十二宫的表层和深层含义都要明白

3. **思维过于呆板**
   - 错误：夫妻宫看婚姻 → 只看夫妻宫
   - 正确：影响婚姻的还有福德宫、迁移宫、官禄宫等
   - 紫微斗数需要灵活联动分析

4. **忽视小星杂曜**
   - 天巫、华盖、白虎、帝旺等不可轻视
   - 在流年盘中小星作用不可低估

5. **忽视格局成格条件**
   - 凡格局，成格均有条件
   - 同一格局在事业宫和六亲宫解释不同，不可硬套

6. **单象断命**
   - 叠象越多越准，单象不足以断命
   - 一个化忌不等于必凶，要看叠了多少层

### 断盘注意事项

- **借宫安星时**：借入星曜的庙旺仍以原宫位为准
- **四化互冲**：注意原局四化是否被大限/流年冲动，冲动后性质是否改变
- **过刚过柔皆不宜**：最佳为刚柔并济、光暗适中
- **推算结论是趋势非必然**：古人规则源于当时社会状态，需辨析修正
- **不可完全依赖命盘做人生决策**

### 学好紫微斗数的四个关键
1. 选定一个门派深入
2. 基础知识必须牢固
3. 分析逻辑要正确（层层递进，由大到小）
4. 勤于实践总结（大量真实命盘练习）

## 三十、命盘结构与地支固定排列

### 命盘外观
命盘是一个 4x4 方格去掉中间 4 格 → 12 格环形结构，类似方形时钟。

### 地支固定位置（永远不变）

```
┌────────┬────────┬────────┬────────┐
│  巳    │  午    │  未    │  申    │
│        │        │        │        │
├────────┼────────┴────────┼────────┤
│  辰    │                  │  酉    │
│        │    （中央空白）   │        │
├────────┤                  ├────────┤
│  卯    │                  │  戌    │
│        │                  │        │
├────────┼────────┬────────┼────────┤
│  寅    │  丑    │  子    │  亥    │
│（左下角│        │        │        │
│ 起点） │        │        │        │
└────────┴────────┴────────┴────────┘
```

- **左下角固定为寅宫**，顺时针排列十二地支
- 地支位置永远不变，变的是命宫落在哪个地支宫位
- 人事十二宫从命宫开始逆时针排入各地支宫位

### 天干排布（五虎遁诀）

根据出生年天干确定寅宫天干，然后顺排：

| 年天干 | 寅宫天干 | 口诀 |
|--------|---------|------|
| 甲/己 | 丙 | 甲己之年丙作首 |
| 乙/庚 | 戊 | 乙庚之岁戊为头 |
| 丙/辛 | 庚 | 丙辛岁首寻庚起 |
| 丁/壬 | 壬 | 丁壬壬位顺行流 |
| 戊/癸 | 甲 | 戊癸甲寅之上求 |

从寅宫天干开始，按甲乙丙丁...顺排各宫（天干只有10个，子宫丑宫的天干分别与寅宫卯宫相同）。

- **庙**：最佳状态，如受供奉（+3）
- **旺**：次佳，星曜有力（+2）
- **平**：中性，不吉不凶（0~+1）
- **陷**：最不适宜，吉曜无力，凶曜增凶（-3）

## 三十二、流年星（流曜）安法

### 流曜概述
流星 = 随岁月时间变化而变化的星曜，**有时效性，过期作废**。

包括：流昌、流曲、流羊、流陀、流火、流铃、流马、流四化、流魁钺等。

### 流昌流曲安法

| 流星 | 起始宫 | 方向 | 特殊规则 |
|------|--------|------|---------|
| **流昌** | 巳宫 | 按天干甲乙顺行 | 跳过四墓宫（辰戌丑未） |
| **流曲** | 酉宫 | 按天干甲乙逆行 | 跳过四墓宫（辰戌丑未） |

注：流曲为中州派独有，其他派别不用。

### 流羊流陀安法
与原盘擎羊陀罗相同：
1. 先定流年禄存位置（根据流年天干）
2. 流羊 = 流禄前一宫
3. 流陀 = 流禄后一宫

**禄存安星口诀：**
> 甲禄到寅宫，乙禄居卯府。丙戊禄在巳，丁己禄在午。
> 庚禄定居申，辛禄酉上补。壬禄亥中藏，癸禄居子户。
> 禄前羊刃当，禄后陀罗府。

### 流四化安法
- 与原盘四化方法相同，只是用流年天干代替出生年天干
- 流月四化：用流月天干（五虎遁月诀推算）
- 流日四化：用流日天干

### 三组四化的层叠
看流年运势时，共有三组四化在作用：
1. **生年四化**（原局，先天定数）
2. **大限四化**（十年环境）
3. **流年四化**（年度变化）

一般流年只看大限+流年两组四化。但当生年四化被限年四化冲会时，须注意其联动作用。

## 三十三、财帛宫与财运

### 财帛宫的核心含义
- 求财的方式、能力、态度、欲望
- 不能单独决定财富多少 → 需结合命宫、官禄宫、福德宫

### 看财运的关联宫位

| 宫位 | 与财运的关系 |
|------|-------------|
| **财帛宫** | 求财方式、理财能力、财运 |
| **福德宫** | 财帛宫的迁移宫，得财的福报 |
| **事业宫** | 财帛宫的财帛宫，工作挣钱模式 |
| **兄弟宫** | 财帛宫的田宅宫，资金储备/现金流 |
| **田宅宫** | 不动产、财库 |

### 主要星曜在财帛宫

| 星曜 | 财运特征 |
|------|---------|
| **紫微** | 财运丰厚稳定。+禄存/化禄=大富。+煞星=先有后破 |
| **武曲** | 正财星，"财入财位"极利。化禄=大富，化权=巨富。陷地+煞=荡尽家财 |
| **太阴** | 正宗财星，偏向长期积累与稳健规划。庙旺财多，陷地财薄 |
| **天同** | 福星，不劳而获型。化忌=财运起伏守不住 |
| **天机** | 靠双手赚取。进财出，来而难守。化禄=流动性大 |
| **太阳** | 经商头脑+理财才能。但好面子，消费大手 |
| **天府** | 库财星，可守成。+空劫=空库，进必有出 |
| **贪狼** | 偏财运。化禄+铃星=横财。但贪欲重易破 |
| **七杀** | 杀破狼格局，冒险中得横财 |
| **破军** | 大进大出，先破后立 |

### 四化在财帛宫的飞化
- 财帛宫化禄飞入的宫位 → 为什么事花钱（甘愿花）
- 财帛宫化忌飞入的宫位 → 不想花又不得不花的事
- 财帛宫化忌冲到的宫位 → 最难赚钱的方向

### 综合判断财运
- 财帛宫+福德宫+子女宫+田宅宫星情皆好 → 真正有钱
- 单看财帛宫不足以判断贫富

## 三十四、时辰校正与定盘

### 为什么要校正时辰？
- 时辰决定命宫位置 → 命宫变则全盘变
- 约20%的命盘因真太阳时等因素排错时辰
- 父母记错、偏远地区时差、夏令时等都可能导致错误

### 导致时辰错误的常见原因

| 原因 | 说明 |
|------|------|
| **真太阳时** | 各地与北京时间有时差（如新疆差约2小时） |
| **夏令时** | 1985-1991年施行，需回拨1小时 |
| **父母记忆** | 记不清或记错时辰 |
| **时辰边界** | 出生在两个时辰分界附近 |

### 定盘方法

**1. 三时断法（最常用）**
- 排出前/中/后三个相邻时辰的命盘
- 逐一对照验证，选最符合的

**2. 事件定盘法（最可靠）**
- 用人生重大事件逆向验证命盘
- 重点事件：搬家、工作变动、婚恋、家庭大事
- 客观事件 > 性格判断（性格似是而非，事件不可模糊）
- 定盘不是时间对就是你的盘，而是事件与盘相符才是

**3. 借盘法（新生儿）**
- 刚出生的宝宝无人生事件可验证
- 用父母当年的事业/健康/财运来定盘

**4. 外貌定盘法（辅助）**
- 通过命主长相特征推断命宫主星
- 不够准确，仅作辅助参考

### 真太阳时换算要点
- 出生地偏离东经120度较远 + 出生时间在时辰分界15分钟内 → 必须校正
- 新疆、西藏、四川西部等地区尤其需要注意
- 精确到分钟最好

## 35. 宫位转换法（立太极）

### 核心概念

宫位转换法的核心思想是"**立太极**"——以任意一宫为新的命宫，重新推演十二宫，分析该宫位所代表之人的各种人生面向。

### 基本原理

- 要看配偶情况：以夫妻宫立太极
- 夫妻之夫妻（即本命财帛宫）= 配偶眼中的你
- 夫妻之事业（即本命福德宫）= 配偶的事业状况
- 父母之父母（即本命福德宫）= 祖父母宫

### 三合宫与气数位

凡任何一宫的三合宫位，均互为气数位：
- 命宫、财帛宫、官禄宫三合：命之气数位为官，官之气数位为财，财之气数位为命
- 兄弟、疾厄、田宅三合：兄之气数位为田，田之气数位为疾，疾之气数位为兄

### 六亲宫位转换表

| 要看的对象 | 立太极宫位 | 转换方法 |
|-----------|-----------|---------|
| 配偶的事业 | 夫妻宫 | 夫妻宫顺数第十宫（本命福德宫） |
| 配偶的财运 | 夫妻宫 | 夫妻宫顺数第五宫（本命财帛宫的对宫） |
| 子女的婚姻 | 子女宫 | 子女宫顺数第七宫 |
| 父亲的健康 | 父母宫 | 父母宫顺数第六宫 |
| 祖父母 | 福德宫 | 福德宫 = 父母之父母 |

### 宫位重叠

命主（宫主）落入不同宫位产生宫位重叠。如命宫宫主到了财帛宫，说明管理命宫的主人去了财帛宫，据此推论命主的行为倾向。

### 注意事项

- 格局在六亲宫的解释与在命宫三方四正的解释不同，不可硬套
- 六亲宫位可结合生肖验证：父母生肖入父母宫或疾厄宫为正位

## 36. 化权与化科深入解读

### 化权星

| 属性 | 内容 |
|------|------|
| 五行 | 阳木（今人论属丙火） |
| 方位 | 南方离宫 |
| 季节象意 | 夏天，万物旺盛 |
| 核心含义 | 权力、权威、能干、成就、竞争力 |
| 性格特征 | 强势、主动、自信、霸气、劳碌 |
| 作用方式 | 集中迸发，有高度也有限度 |

**化权要点**：
- 先天带化权 → 事业心强，喜欢权力
- 大小限流年化权 → 增加权力，做事可成，但辛苦
- 女命不宜（夺夫权），男命宜
- 小星化权（如天同、天机）→ 有挣扎感，力不从心
- 化权是赋予责任和使命，承担重任之权

### 化科星

| 属性 | 内容 |
|------|------|
| 五行 | 水（今人论属甲木） |
| 方位 | 东方震宫 |
| 季节象意 | 春天，万物滋生 |
| 核心含义 | 名誉、科甲、贵人、声誉、学识 |
| 性格特征 | 温文尔雅、谦逊、随和 |
| 作用方式 | 稳步发展，柔性提升 |

**化科要点**：
- 不仅代表学业，更代表声名、名望
- 增加聪明才智，但不一定从事文化类工作
- 男命女命皆可
- 化科如"免战牌"，坐命之人不易遭受突如其来的灾厄
- 爱面子，擅文采，形象风度佳

### 化权 vs 化科核心对比

| 维度 | 化权 | 化科 |
|------|------|------|
| 力量性质 | 刚性，主动争取 | 柔性，润物无声 |
| 命宫表现 | 自我意识强，希望主导局面 | 注重形象塑造，追求高雅品味 |
| 财帛宫表现 | 强掌控欲，善投资理财 | 财运平稳，靠声誉获得机会 |
| 事业宫表现 | 有领导力，职权提升 | 专业受认可，学术成就 |

### 经典口诀

- "权科双夹，必主大贵"
- "权禄重逢，财官双美"
- 化权偏重"实力与行动"，化科偏重"名声与智慧"

## 37. 阴阳男女顺逆行规则

### 阴阳年的判定

天干分阴阳：
- **五阳干**：甲、丙、戊、庚、壬
- **五阴干**：乙、丁、己、辛、癸

出生年干为阳干 → 阳年；出生年干为阴干 → 阴年

### 四种类型

| 类型 | 定义 | 大限方向 |
|------|------|---------|
| 阳男 | 阳年出生的男命 | **顺时针** |
| 阴女 | 阴年出生的女命 | **顺时针** |
| 阴男 | 阴年出生的男命 | **逆时针** |
| 阳女 | 阳年出生的女命 | **逆时针** |

**口诀：阳男阴女顺行，阴男阳女逆行。**

### 大限起始年龄

大限以命盘五行局数为基准，由命宫起始：

| 五行局 | 起大限年龄 | 第一大限范围 |
|--------|-----------|------------|
| 水二局 | 2岁 | 2～11岁 |
| 木三局 | 3岁 | 3～12岁 |
| 金四局 | 4岁 | 4～13岁 |
| 土五局 | 5岁 | 5～14岁 |
| 火六局 | 6岁 | 6～15岁 |

### 顺逆行经过的宫位

- **顺行**：第二大限在父母宫 → 第三大限在福德宫 → 第四大限在田宅宫 → ...
- **逆行**：第二大限在兄弟宫 → 第三大限在夫妻宫 → 第四大限在子女宫 → ...

### 举例

男命1977丁巳年生，丁为阴干 → 阴男 → 逆时针排大限。火六局，命宫大限为6～15岁，第二大限逆行至兄弟宫（16～25岁）。

### 与八字的对应

八字中也有同样规则：阳男阴女大运顺推，阴男阳女大运逆推。两套系统在这一点上完全一致。

## 38. 自化详解

### 什么是自化

"自化"是指某宫的**宫干**引发**本宫中某主星**产生四化。例如亥宫宫干为丁，丁干四化为太阴化禄、天同化权、天机化科、巨门化忌。若太阴恰好在亥宫，则为"太阴自化禄"。

### 自化与生年四化的关系

| 概念 | 说明 |
|------|------|
| 生年四化 | 象之"体"，与生俱来的本质现象，平衡状态 |
| 自化 | 象之"用"，因时间推进产生的变化力量 |
| 两者关系 | 阴阳对待，自化改变生年四化的"质"与"量" |

### 自化的分类

**同组与不同组**：
- 生年四化再自化**同类**（如生年化禄 + 自化禄）→ 量相减（1-1=0），由有变无
- 生年四化再自化**不同类**（如生年化禄 + 自化忌）→ 量相加，产生新的变化

### 自化与飞宫的力量对比

| 类型 | 力量轨迹 | 力量大小 |
|------|---------|---------|
| 生年四化 | 最基础的力量 | 最大 |
| 自化 | 直线力量（本宫/对宫） | 中等，恒大于飞宫 |
| 飞宫 | 抛物线力量 | 较小 |

- 飞宫遇生年 → 现象一定发生
- 飞宫遇自化 → 现象也往往会发生

### 化入与化出

- **化入**（吉）：四化星入命宫、官禄宫、财帛宫、田宅宫
- **化出**（泄）：四化星入兄弟、夫妻、子女、疾厄、迁移、仆役、福德、父母宫
  - 化禄化出 → 守不住财
  - 化权化出 → 喜争权，易有纠纷
  - 化忌化出 → 执念外放

### 飞化转换链

| 转换类型 | 含义 |
|---------|------|
| 禄转禄 | A化禄入B，B化禄入C → 皆大欢喜 |
| 禄转忌 | A化禄入B，B化忌入C → 此处得彼处失 |
| 忌转禄 | A化忌入B，B化禄入C → 跌倒处别处站起 |
| 忌转忌 | A化忌入B，B化忌入C → 执迷之处即败亡处 |

### 实际应用举例

- **命宫自化禄**：多情感性，不愿得罪人，因自化射向对宫迁移宫，人缘好
- **事业宫自化忌**：须追究化禄踪迹——若化禄飞往我宫，自化忌是为自己付出，能获合理回报；若化禄飞往他宫，则损己
