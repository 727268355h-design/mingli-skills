# 紫微斗数单年深度分析 — 执行指令

> **适用场景**：Phase 5 大限分析中识别的关键流年，或命主指定要看的某一年
> **方法论基础**：7 套流派方法统一为 5 层漏斗，取交集定论
> **核心原则**：单层结论只是假设，5 层中 ≥4 层指向同一方向才是定论

## 方法论来源声明

本 harness 综合以下 7 套方法论的核心逻辑：

| # | 方法论 | 来源 | 在本框架中的位置 |
|---|--------|------|----------------|
| 1 | 星情格局法 | 三合派(令东来) | Layer 1 定性 + Layer 5 校验 |
| 2 | 飞宫四化法 | 飞星派(许铨仁) | Layer 2 能量 + Layer 3 因果 |
| 3 | 河洛一六共宗 | 河洛派(紫微精成) | Layer 3 六线 + 忌传播 |
| 4 | 宫位移换法 | 中州派(王亭之) | Layer 1 叠宫 |
| 5 | 在数因缘三阶 | 象数心学(刘金府) | Layer 4 应期 |
| 6 | 四化上传下应 | 紫微精成(综合) | Layer 2 传导 |
| 7 | 小限配合验证 | 多派共识 | Layer 1 验证 + Layer 5 校验 |

## Layer 1：定主题（结构定位）

**目的**：用最快的方法确定"今年的主题是什么"。不涉及吉凶判断。

### 1.1 流年十二宫叠宫（中州派宫位移换法）

以流年命宫为起点，逆时针排列流年十二宫，并逐一标注每个流年宫位叠在原局哪个宫上：

```
| 流年宫位 | 落在地支 | 叠原局宫位 | 叠宫含义 |
|---------|--------|-----------|---------|
| 流年命宫 | [支] | [原局宫名] | 今年主题=[该宫含义] |
| 流年兄弟 | [支] | [原局宫名] | 同侪关系叠[宫]→[含义] |
| 流年夫妻 | [支] | [原局宫名] | 感情叠[宫]→[含义] |
| 流年子女 | [支] | [原局宫名] | 投资/桃花叠[宫]→[含义] |
| 流年财帛 | [支] | [原局宫名] | 赚钱叠[宫]→[含义] |
| 流年疾厄 | [支] | [原局宫名] | 健康叠[宫]→[含义] |
| 流年迁移 | [支] | [原局宫名] | 外出叠[宫]→[含义] |
| 流年交友 | [支] | [原局宫名] | 人际叠[宫]→[含义] |
| 流年官禄 | [支] | [原局宫名] | 事业叠[宫]→[含义] |
| 流年田宅 | [支] | [原局宫名] | 家庭叠[宫]→[含义] |
| 流年福德 | [支] | [原局宫名] | 精神叠[宫]→[含义] |
| 流年父母 | [支] | [原局宫名] | 长辈叠[宫]→[含义] |
```

**关键叠宫信号**：
- 流年命宫叠原局强宫（庙旺+吉化多）→ 年度基调偏吉
- 流年命宫叠原局弱宫（陷+忌+煞）→ 年度基调偏凶
- 流年命宫叠原局命宫 → 格局原样激活（关键年）
- 流年命宫叠大限命宫 → 大限能量集中释放
- 流年某宫叠生年忌所在宫 → 该流年宫的事项受忌影响

### 1.2 小限位置（多派共识验证法）

从 Phase 0 排盘数据卡中查找目标虚岁对应的小限宫位。

```
小限宫位：[地支] = 原局[宫名]
小限主星：[星名][庙旺]
```

**小限验证规则**：
- 小限与流年命宫同宫 → 信号放大（女命才可能，男命永不同宫）
- 小限与大限命宫同宫 → 大限主题在今年格外突出
- 小限落在生年忌宫 → 今年确实与忌的课题相关（验证）
- 小限落在强宫 → 辅助确认年度偏吉；落弱宫 → 辅助确认偏凶

### Layer 1 产出

```
年度主题：[一句话，来自叠宫]
小限验证：[支持/中性/矛盾]
Layer 1 指向：[偏吉/偏凶/中性]
```

## Layer 2：看能量（四化能量流向）

**目的**：确定今年的能量往哪几个方向流。

### 2.1 流年四化（飞星派核心）

以流年天干查四化表：

```
流年天干[X]：
化禄[星] → [宫]（原局宫位）→ 落在流年[宫名]
化权[星] → [宫]（原局宫位）→ 落在流年[宫名]
化科[星] → [宫]（原局宫位）→ 落在流年[宫名]
化忌[星] → [宫]（原局宫位）→ 落在流年[宫名]，冲[对宫]
```

**流年四化与生年四化叠合检查**：
- 流年四化落在生年四化同宫 → 叠合效应（禄叠禄/权叠权/忌叠忌等）
- 流年忌与生年忌同宫同星 → 重叠忌（极凶），标注忌形态
- 流年禄入生年忌宫 → 禄解忌，缓解效应
- 流年忌冲生年禄宫 → 忌冲禄，机会受阻

### 2.2 上传下应（紫微精成传导法）

检查：大限四化是否被今年"踩到"？

```
规则：大限四化落在哪个宫，流年走到那个宫时应验该四化的吉凶。

大限化禄→[宫]：流年命宫/流年相关宫是否走到此宫？→ [是(应吉)/否]
大限化权→[宫]：流年是否走到？→ [是/否]
大限化科→[宫]：流年是否走到？→ [是/否]
大限化忌→[宫]：流年是否走到？→ [是(应凶)/否]
```

**上传下应的"走到"判定**：
- 流年命宫本身 = 大限某四化宫 → 最强应验
- 流年的某功能宫（如流年官禄）= 大限某四化宫 → 在该功能领域应验
- 流年天干 = 大限天干 → 大限四化被完全重新引动

### Layer 2 产出

```
能量流向：
- 吉化(禄/权/科)主要流向：[列出方向]
- 忌化主要冲击：[列出方向]
- 大限四化应验：[是/否，具体哪条]
Layer 2 指向：[偏吉/偏凶/混合]，[具体哪个领域吉/凶]
```

## Layer 3：追因果（忌的传播链 + 六线分析）

**目的**：找出今年最大的吉凶因果链。

### 3.1 化忌因果链（飞星派 + 河洛派联合）

对流年化忌执行三层追踪（河洛法）：

```
第一层（坐忌）：流年忌[星]入[宫] → 该宫增加困难
第二层（冲宫）：冲照[对宫] → 真正的破坏点
第三层（冲宫交友）：[对宫]的交友宫=[宫] → 被牵连的最凶宫位

同时追踪忌转忌：
流年忌入[A宫] → A宫干化忌入[B宫] → B宫干化忌入[C宫]
```

**与原局/大限忌的叠合**：
- 流年忌入某宫 + 原局已有忌入同宫 → 按 Phase 5 忌形态表判定（8 种形态）
- 流年忌入某宫 + 大限忌在同宫 → 忌形态判定
- 三忌同宫（生年+大限+流年）→ 超凶信号

### 3.2 六线检查（河洛六条线法）

命盘六条线：命遷、父疾、兄友、夫官、子田、财福。

检查今年哪条线被激活（有四化坐或冲）：

```
| 线 | 两宫 | 今年是否有四化激活 | 激活类型 | 含义 |
|----|------|------------------|---------|------|
| 命遷线 | 命宫↔迁移 | [是/否] | [禄/权/科/忌] | 社会活动线：人事格局 |
| 父疾线 | 父母↔疾厄 | [是/否] | [禄/权/科/忌] | 文书线：名声运数 |
| 兄友线 | 兄弟↔交友 | [是/否] | [禄/权/科/忌] | 成就线：财利成就 |
| 夫官线 | 夫妻↔官禄 | [是/否] | [禄/权/科/忌] | 事业感情线 |
| 子田线 | 子女↔田宅 | [是/否] | [禄/权/科/忌] | 家庭投资线 |
| 财福线 | 财帛↔福德 | [是/否] | [禄/权/科/忌] | 财运精神线 |
```

**六线判读规则**：
- 吉化(禄权科)入某线 → 该线主题今年有利
- 化忌入某线 → 该线主题今年有压力
- 化忌冲某线 → 该线主题今年有凶变
- 同一线被多颗四化激活 → 该线是今年的焦点

### Layer 3 产出

```
忌因果链：[忌星]→[坐宫]→[冲宫]→[冲宫交友]
忌形态：[八种之一] / 无叠合
忌转忌：[A→B→C] / 无
六线激活：[列出被激活的线+激活类型]
今年最大吉方向：[具体宫/线]，因果=[简述]
今年最大凶方向：[具体宫/线]，因果=[简述]
Layer 3 指向：[逐方向标注吉/凶]
```

## Layer 4：定应期（选做，仅对关键年展开）

**目的**：如果能定，定出关键事件大约在哪几个月。

### 4.1 在数因缘应期检查（象数心学）

检查 Phase 2 中的自化记录是否被今年触发：

```
规则：自化所在宫被流年命宫或流年相关宫踩到 → 该自化在今年应验

Phase 2 自化记录：
- [宫位]自化[X]：今年流年[某宫]是否叠在此宫？→ [是(应验)/否]
```

### 4.2 斗君法推流月（多派共识）

```
斗君推算：
1. 以流年太岁宫(=[流年地支])起正月
2. 逆数到出生月(冬月=十一月)
3. 从该宫起子时，顺数到出生时(申时)
4. 到达的宫位 = 今年的斗君(正月流月命宫)
5. 之后每月顺移一宫

关键月份标注：
- 流月命宫走到流年忌宫的月份 → 该月应凶
- 流月命宫走到流年禄宫的月份 → 该月应吉
- 流月命宫走到大限忌宫的月份 → 大限凶事在该月应验
```

### Layer 4 产出

```
自化应期：[是否触发，具体哪条]
关键月份：[月份+原因] / 未推算
Layer 4 指向：[补充信息] / 不适用
```

## 与 Phase 5 的集成

Phase 5 大限分析中，对满足以下条件的流年调用本 harness：

1. 流年天干 = 生年天干（四化全面引动年）
2. 流年命宫 = 原局命宫/大限命宫（格局激活年）
3. 流年命宫 = 生年忌所在宫（走到忌宫年）
4. 大限忌与流年忌同宫（重叠忌年）
5. 流年天干 = 大限天干（大限四化重新引动年）
6. 命主指定要看的特定年份

其他流年在 Phase 5 中用简略分析即可（流年四化+基础叠合），不调用本 harness。
