## 步骤 1：日干五合检查（先天缘分）

检查两人日干是否构成天干五合：
甲己合/乙庚合/丙辛合/丁壬合/戊癸合

| 结果 | 含义 |
|------|------|
| 日干五合 | 先天缘分极深，有磁场吸引 |
| 日干不合 | 缘分一般，看其他维度 |

```

```

## 步骤 2：日支关系检查（相处模式）

检查两人日支关系：

| 关系 | 含义 | 评分 |
|------|------|------|
| 六合 | 自然契合，相处舒服 | +3 |
| 三合 | 方向一致，目标相同 | +2 |
| 无关系 | 中性 | 0 |
| 六冲 | 性格对冲，矛盾频发 | -3 |
| 六害 | 暗中伤害，表面和但内耗 | -2 |
| 三刑 | 互相折磨，越亲越伤 | -3 |

**六合速查**：子丑/寅亥/卯戌/辰酉/巳申/午未
**六冲速查**：子午/丑未/寅申/卯酉/辰戌/巳亥
**六害速查**：子未/丑午/寅巳/卯辰/申亥/酉戌
**三刑速查**：寅巳申（无恩之刑）/丑戌未（恃势之刑）/子卯（无礼之刑）/辰辰、午午、酉酉、亥亥（自刑）

## 步骤 4：用神互补检查（最核心）

这是合婚最关键的一步：
- 甲方的喜用神五行 是否在 乙方的八字中旺相？
- 乙方的喜用神五行 是否在 甲方的八字中旺相？
- 最佳：互为喜用（甲方忌神是乙方喜用，反之亦然）
- 最差：互为忌神（甲方忌神在乙方也是忌神，两人放大彼此弱点）

**执行步骤**：

1. 读取甲方 Phase 3 用神体系卡：用神=[X]，喜神=[X]，忌神=[X]
2. 读取乙方 Phase 3 用神体系卡：用神=[X]，喜神=[X]，忌神=[X]
3. 检查甲方八字五行力量分布中，乙方用神五行的旺衰程度
4. 检查乙方八字五行力量分布中，甲方用神五行的旺衰程度
5. 综合判断互补度

| 互补度 | 评分 | 说明 |
|--------|------|------|
| 完全互补（互为喜用） | +5 | 甲方旺的五行恰好是乙方需要的，反之亦然 |
| 部分互补（一方为另一方喜用） | +3 | 单向互补，一方受益更多 |
| 中性（无互补也无互克） | 0 | 各过各的，不帮不碍 |
| 部分冲突 | -3 | 一方的忌神在另一方也旺 |
| 完全冲突（互为忌神） | -5 | 两人放大彼此弱点，越亲越衰 |

```

乙方用神 = [X]（[五行]），在甲方八字中：[旺/相/休/囚/死]
互补度：[完全互补/部分互补/中性/部分冲突/完全冲突]
评分：[+5/+3/0/-3/-5]
```

## 步骤 5：子女星同步性

检查两人的子女星（食伤）是否同步：

**子女星对照**：
- 男命：食神=儿子，伤官=女儿
- 女命：伤官=儿子，食神=女儿

**检查项**：

1. 两人子女星的旺衰是否同步（都旺=子女缘厚，都弱=子女缘薄）
2. 两人时柱的关系（合=好，冲=子女问题）
3. 两人第一胎性别预测是否一致

| 同步度 | 评分 |
|--------|------|
| 完全同步（子女星旺衰一致+时柱相合） | +2 |
| 部分同步（旺衰一致或时柱无冲） | +1 |
| 不同步（一方有子女星一方无，或时柱相冲） | -1 |
| 严重冲突（时柱六冲+子女星相克） | -2 |

```
甲方子女星：[食神/伤官] = [旺/弱]，时柱 = [干支]
乙方子女星：[食神/伤官] = [旺/弱]，时柱 = [干支]
时柱关系：[合/冲/刑/无关系]
同步度评分：[+2/+1/-1/-2]
```

## 步骤 6：大运同步性

检查两人关键大运是否重叠：

**检查项**：

1. 双方事业旺运是否同期？（同期=共同上升，利家庭稳定）
2. 双方桃花/婚姻运是否同期？（同期=感情同步，异期=外遇风险）
3. 一方最差运是否恰好是另一方最好运？
   - 互补型（一方好运支撑另一方差运）→ 加分
   - 同步衰（两人同时走差运）→ 减分

| 同步度 | 评分 |
|--------|------|
| 高度同步且互补（好运期重叠，差运期错开互补） | +4 |
| 部分同步（事业运或婚姻运同期） | +2 |
| 中性（各走各的，无明显互补或冲突） | 0 |
| 同步衰（差运期重叠，同时走低谷） | -2 |
| 严重错位（一方最好运=另一方最差运，且无互补） | -4 |

```
甲方关键好运期：第[X]步大运（[年龄段]）
乙方关键好运期：第[X]步大运（[年龄段]）
重叠度：[高/中/低]

甲方关键差运期：第[X]步大运（[年龄段]）
乙方关键差运期：第[X]步大运（[年龄段]）
互补性：[互补/同步衰/错位]

评分：[+4/+2/0/-2/-4]
```
