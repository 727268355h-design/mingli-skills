# 多派系并行分析引擎

> 对 7 个有争议的判定点，不选派系，而是所有派系各自跑完再对比。
> 类似 ML 的 ensemble —— 多个弱分类器投票比单个强分类器更稳。

---

## 触发条件

在以下 7 个判定点，自动切换为"多派系并行模式"：

| # | Phase | 判定点 | 涉及派系 |
|---|-------|--------|---------|
| 1 | 2 | 失令+得地+得势→身旺还是身弱 | 旺衰派 vs 格局派 |
| 2 | 3 | 用神优先级：调候 vs 格局 | 穷通宝鉴派 vs 子平真诠派 vs 旺衰派 |
| 3 | 5 | 午未合化土还是化火 | 传统派 vs 现代派 |
| 4 | 5 | 巳酉是否算暗合 | 认可派 vs 否定派 |
| 5 | 6 | 女命伤官见官=克夫？ | 传统派 vs 现代派 |
| 6 | 7 | 大运干支分管前后5年？ | 三命通会派 vs 滴天髓派 |
| 7 | 7 | 岁运并临必凶？ | 古法 vs 现代 |

## 执行方式

遇到上述判定点时，不做单一选择，按以下格式输出：

```
┌─── 多派系分析 ───────────────────────────┐
│ 判定点：[名称]                            │
│                                           │
│ 【派系 A：XX派】                          │
│ 判定：[结论]                              │
│ 依据：[一句话理由]                         │
│                                           │
│ 【派系 B：XX派】                          │
│ 判定：[结论]                              │
│ 依据：[一句话理由]                         │
│                                           │
│ 【派系 C：XX派】（如有第三派）             │
│ 判定：[结论]                              │
│ 依据：[一句话理由]                         │
│                                           │
│ 【共识/分歧】                             │
│ ├── 共识部分：[各派都同意的]              │
│ └── 分歧部分：[各派不同的]                │
│                                           │
│ 【本次采信】[选哪个] — 理由：[为什么]      │
│ 【如果采信另一派，结论会变为】[备选结论]    │
└──────────────────────────────────────────┘
```

## 7 个争议点的各派系立场速查

### 1. 失令+得地+得势 → 旺还是弱？

| 派系 | 立场 | 理由 |
|------|------|------|
| **旺衰派** | 失令即弱，得地得势只是"弱中不太弱" | 月令占50%权重，失令就是先输一半 |
| **格局派** | 得地得势可补失令，可判中和偏强 | 格局法不以旺衰定命好坏，旺衰只是参考 |
| **新派** | 月令50%+日支20%+其他30%，量化计算 | 公式化，可能偏强也可能偏弱看总分 |

**采信规则**：三派各自算一次，如果两派以上同方向则取多数；全不同则标注"旺衰模糊⚠️"在报告中说明。

### 2. 用神优先级

| 派系 | 排序 | 依据 |
|------|------|------|
| **格局派**（子平真诠） | 格局 > 一切 | "八字用神，专求月令" |
| **调候派**（穷通宝鉴） | 调候 > 格局（冬夏） | "调候为急，先其所急" |
| **旺衰派** | 扶抑 > 病药 > 通关 > 调候 | 日主旺衰是基础 |

**采信规则**：
- 冬月（子丑）夏月（午未）→ 调候派优先
- 春秋月（寅卯申酉）→ 格局派优先
- 四季月（辰戌丑未）→ 看具体情况，旺衰派为辅

### 3. 午未合化土还是化火？

| 派系 | 立场 | 理由 |
|------|------|------|
| **主流** | 合火 | 午未合火，午本气丁火+未中丁火余气 |
| **少数派** | 合土 | 未本气己土，午中己土中气，土旺时化土 |

**采信规则**：默认取合火。若命局土极旺（四柱多辰戌丑未）则注明"可能化土"。

### 4. 巳酉暗合？

| 派系 | 立场 |
|------|------|
| **认可派** | 巳中丙+酉中辛→丙辛合水，算暗合 |
| **否定派** | 巳酉本身半合金，不另论暗合 |

**采信规则**：不主动论巳酉暗合，仅在命局无其他明显关系时作为辅助参考。

### 5. 女命伤官见官

| 派系 | 立场 |
|------|------|
| **传统派** | 克夫，"伤官见官，为祸百端" |
| **现代派** | 不简单等同克夫，需看紧贴/制化/用忌 |

**采信规则**：两派结论都列出。传统解读+现代解读并列，标注条件差异。

### 6. 大运干支分管

| 派系 | 立场 |
|------|------|
| **三命通会** | 天干管前5年，地支管后5年 |
| **滴天髓** | 干支不可分割，统管十年 |
| **权重派** | 前5年干70%支30%，后5年反转 |

**采信规则**：默认用权重派（折中）。如命主追问"这步运前半段和后半段有什么不同"时，补充三命通会的分段解读。

### 7. 岁运并临

| 派系 | 立场 |
|------|------|
| **古法** | "不死自己死长辈"，必凶 |
| **现代** | 看喜忌，喜则吉忌则凶，但事件必大 |

**采信规则**：两派都列。先说"岁运并临=力量加倍，事件必大"（共识），再分"为喜则X，为忌则Y"（现代补充）。

## 与现有 SOP 的集成

在 bazi-prompt.md 的执行规则中增加：

```
7. 遇到争议判定点时，按 `multi-school-engine.md` 的格式输出多派系对比，
   不单选一派。最终采信需给出理由。备选结论需保留。
```

在对应的 Phase harness 中，在争议步骤旁标注 `→ 多派系模式`。
